# 메타 식세기 리타겟팅 세팅 어드바이스 (모수 제약 반영)

작성: 2026-06-12 광고 하치 / 검토 대상: 성락님
전제: 일예산 5~10만원 검토 중, "모수가 작아서 세팅이 안 된다"는 진단에 대한 다각도 대응안

---

## 0. 결론 먼저

1. **"좁은 리타겟팅(ATC/IC 14일)"은 실제로 모수 부족이 맞음** — 성락님 진단 동의. 30일 ATC∪IC 미구매 풀이 약 1,500~2,000명 수준이라 단독 게재 불가/비효율.
2. 따라서 **순수 리타겟팅이 아니라 "웜 오디언스 스택"으로 풀을 3~5만명까지 키워서 세팅**하는 게 답. (방문자 90~180일 + IG/FB 참여자 90일 + 영상 시청자)
3. **소재에 오퍼(쿠폰/무료배송) 없이는 하지 말 것.** 과거 리타겟팅(260330) ROAS 1.75로 ASC(1.86)보다 낮았음 — IC 이탈의 원인이 "잊어버림"이 아니라 "가격 비교"라서, 같은 광고 또 보여주기만 하면 효과 없음이 이미 검증됨.
4. 예산은 **5만으로 시작** (10만은 풀 3만+ 확보 확인 후). 작은 풀에 10만 태우면 빈도 폭발 → 피로 → CPM 급등.

### 이를 유니크 인원으로 환산한 리타겟팅 풀 추정

(이벤트→유니크 중복 제거 약 70~80% 가정)

| 오디언스 | 추정 유니크 | 단독 게재 가능? |
|:--|:--|:--|
| ATC∪IC 미구매 14일 | 약 700~900명 | ❌ 불가 수준 |
| ATC∪IC 미구매 30일 | 약 1,500~2,000명 | ❌ 게재는 되나 비효율 (성락님 진단 구간) |
| 사이트 방문 30일 | 약 5,000~6,000명 | ⚠️ 한계선 |
| 사이트 방문 90~180일 | 약 12,000~20,000명 | ⚠️~✅ |
| **IG/FB 계정·광고 참여자 90일** | **수만 명** (45일 노출 104만회, 도달 43만명 기반) | ✅ 여기가 열쇠 |
| 영상 시청자(50%+) 90일 | 수천~수만 (영상 소재 비중에 비례) | ✅ 보조 |

> 메타 커스텀 오디언스는 기술적 최소 100명이면 만들어지지만, **실무 안정 게재는 수천 명, 일 5만원 집행엔 3만 명+ 필요.** "세팅이 안 된다"는 건 좁은 풀 기준으로는 정확한 판단.

### 빈도 산수 (왜 예산 상한이 모수에 묶이는지)

리타겟팅 CPM은 과거 실측 36,000원대 (260330 캠페인).

| 풀 크기 | 일예산 5만 | 일예산 10만 |
|:--|:--|:--|
| 2,000명 | 주간 빈도 ~4.9 🔴 | ~9.7 🔴🔴 |
| 6,000명 | ~1.6 ✅ | ~3.2 ⚠️ |
| 30,000명 | ~0.3 ✅ | ~0.6 ✅ |

주간 빈도 2.5 초과 시 피로 구간(우리 기준표). **즉 10만원을 쓰려면 풀 3만 명 이상이 선행 조건.**

## 2. 과거 리타겟팅 실측 (260330 Retargeting 식세기) — 가장 중요한 교훈

| 지표 | 260330 리타겟팅 | 260428 ASC (대조) |
|:--|:--|:--|
| 지출 | 124,000원 | - |
| 도달 | **1,855명** (풀 부족 증거) | 430,368명 |
| CPM | **36,191원** (ASC의 2.3배) | 15,409원 |
| CPC | 3,024원 | 1,460원 |
| Click→ATC | 41.0% (웜답게 높음) | 24.7% |
| **ROAS** | **1.75** | **1.86** |

해석: 웜 오디언스라 퍼널 전환율은 좋았지만(ATC율 41%), **풀이 작아 CPM이 2.3배** → 비싸게 사서 본전. 그리고 ROAS가 ASC를 못 넘었다는 건 **IC 이탈자가 "리마인드"로는 안 돌아온다**는 뜻. 자사몰 식세기 IC→구매 이탈 65~70%의 주원인은 가격 비교(자사몰 40,900 vs 쿠팡 시세)로 추정되므로, **돌아올 이유(오퍼)를 줘야 함.**

## 3. 세팅 옵션 비교 (다각도)

### 옵션 A — 웜 오디언스 스택 리타겟팅 ⭐ 추천

- 타겟: `사이트 방문 180일 ∪ IG 참여자 90일 ∪ FB 참여자 90일 ∪ 영상시청 50% 90일`
- 제외: `구매자 180일` + 현재 ASC와 중복 노출 방지 위해 ASC 기존고객 예산 비중 확인
- 추정 풀: 3~5만 명 → 일 5만원 집행 가능
- 장점: 모수 문제 해결, 웜 전환율 활용, ASC 학습 무영향
- 단점: 참여자까지 넓히면 "리타겟팅 순도" 하락 (방문자보다 의도 낮음) → 기대 ROAS 3+가 아니라 2.0~2.5 현실적
- **필수 조건: 오퍼 소재** (첫구매 쿠폰/무료배송/세트 구성가). 오퍼 없으면 260330 재방송.

### 옵션 B — Advantage+ 오디언스(확장 허용) 수동 캠페인

- 시드: ATC/IC 30일 좁은 풀 그대로 넣고 **타겟 확장 허용** (Advantage+ audience)
- 메타가 시드 유사자로 자동 확장 → 게재 막힘 해소
- 장점: 세팅 가장 간단, 좁은 풀 그대로 활용
- 단점: 확장분은 사실상 콜드 → "리타겟팅"이라기보다 시드 기반 프로스펙팅. ASC와 오디언스 중복 커짐 (내부 경매 경합 주의)

### 옵션 C — DPA (카탈로그 동적 리타겟팅)

- ViewContent/ATC 14일에게 본 상품 자동 노출. **작은 풀에 가장 강한 포맷** (동적 게재라 풀 1~2천도 작동)
- 장점: 모수 제약에 구조적으로 가장 강함, 운영 자동
- 단점: 카페24 상품 피드→메타 카탈로그 연동 선행 작업 필요 (반나절~1일). SKU 적은 우리에겐 오버스펙일 수 있음
- 판단: 지금 당장은 아니고, A가 성과 나면 2단계로

### 옵션 D — 신설 없이 ASC 기존고객 비중으로 해결

- ASC 설정의 "기존 고객 예산 한도"를 올려 웜 재노출을 ASC 안에서 소화
- 장점: 캠페인 추가 0개, 중복 노출 걱정 없음
- 단점: 오퍼 소재를 웜에게만 보여줄 수 없음 (콜드에게도 쿠폰 노출 → 마진 누수). 통제 불가
- 판단: 오퍼 전략을 쓸 거면 부적합

### 옵션 E — 리치 목적 + 빈도 캡 (보조)

- 방문자 30일에게 리치 캠페인(빈도 캡 주 2회)으로 저비용 리마인드. CPM 절반 이하
- 일 1~2만원 보조용. 단독 전환 기대 말 것

## 4. 추천 세팅안 (옵션 A 기준 구체안)

```
캠페인: 260612 Retargeting-Warm (식세기)  [판매 목적, 수동]
├ 일예산: 50,000원 (시작) → 풀/빈도 확인 후 증액 판단
├ 광고세트 1개 (분리하지 말 것 — 풀 작을수록 통합)
│   ├ 포함: 사이트방문 180d ∪ IG참여 90d ∪ FB참여 90d ∪ 영상50% 90d
│   ├ 제외: 구매 180d
│   ├ 타겟 확장: OFF (켜면 옵션 B가 되어버림)
│   ├ 노출 위치: Advantage+ 자동 (수동으로 줄이면 작은 풀에서 CPM 더 오름)
│   └ 전환 이벤트: 구매
└ 소재 2~3개
    ├ ① 오퍼형: 무료배송/첫구매 쿠폰 명시 ⭐ 메인
    ├ ② 리뷰/안심형: 후기 캐러셀 (가격 외 가치)
    └ ③ 기존 에이스 소재 1개 (대조군)
```

### 평가 기준 (7일)

| 지표 | 합격 | 중단 |
|:--|:--|:--|
| ROAS | 2.3+ (오퍼 비용 감안 ASC 1.8 대비 우위) | 1.5 미만 |
| CPM | 25,000원 이하 (풀 확보 검증) | 35,000원+ = 풀 여전히 부족 |
| 주간 빈도 | 2.5 이하 | 3.5+ |
| 도달 | 7일 내 10,000+ | 3,000 미만 = 260330 재현 |

- **10만원 증액 조건: 위 4개 합격 + 도달 기준 풀 3만+ 확인 후**
- 측정 주의: 리타겟팅은 어차피 살 사람을 가로채는 어트리뷰션 거품이 생기기 쉬움 → **계정 전체 ROAS(ASC+리타겟팅 합산)가 같이 올라가는지**를 진짜 KPI로 볼 것

## 5. 체크리스트 (세팅 전)

- [ ] 광고관리자에서 각 커스텀 오디언스 생성 후 **추정 크기 실측 확인** (위 추정치 검증 — 이게 1순위)
- [ ] 구매자 180일 제외 오디언스 정상 작동 확인 (픽셀 Purchase 이벤트 기준)
- [ ] ASC 기존고객 예산 비중 현황 확인 (중복 노출 정도 파악)
- [ ] 오퍼 정책 결정: 쿠폰이면 발급 한도/마진 영향 확인 (자사몰 40,900원 기준 마진 시뮬)
- [ ] 6/15 ASC 증액 판단과 분리 운영 — 동시에 두 변수 바꾸면 효과 측정 불가. **리타겟팅 먼저 띄우고 ASC 증액은 그 다음 주 권장**
