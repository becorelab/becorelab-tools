## 3. 타겟 정의

### 3.1 대상 제품
- **쿠팡 등록된 iLBiA·Omomo 전 제품** (건조기시트, 식기세척기 세제, 캡슐 세제, 얼룩 제거제 등)
- 미오가 유튜버 채널 성격에 맞는 제품을 **자동 매칭** (주부 채널 → 세탁·세제 / 살림팁 채널 → 얼룩제거제 등)

### 3.2 대상 유튜버
- **규모**: 마이크로 (구독자 1만 ~ 10만)
- **주제**: 살림·청소·빨래·주부 브이로그·생활팁·제품 리뷰·주방
- **필수 조건**:
  - 최근 30일 이내 업로드 활동 있음
  - 한국어 콘텐츠
  - 비즈니스 이메일 공개

### 3.3 초기 리스트
- **50명** (테스트 사이클)
- 스크리닝은 500~1,000명 풀에서 Haiku 필터 → 대표님 확인 → 상위 50명 발송

## 5. 파이프라인 8단계 명세

### Stage 1: 탐색 (자동)
- **주기**: 수동 트리거 (테스트는 1회)
- **입력**: 키워드 세트 (`살림`, `주부 브이로그`, `세탁`, `청소 꿀팁`, `건조기`, `주방 살림` 등 ~20개)
- **출력**: 후보 1,000명 (channel_id, subscribers, 최근 업로드, 비즈니스 이메일)
- **구현**: YouTube Data API v3 `search.list` + `channels.list`

### Stage 2: 스크리닝 (룰 + Haiku)
- **1차 룰 필터**: 구독자 범위, 최근 30일 업로드, 이메일 공개 여부
- **2차 Haiku**: 채널 설명·최근 영상 제목 3개 읽고 **제품 적합도 점수 (0~100)**
- **출력**: Top 200명 리포트 (대표님 확인용)
- **비용**: 약 1만원 (Haiku)

### Stage 3: 대표님 확정 (수동, 약 20분)
- 웹 대시보드에서 Top 200명 테이블 → 체크박스로 **최종 50명** 선택
- 대표님이 제외할 채널 (경쟁사·부적절 등) 표시

### Stage 4: 개인화 아웃리치 (미오 자동)
- **미오 툴**: `yt_channel_enrich` → `yt_pitch_write`
- 채널당 1건 맞춤 이메일 (채널 톤·최근 영상·시청자 특성 반영)
- **제품 매칭**: 미오가 채널 성격 보고 iLBiA·Omomo 중 1~2개 제품 자동 선택
- **페르소나**: "비코어랩 마케팅팀 [담당자명]"
- **발송 속도**: 하루 10통 × 5일 = 50통 (도메인 워밍업)
- **비용**: 약 1만 5천원 (Opus 50회, 캐싱 적용)

### Stage 5: 답장 처리 (미오 자동 + 대표님 승인)
미오가 답장을 **3단계 분류** (`yt_reply_classify`):

| 분류 | 예시 | 처리 |
|---|---|---|
| 🟢 **자동 응답** | "관심있어요, 샘플 보내주세요" → 주소 요청 | 미오가 즉시 답장 |
| 🟡 **대표님 승인** | "단가 얼마예요?" / "고정비 가능한가요?" | 일 1회 결재함 (오후 5시) |
| 🔴 **즉시 에스컬** | 대형 채널 관심 / 컴플레인 / 법무 이슈 | 텔레그램 즉시 알림 |

### Stage 6: 샘플 발송 (수작업)
- 주 2회 (월/목) 배치 발송
- 웹 대시보드에 "오늘 발송할 주소 리스트" 표시
- 대표님이 송장번호 입력 → 미오가 유튜버에게 자동 알림

### Stage 7: 업로드 모니터링·검수 (미오 자동)
- 샘플 발송 후 4주까지 **주 1회 채널 업로드 체크**
- 협업 영상 감지되면 `yt_video_review` 실행:
  - ✅ #광고 고지 포함 여부
  - ✅ 제품명·기능 정확성
  - ✅ 쿠팡 파트너스 링크 설명란 포함 여부
  - ⚠️ 경쟁사 동시 노출 여부
- 이슈 있으면 대표님께 알림 + 수정 요청 초안 작성

### Stage 8: 잠수 처리 (배치 리포트)
- **감지 지점 4가지**:
  - ① 아웃리치 미응답 7일 → 1회 리마인드 후 "잠수 대기함"
  - ② 관심 표명 후 주소 미제공 3일 → 1회 리마인드 후 "잠수 대기함"
  - ③ 샘플 수령 후 4주 무업로드 → "잠수 대기함"
  - ④ 수정 요청 무시 3일 → "잠수 대기함"
- **보고**: 매주 월요일 오전 9시 텔레그램 → 대표님 결재
  - 대표님 선택: [계속 리마인드 / 종료 / 블랙리스트]

## 6. 미오 에이전트 설계

### 6.1 에이전트 스펙
- **이름**: `미오 파트너스 에이전트`
- **모델**: Opus 4.6 (Haiku는 스크리닝 전용 별도)
- **Environment**: cloud / networking: unrestricted (YouTube 접속)
- **시스템 프롬프트**: 비코어랩 브랜드·제품 지식 + 톤앤매너 (사무실 하치 수준의 이해도)

### 6.2 커스텀 툴 6개

| 툴명 | 입력 | 출력 |
|---|---|---|
| `yt_channel_enrich` | channel_id | 최근 영상 5개 요약, 주시청층, 톤 분석 |
| `yt_pitch_write` | channel_context + 제품 후보 | 맞춤 이메일 본문 (한국어) |
| `yt_reply_classify` | 수신 이메일 본문 | {category, summary, suggested_action} |
| `yt_conversation` | 스레드 + 결재 여부 | 답장 초안 |
| `yt_video_review` | video_url | 검수 리포트 (체크리스트) |
| `yt_ghost_report` | 잠수 대기함 | 대표님 결재용 주간 표 |

### 6.3 재사용
- 기존 `sourcing/mio/` 패턴 복붙 → `marketing/coupang_partners/mio/`
- setup.py / run.py / tools.py 구조 유지
- 개발 시간 **50% 단축 예상**
