## 1. 광고 유형 비교

### 1.1 전체 유형 개요

쿠팡 광고 센터는 2026년 기준 **5가지 광고 유형**을 제공합니다.

| 구분 | AI스마트광고 | 매출최적화 광고 | 매출스타트 광고 | 수동성과형 광고 | 신규고객타겟팅 광고 |
|:--|:--|:--|:--|:--|:--|
| **상품 선택** | 쿠팡 자동 | 셀러 직접 | 셀러 직접 | 셀러 직접 | 셀러 직접 |
| **키워드** | 쿠팡 자동 | 쿠팡 자동 | 쿠팡 자동 | 셀러 직접 + 스마트타겟팅 | 쿠팡 자동 |
| **입찰가** | 자동 | 자동 (ROAS 기반) | 자동 (예산 소진형) | 수동 | 자동 |
| **비검색영역** | 자동 노출 | 강제 노출 (OFF 불가) | 자동 노출 | ON/OFF 입찰가 조절 가능 | 자동 |
| **목표 ROAS** | 설정 가능 | 설정 가능 (핵심) | 설정 불필요 | 해당 없음 | - |
| **제외키워드** | 가능 | 가능 | 불가 | 가능 | - |
| **상품 ON/OFF** | 불가 (전체 자동) | 가능 (개별 OFF) | - | 가능 | 가능 |
| **캠페인 수** | 계정당 1개 | 여러 개 가능 | 여러 개 가능 | 여러 개 가능 | 여러 개 가능 |
| **추천 대상** | 초보/시간 부족 | 중급/특정 상품 집중 | 신제품 초기 노출 | 상급/세밀 제어 | 브랜드 인지도 확장 |

### 1.2 공통 과금 방식: CPC (클릭당 과금)

- 모든 쿠팡 광고는 **CPC(Cost Per Click)** 방식
- 고객이 광고를 클릭할 때만 비용 차감
- **차순위 입찰가(Second-Price Auction)** 적용: 실제 과금 = 2위 입찰가 + 10원
  - 예: 내 입찰가 500원, 2위 입찰가 200원 → 실제 과금 210원
  - 경쟁자 없으면 최소 입찰가 100원으로 과금
- 입찰 범위: 최소 100원 ~ 최대 100,000원/클릭
- 실제 평균 CPC: 200~300원 (경쟁 심한 카테고리 500원+)

### 1.3 전환 어트리뷰션 기준

- **14일 전환 기간** 적용
- 광고 클릭 후 14일 이내 구매 = 해당 광고의 전환으로 인정
- **직접 전환**: 광고 클릭한 그 상품을 구매
- **간접 전환**: 광고 클릭 후 같은 셀러의 다른 상품 구매
- **소급 반영**: 전환 데이터가 실시간이 아닌 소급 반영됨 (CLAUDE.md 참조: JSON vs XLSX 차이 주의)

### 1.4 셀러가 컨트롤할 수 있는 것 vs 없는 것

| | AI스마트 | 매출최적화 | 수동성과형 |
|:--|:--|:--|:--|
| **컨트롤 가능** | 일예산, 목표ROAS, 기간, 제외키워드 | 일예산, 목표ROAS, 상품선택, 제외키워드, 상품ON/OFF | 일예산, 상품, 키워드, 입찰가, 비검색영역 입찰가, 시간대 |
| **컨트롤 불가** | 상품 선택, 키워드 추가, 상품별 ON/OFF, 입찰가, 비검색영역 비중 | 키워드 추가, 입찰가 직접 조정, 비검색영역 OFF | - (거의 모두 제어 가능) |

## 2. AI스마트광고

### 2.1 동작 원리

**핵심 개념**: 셀러의 전체 상품 중 AI가 판매 가능성이 높은 상품을 자동 선별하여 광고 집행

**동작 흐름**:
1. 셀러가 일예산 + 목표ROAS + 기간만 설정
2. 쿠팡 AI가 셀러의 전체 상품을 분석
3. 판매 가능성 높은 상품을 자동 선별
4. 상품명, 카테고리, 검색태그, 옵션을 기반으로 키워드 자동 생성
5. 전환 확률 높은 사용자에게 자동 노출
6. 예산 배분도 AI가 상품별로 자동 결정

### 2.2 키워드 제어

| 항목 | 가능 여부 | 상세 |
|:--|:--|:--|
| 키워드 직접 추가 | **불가** | AI가 자동으로 계속 추가 |
| 키워드 입찰가 조정 | **불가** | AI가 자동 결정 |
| 제외 키워드 설정 | **가능** | 원하지 않는 키워드 제외 가능 |
| 키워드 보고서 확인 | **가능** | 어떤 키워드에 노출되었는지 확인 |

**키워드 추출 방식**: 카테고리 + 상품명 + 검색태그 + 옵션의 조합을 기반으로 "팔릴 가능성이 있는 모든 키워드를 자동으로 계속 추가"

### 2.3 상품 관리

- **상품 직접 추가/제거**: 원칙적으로 불가 (AI 자동 선별)
- **특정 상품 제외**: 가능 (2024.07 업데이트 — 쿠팡 공식 영상 제공)
- **자동 제외 대상**: 중복 상품(다른 캠페인 운영 중), 품절 상품, 아이템위너 아닌 상품
- **계정당 1개 캠페인만** 운영 가능

### 2.4 비검색영역

- **ON/OFF 설정**: 불가
- 검색영역 + 비검색영역 모두 자동 노출
- 비검색영역 입찰가 별도 조절 불가

### 2.5 목표 ROAS 설정 시 AI 동작

| 목표 ROAS | AI 동작 | 결과 |
|:--|:--|:--|
| 낮게 설정 (100~200%) | 넓은 오디언스에 공격적 노출 | 노출 많음, 예산 빠르게 소진, ROAS 낮을 수 있음 |
| 적정 (300~350%) | 구매 확률 중간 이상 사용자에게 노출 | 균형잡힌 노출과 효율 |
| 높게 설정 (500%+) | 구매 확률 매우 높은 소수에게만 노출 | 노출 급감, 예산 미소진, 데이터 축적 어려움 |

### 2.6 예산 배분 로직

- AI가 **매출 예측이 높은 상품**에 예산을 자동 집중
- 셀러가 밀고 싶은 상품 ≠ AI가 판단한 상품일 수 있음 (주요 단점)
- 예산 배분 비중은 셀러가 직접 조절할 수 없음
- **우리 경험 (채움컴퍼니 사례)**: 베이비크림에 예산 13%→68% 자동 이동, 입테이프는 73%→22%로 감소

### 2.7 학습 기간

- **최적화 기간: 7~14일** (공식)
- 학습 완료까지 실질적으로 **2~4주** 소요
- 학습 기간 중 **절대 하지 말아야 할 것**:
  - 캠페인 자주 ON/OFF 토글
  - 일예산 변경
  - 목표 ROAS 변경
  - 예산과 ROAS 동시 조정
- **최소 일예산 3만원 이상** 유지 권장 (데이터 축적 필요)

### 2.8 AI스마트광고 vs 매출최적화 — 핵심 차이

| 비교 항목 | AI스마트광고 | 매출최적화 광고 |
|:--|:--|:--|
| 상품 선택 | AI 자동 | 셀러 직접 |
| 상품별 예산 통제 | 불가 | 캠페인 분리로 가능 |
| 상품별 ON/OFF | 불가 (제외만 가능) | 가능 |
| 캠페인 수 | 계정당 1개 | 제한 없음 |
| 공격성 | 보수적 | 더 공격적 (비검색 강제 노출) |
| 평균 ROAS | 200~300% | 상품에 따라 다양 |
| 적합 상황 | 다수 상품 전체 노출 | 특정 상품 집중 투자 |

## 3. 매출최적화 광고

### 3.1 정확한 입찰 원리

**핵심**: 셀러가 목표 ROAS + 일예산을 설정하면, AI가 입찰가를 자동 결정

**AI 입찰 로직**:
1. 쿠팡 내부 구매 데이터 + 상품 수요 데이터로 사용자 분류
2. 목표 ROAS 달성 가능성이 높은 사용자/키워드/시간대 식별
3. 해당 조건에서의 입찰가를 자동 결정
4. 차순위 입찰가 + 10원으로 과금

**입찰가에 영향 주는 요소**:
- 목표 ROAS (높을수록 → 낮은 입찰가 / 낮을수록 → 높은 입찰가)
- 일예산 (높을수록 → 더 많은 구좌에 입찰)
- 상품의 전환율 이력
- 키워드 경쟁 강도
- 시간대별 전환율

### 3.2 목표 ROAS 변경 시 실제로 일어나는 일

#### 목표 ROAS를 올리면 (예: 300% → 500%)

```
목표 ROAS 상향
  → AI가 "이 ROAS를 달성하려면 더 효율적인 구좌만 써야 함" 판단
    → 입찰가를 낮춤 (저렴한 구좌에만 노출)
      → 노출 감소 (비싼 시간대/키워드 포기)
        → 구매 확률 높은 소수에게만 노출
          → 일예산 미소진 가능성 ↑
```

**구체적 변화**:
- CPC 하락 (저렴한 구좌로 이동)
- 노출수 감소
- 클릭수 감소
- 전환율은 올라갈 수 있으나 전체 매출은 감소할 수 있음
- **500% 이상 설정 시 노출 급감 사례 빈번**

#### 목표 ROAS를 내리면 (예: 500% → 200%)

```
목표 ROAS 하향
  → AI가 "효율이 낮아도 괜찮으니 더 넓게 노출" 판단
    → 입찰가를 높임 (비싼 구좌에도 입찰)
      → 노출 증가 (프라임 시간대/메인 키워드)
        → 검색 결과 상위 노출
          → 일예산 빠르게 소진
```

**구체적 변화**:
- CPC 상승 (경쟁 높은 구좌 진입)
- 노출수 증가
- 클릭수 증가
- 전환율은 떨어질 수 있으나 전체 매출은 증가할 수 있음

> **결론: 목표 ROAS를 올리는 것은 "더 비싼 자리에 입찰"이 아니라 "소진을 줄이면서 효율적 구좌에만 노출"하는 것**

### 3.3 키워드 제어

| 항목 | 가능 여부 | 상세 |
|:--|:--|:--|
| 키워드 직접 추가 | **불가** | AI가 상품명/카테고리/태그 기반 자동 매칭 |
| 키워드 입찰가 수동 조정 | **불가** | AI가 자동 결정 |
| 제외 키워드 설정 | **가능** | 비효율 키워드 제외 가능 |
| 키워드 보고서 | **가능** | 키워드별 노출/클릭/전환/ROAS 확인 |

**제외 키워드 활용 전략**:
- 키워드 보고서에서 클릭은 있으나 전환 없는 키워드 식별 → 제외
- 메인 키워드에서 경쟁력 없으면 과감히 제외 → 수동광고에서 낮은 입찰가로 재운영
- 상품과 관련 없는 키워드가 매칭되었을 때 즉시 제외

### 3.4 비검색/검색 비중 조절

- **비검색영역 OFF**: **불가능** ⚠️
- 매출최적화 광고는 **무조건 비검색영역에 노출**됨
- 비검색영역 입찰가 별도 조절도 불가
- 비검색영역 효율이 낮은 상품에서는 구조적 손실 발생

> **대안**: 비검색영역 제어가 필요하면 **수동성과형 광고**로 전환해야 함

### 3.5 상품 단위 vs 캠페인 단위 세팅

- **캠페인 = 상품 1:1 매칭 권장**: "가장 인기 있는 옵션 한 가지를 선택하여 캠페인 안에 한 가지 상품만"
- 여러 상품을 하나의 캠페인에 넣으면 AI가 예산을 특정 상품에 편중 배분할 수 있음
- **그룹화 기능**: 프리미엄/가성비/신제품 등으로 그룹 나누어 전략 분리 가능
- **상품(SKU) 단위 ON/OFF**: 가능 — 광고비 > 매출인 상품을 주기적으로 OFF

### 3.6 학습 기간 및 초기 세팅

- **학습 기간**: 2~4주 (7~14일 최적화 + 안정화)
- **초기 권장 설정**:
  - 일예산: 3~5만원 이상
  - 목표 ROAS: 300~350% (보수적 시작)
  - 너무 높게(500%+) 시작하면 노출 자체가 안 됨
- **학습 중 변경 금지**: 처음 2주는 예산/ROAS 변경 자제
- **점진적 조정**: 데이터 축적 후 ROAS를 10~20% 단위로 점진 상향

### 3.7 일예산 소진 방식

- 일예산은 하루 동안의 최대 광고비 상한
- AI가 시간대별 전환율을 고려하여 예산을 분배
- **일예산 + 높은 ROAS**: 저렴한 시간대에만 노출, 예산 미소진 가능
- **일예산 + 낮은 ROAS**: 프라임 시간대에도 노출, 예산 빠르게 소진
- 예산 소진율이 10% 미만이면 → ROAS 목표를 낮추거나 일예산을 줄여야 함
- 예산이 매일 100% 소진되면 → 일예산 증액 또는 ROAS 상향 고려

## 4. 캠페인 전략

### 4.1 동시 운영 시 카니발라이제이션 방지

**쿠팡의 자체 방지 메커니즘**:
- AI스마트광고에서 다른 캠페인(매출최적화/수동)에서 이미 운영 중인 **중복 상품은 자동 제외**
- 따라서 같은 상품이 AI스마트 + 매출최적화에 동시 노출되는 일은 시스템상 방지됨

**실전 운영 방식**:
```
전체 상품 → AI스마트광고 (자동 커버)
주력 상품 → 매출최적화 or 수동 (별도 캠페인)
→ 주력 상품은 AI스마트에서 자동 제외됨
```

**주의사항**:
- AI스마트 + 수동광고에서 같은 키워드를 사용하면 키워드 카니발라이제이션 가능 (미확인 — 쿠팡 공식 문서에 명시 없음)
- 5종 동시 운영 시 예산 분산으로 효과 추적 어려움

### 4.2 AI스마트에서 매출최적화로 분리할 때 베스트 프랙티스

**분리 판단 기준**:
- AI스마트에서 특정 상품의 예산 비중이 급증 (AI가 밀어주는 신호)
- 해당 상품의 ROAS가 안정적 (300%+)
- 더 공격적으로 밀고 싶은 주력 상품

**분리 절차**:
1. AI스마트 키워드 보고서에서 해당 상품의 전환 키워드 확인
2. 매출최적화 캠페인 신규 생성 (상품 1개)
3. 목표 ROAS = AI스마트 실적 ROAS 기준 ±30%로 설정
4. 일예산 = AI스마트에서 해당 상품이 소진하던 예산 기준
5. AI스마트에서 해당 상품은 자동 제외됨 (중복 방지)
6. 2주 학습 후 성과 비교

**주의**: AI스마트에서 떼어낸 직후 학습 리셋이 발생하므로 초기 2주는 변동 감내

### 4.3 목표 ROAS 올리기 = "더 비싼 자리" vs "소진 줄이기"

**정답: "소진을 줄이는" 방향** ✅

```
목표 ROAS ↑ → AI가 "효율 높은 곳만" 선택 → CPC ↓ → 노출 ↓ → 소진 ↓
목표 ROAS ↓ → AI가 "넓게 노출" → CPC ↑ → 노출 ↑ → 소진 ↑
```

- 목표 ROAS를 올린다 = "더 비싼 자리에 입찰"이 **아님**
- 목표 ROAS를 올린다 = "효율 낮은 구좌를 포기하고 저렴하면서 전환 높은 곳만 공략"
- 결과적으로 CPC 하락, 노출 감소, 예산 미소진 발생 가능

### 4.4 신제품 런칭 시 캠페인 유형 선택

| 단계 | 권장 유형 | 이유 |
|:--|:--|:--|
| 출시 직후 (1~2주) | **매출스타트** | 노출 최대화, 예산 소진형으로 초기 데이터 확보 |
| 데이터 축적기 (2~4주) | **매출최적화** (ROAS 200~300%) | AI 학습용 전환 데이터 수집 |
| 안정기 (4주+) | **매출최적화** (ROAS 점진 상향) or **수동성과형** | 효율 중심 최적화 |

**단가별 전략**:
- 1만원 이하: 매출최적화 우선 + 매출스타트 보조
- 1만~3만원: 매출최적화 우선 + 수동성과형 보조
- 3만~10만원: 수동성과형 우선 + 매출최적화 보조
- 10만원+: 수동성과형 + 신규고객타겟팅

### 4.5 이중 캠페인 구조 (실전 추천)

```
주력 상품 1개당:
├── 매출최적화 캠페인 (메인) — 광범위 고객 타겟
│   - 목표 ROAS: 250~350%
│   - 일예산: 3~10만원
│   - 비효율 키워드 → 제외키워드로 이동
│
└── 수동성과형 캠페인 (보조) — 세부 키워드 정밀 타격
    - 입찰가: 100~180원
    - 일예산: 1~5만원 (자투리 예산)
    - 매출최적화에서 발견된 고효율 키워드 직접 운영
    - 비검색영역 입찰가 최소화
```

### 5.3 광고 유형 추가/변경

- **매출스타트 광고**: 예산을 최대한 소진하며 노출량을 극대화하는 유형. ROAS 설정 불필요. 신제품 초기 노출에 특화.
- **신규고객타겟팅 광고**: 기존에 내 상품을 구매하지 않은 신규 고객 확보 목적. 브랜드 인지도 확장에 적합.
- **오디언스 플러스**: 상품광고 + 신규고객타겟팅의 부가기능으로, 쿠팡 외부(유튜브/메타/검색사이트)까지 광고 노출 확장.

## 6. 실전 FAQ

### Q1. AI스마트광고에서 특정 상품만 광고하고 싶은데?
**A**: AI스마트광고는 상품 직접 선택이 불가. 대안으로:
1. 원하지 않는 상품은 "상품 제외" 기능으로 제거
2. 주력 상품은 매출최적화로 별도 캠페인 생성 → AI스마트에서 자동 제외됨

### Q2. 매출최적화에서 비검색영역 노출을 끄고 싶은데?
**A**: 불가능. 매출최적화는 무조건 비검색영역에 노출됨.
- **대안**: 수동성과형 광고로 전환하여 비검색영역 입찰가를 최소로 설정

### Q3. 목표 ROAS를 얼마로 설정해야 하나?
**A**: 마진율 기준으로 역산:
- 마진율 30%+ → 목표 ROAS 300%
- 마진율 20% → 목표 ROAS 400%
- 마진율 15% 이하 → 목표 ROAS 500%+
- **초기 시작**: 300~350% 권장 → 2주 후 데이터 보고 점진적 상향

### Q4. 목표 ROAS를 500% 이상으로 설정하면?
**A**: 노출 급감 사례 빈번. AI가 해당 ROAS 달성이 어렵다고 판단하면 입찰 자체를 포기.
- 예산 소진율 10% 미만이면 ROAS가 너무 높다는 신호
- **권장**: 300%에서 시작, 예산이 100% 소진되면 50% 단위로 상향

### Q5. AI스마트와 매출최적화를 동시에 쓰면 같은 상품이 중복 과금되나?
**A**: 아니오. 매출최적화에 등록된 상품은 AI스마트에서 자동 제외됨. 중복 노출/과금 없음.

### Q6. 학습 기간 중에 성과가 안 나오면 꺼야 하나?
**A**: 아니오. 2~4주 학습 기간은 AI가 데이터를 쌓는 투자 기간.
- 학습 중 캠페인 ON/OFF 반복 → 학습 리셋 (최악)
- 학습 중 예산/ROAS 변경 → 학습 방해
- **원칙**: 처음 2주는 건드리지 말 것. 일예산 3만원+ 유지.

### Q7. 매출최적화 제외키워드를 어떻게 활용하나?
**A**: 키워드 보고서 분석 후:
1. 클릭 높으나 전환 0인 키워드 → 제외
2. 상품과 관련 없는 키워드 → 제외
3. 메인 키워드 경쟁력 없을 때 → 매출최적화에서 제외 + 수동광고에서 낮은 입찰가로 재운영

### Q8. ROAS가 비정상적으로 높게 나오면? (1000%+)
**A**: 오가닉 어트리뷰션 문제 의심.
- 클릭 0인데 주문이 잡힌 건 → 오가닉 매출이 광고에 귀속
- 비검색영역 CVR이 100%+ → 물리적 불가능, 오가닉 포함 확실
- **(우리 경험)**: 바이올렛 머스크 ROAS 1,118% 중 실질 광고 기여는 142%

### Q9. 같은 상품을 매출최적화 + 수동 동시 운영해도 되나?
**A**: 가능하며 추천되는 전략 (이중 캠페인 구조).
- 매출최적화: 광범위 자동 노출 (메인)
- 수동: 핵심 키워드 정밀 타격 (보조)
- 중복 키워드가 있으면 쿠팡 내부적으로 더 높은 입찰가를 가진 쪽이 노출

### Q10. 평일/주말 ROAS가 다르면?
**A**: 매출최적화에서는 시간대별 제어 불가.
- **대안 1**: 주말에 목표 ROAS를 올려서 노출 억제 (수동으로 변경해야 함)
- **대안 2**: 수동광고에서 시간대별 입찰가 조정
- **대안 3**: 자동 규칙 활용 — 목표 ROAS 달성 시 자동 예산 증액

### 채움컴퍼니 AI스마트광고 운영

현재 입테이프 + 얼룩제거제를 AI스마트에서 운영 중. 베이비크림은 5/15 매출최적화로 분리됨.

**AI 예산 배분 신호 해석**:
- AI가 예산을 집중시키는 상품 = 전환 잠재력 높다고 AI가 판단
- 예산 비중 줄어들면서 ROAS 올라가는 상품 = 소규모 예산이 적절

### 캡슐표백제 목표ROAS 270→300 상향 (5/16)

- 목표 ROAS 상향 = CPC 하락 + 노출 감소 방향
- 5/19 체크포인트에서 확인할 것: CPC 변화, 노출수 변화, 예산 소진율
- 예산 소진율이 급감하면 300%이 너무 높은 것 → 280%으로 재조정 고려

### 코튼블루 매출최적화 (목표ROAS 340%)

- 340%는 적정 범위 내 (300~500% 권장 존)
- CVR 하락이 지속되면 → 키워드 보고서에서 비효율 키워드 제외 점검
- 비검색영역 비중이 높고 전환 없으면 → 수동광고로 구조 변경 검토
