# 성장 전략 지침 — 비코어랩

5명의 검증된 이커머스 경영인에게서 추출한 성장 전략 플레이북이에요.

---

## D2C 성장 로드맵 (5명 공통 패턴)

```
Phase 1: 제품-시장 핏 (PMF)
  → 대표가 직접 써보고, 고객 반응 확인
  → 부정적 리뷰가 최고의 피드백

Phase 2: 자사몰/D2C 기반 구축
  → 플랫폼 수수료 절약 → R&D 재투자
  → 충성고객 + 데이터 확보

Phase 3: 콘텐츠 마케팅으로 성장
  → SNS 바이럴, 체험 콘텐츠
  → 광고보다 콘텐츠가 먼저

Phase 4: 카테고리/채널 확장
  → 검증된 제품력으로 카테고리 확대
  → 온라인 인지도 → 오프라인 확장

Phase 5: 브랜드 → 플랫폼 진화
  → 제품 판매 → 브랜드 → 콘텐츠 → 플랫폼
```

---

## 경영인별 핵심 전략

### 김병훈 (에이피알) — 미디어커머스 + 공격적 투자
- 매출의 20%, 연 1,000억+ 광고선전비 투자
- SNS(틱톡/인스타) 바이럴 마케팅 적극 활용
- D2C 자사몰로 데이터 확보 → 해외 확장
- **비코어랩 적용**: 광고비 비율은 규모에 맞게 조절하되, SNS 콘텐츠 투자는 참고

### 이수연 (젝시믹스) — 자사몰 90% + 극한 품질
- 플랫폼 수수료(20~30%) 절약 → R&D 재투자 선순환
- 대표가 모든 제품 직접 테스트, 100번 수정
- 부정적 리뷰 적극 수집 → 즉시 제품 개선
- **비코어랩 적용**: 자사몰 비중 확대 전략, 고객 리뷰 기반 제품 개선 사이클

### 신애련 (안다르) — 발품 영업 + 초고속 회전
- 전국 5,600곳 직접 전화+방문 판매
- 매달 120종 신제품, 초도 물량 2~3주 내 소진
- 소비자 불편에서 출발하는 제품 개발
- **비코어랩 적용**: 소모품 특성 활용한 빠른 제품 회전, 오프라인 접점 확대 가능성

### 남대광 (블랭크코퍼레이션) — 콘텐츠 커머스
- 일반인 체험 영상 → SNS 바이럴 → 자사몰 전환 = "디지털 방문판매"
- 경쟁사와 싸우지 않고 소비자 설득에만 집중
- 직원 인센티브: 지분 증여, 적금 대납 → 퇴사율 9%
- **비코어랩 적용**: 건조기 시트 등 체험형 콘텐츠 제작, 사용 전/후 비교 영상

### 김슬아 (컬리) — 속도보다 방향 + 데이터
- '좋은 상품만 판다' 원칙 10년 고수
- 대표 직접 참석 상품위원회, 70개 항목 평가
- 빅데이터+ML 자체 알고리즘으로 수요 예측
- **비코어랩 적용**: 제품 품질 기준 명확화, 데이터 기반 발주/재고 관리

## 비코어랩에 가장 참고할 모델

### 1순위: 블랭크코퍼레이션 (남대광)
- **이유**: 생활용품 + 콘텐츠 커머스 + 3년 1,000억 성장
- 마약베개, 바디럽 등 일상 소비재로 성공
- 일비아 건조기 시트/세제류와 카테고리 유사

### 2순위: 젝시믹스 (이수연)
- **이유**: 자사몰 90% D2C 전략 + 재구매율 90%
- 소모품 재구매 중심 비즈니스와 구조 유사
- 리뷰 기반 제품 개선 사이클 참고

### 3순위: 안다르 (신애련)
- **이유**: 자본금 2천만 원 → 4년 400억
- 중소기업 성장 초기 단계의 현실적 롤모델
- 발품 영업과 빠른 제품 회전 전략
