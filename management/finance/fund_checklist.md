# 자금 관리 체크리스트 — 비코어랩

대표님의 자금 관리 루틴용 체크리스트예요. 주기별로 확인하세요.

---

## 매일 체크

- [ ] 통장 잔액 확인 (전일 대비 변동)
- [ ] 오늘 입금 / 출금 예정 확인
- [ ] 주문 / 반품 / 환불 현황 확인

---

## 매주 체크

- [ ] 채널별 정산 예정 금액 확인 (쿠팡/네이버/자사몰 등)
- [ ] 광고비 집행 현황 vs 매출 대비 비율
- [ ] 재고 수량 및 발주 필요 여부
- [ ] 미수금 / 미지급금 현황

---

## 매월 체크

- [ ] 월간 손익 정산 (매출 - 원가 - 수수료 - 광고비 - 물류비 - 인건비 - 기타)
- [ ] 채널별 / 제품별 공헌이익 분석
- [ ] 현금흐름표 작성 (영업 / 투자 / 재무 활동별)
- [ ] CAC, LTV, ROAS 추이 분석
- [ ] 재고 회전율 점검 (과잉재고 / 부족재고)
- [ ] 세금 납부 일정 확인 (부가세, 원천세 등)

---

## 분기별 체크

- [ ] 분기 손익 vs 목표 달성률 비교
- [ ] 현금 보유량 vs 향후 3개월 예상 지출
- [ ] 제품별 수익성 재평가 (공헌이익 마이너스 제품 정리)
- [ ] 자금 조달 필요성 검토
- [ ] 세무 신고 일정 관리

---

## 위험 신호 (이것만은 놓치지 마세요)

| 신호 | 의미 | 대응 |
|------|------|------|
| 순 현금 포지션 < 월 고정비 1.5배 | 1.5개월 내 자금 부족 가능 | 즉시 자금 조달 검토 |
| 공헌이익 마이너스 제품 존재 | 팔수록 손해 | 가격 인상 or 광고 중단 or 단종 |
| 재고 회전율 급락 | 현금이 재고에 묶임 | 할인 판매 or 발주 중단 |
| 광고비 / 매출 비율 30% 초과 | 마케팅 효율 저하 | 채널별 ROAS + 공헌이익 재점검 |
| 특정 채널 정산 지연 | 현금흐름 차질 | 다른 채널 비중 확대 검토 |

## 자금 관리 원칙 (전문가 종합)

1. **현금이 왕이다** — 이익보다 현금 보유가 생존을 결정 (김승호)
2. **운전자본을 관리하라** — 재고+외상+선납금이 현금을 잡아먹는다 (강대준)
3. **고정비를 경계하라** — 불황기에는 고정비가 기업을 죽인다 (김승호)
4. **숫자로 판단하라** — 감이 아닌 데이터 기반 의사결정 (최종학)
5. **먼저 받고 나중에 줘라** — 정산 주기와 매입 지급 주기를 관리 (강대준)
