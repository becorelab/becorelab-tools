# 손익 분석 지침 — 비코어랩

이커머스 손익 분석 시 이 지침을 따라 분석해요.

---

## 이커머스 손익 계산 구조

```
총 매출액 (GMV)
  - 할인 / 쿠폰 / 적립금
  ─────────────────────
  = 순매출액
  - 매출원가 (제품 원가 + 물류/포장비)
  ─────────────────────
  = 매출총이익 (GP)
  - 판매수수료 (채널별)
  - 광고 / 마케팅비
  - 배송비 (택배비)
  ─────────────────────
  = 공헌이익 (CM) ← 1차 수익성 판단 기준
  - 인건비
  - 임차료
  - 기타 고정비 (통신, 보험, 소모품 등)
  ─────────────────────
  = 영업이익 (OP)
  - 이자비용
  - 세금
  ─────────────────────
  = 순이익
```

---

## 핵심 지표 7가지

### 1. 공헌이익 (Contribution Margin)
- **계산**: 매출액 - 변동비 (원가 + 수수료 + 광고비 + 물류비)
- **의미**: 고정비를 커버하고 남는 이익. 제품/채널별로 반드시 개별 추적
- **기준**: 공헌이익이 마이너스인 제품/채널은 즉시 재검토

### 2. 공헌이익률
- **계산**: 공헌이익 / 매출액 × 100
- **목표**: 최소 20~30% 이상 유지
- **활용**: 제품 간, 채널 간 수익성 비교의 핵심 지표

### 3. CAC (고객획득비용)
- **계산**: 마케팅 총비용 / 신규 고객 수
- **활용**: 채널별 CAC 비교 (쿠팡 vs 스마트스토어 vs 자사몰)

### 4. LTV (고객생애가치)
- **계산**: 고객 1명이 평생 가져다줄 매출 총합
- **비코어랩**: 건조기 시트 같은 소모품은 재구매율이 LTV 핵심

### 5. LTV:CAC 비율
- **기준**: 최소 3:1 이상
- **판단**: 3:1 미만 → 마케팅 효율 재검토, 1:1 이하 → 즉시 중단 검토

### 6. ROAS (광고수익률)
- **계산**: 광고매출 / 광고비
- **⚠️ 주의**: 단독 사용 금지! 반드시 공헌이익과 함께 판단

### 7. 재고 회전율
- **계산**: 매출원가 / 평균 재고
- **의미**: 높을수록 현금 회전이 좋음
- **활용**: 제품별 재고 회전율 추적 → 과잉재고 조기 발견

---

## ⚠️ ROAS의 함정

> ROAS가 높아도 돈을 잃을 수 있다!

**예시:**
| 상품 | ROAS | 광고비 | 매출 | 변동비 | 공헌이익 |
|------|------|--------|------|--------|----------|
| A | 500% | 20만 | 100만 | 80만 | **20만** |
| B | 250% | 40만 | 100만 | 50만 | **50만** |

→ ROAS만 보면 A가 좋아 보이지만, 실제 남는 돈은 B가 2.5배 더 많음
→ **반드시 공헌이익과 함께 판단!**

## 비코어랩 손익 분석 프레임워크

### 분석 시 이 순서로 진행

```
1. 채널별 매출 집계 (쿠팡/네이버/자사몰/11번가/G마켓/옥션/오늘의집)
2. 제품별 매출 집계 (건조기시트/식세기세제/캡슐세제/얼룩제거제)
3. 채널×제품 매트릭스로 공헌이익 계산
4. 공헌이익 마이너스 셀 식별 → 원인 분석
5. 고정비 차감 → 영업이익
6. 전월 대비 변동 분석
```

### 제품별 분석 시 포함할 항목
- 매출액, 판매수량
- 원가율 (매출원가 / 매출액)
- 채널 수수료
- 광고비 배분
- 물류비
- 공헌이익 / 공헌이익률

### 채널별 분석 시 포함할 항목
- 채널 매출액, 매출 비중
- 채널 수수료율
- 채널별 광고비
- 채널별 공헌이익률
- 정산 주기 (현금흐름 영향)
