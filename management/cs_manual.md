## 1. CS 업무 개요

### 1-1. 일비아 CS의 역할과 목표

일비아(iLBiA)는 **"일상의 편리함을 추구하는, 가족을 사랑하는 브랜드"** 입니다.
CS 담당자는 단순히 문의에 답변하는 것이 아니라, **고객이 일비아 제품을 통해 더 편리한 일상을 만들 수 있도록 돕는 역할**을 합니다.

**CS 핵심 목표:**
- 고객 문의에 **정확하고 빠르게** 응대한다
- 불편한 상황에서는 **공감을 먼저**, 해결은 그다음
- 과장이나 허위 없이 **사실에 근거한 안내**만 한다
- 한 번의 응답으로 **고객이 필요한 정보를 충분히** 전달한다

### 1-2. 근무 시간 및 응답 목표

| 항목 | 내용 |
|------|------|
| **CS 운영 시간** | 평일 10:00 ~ 17:00 (점심 12:00~13:00) |
| **응답 목표 시간** | 운영시간 내 문의: **30분 이내** 첫 응답 |
| **피크 시간대** | 10시~16시 (전체 문의의 약 50%, 최다 시간 13시) |
| **비운영시간 문의** | 다음 영업일 오전 10시 이후 순차 응대 |

### 1-3. 사용 도구

| 도구 | 용도 |
|------|------|
| **채널톡** | 고객 상담 메인 채널 (채팅 상담) |
| **카페24 관리자** | 주문 조회, 주문 상태 확인, 배송지 변경, 취소/환불 처리 |
| **택배사 사이트** | 배송 현황 추적 (로젠택배, 롯데택배) |
| **네이버 스마트스토어 관리자** | 네이버 주문 관리 |
| **쿠팡 윙** | 쿠팡 주문 관리 |

## 2. 응대 기본 원칙

### 2-1. 톤앤매너

일비아 CS의 말투는 **"정중하지만 따뜻한"** 톤입니다.

**핵심 규칙:**
- **항상 존칭** 사용: "고객님", "~하시다", "~드리다", "~부탁드립니다"
- **이모티콘** 적절히 활용: 😊 :) ㅠㅠ 정도만 사용 (과하지 않게)
- **공감 먼저**: 불편 상황에서는 반드시 공감/사과를 먼저, 해결은 그다음
- **간결하되 충분하게**: 한 번 응답으로 고객이 필요한 정보를 모두 전달
- **해결 의지 표현**: "빠르게 처리 도와드리겠습니다", "확인하여 안내 도와드리겠습니다"

### 2-3. 절대 하면 안 되는 것

| 금지 사항 | 설명 |
|-----------|------|
| **고객 개인정보 언급** | 이름, 전화번호, 주소 등을 응답에 포함하지 않기 |
| **허위 정보 안내** | 확인되지 않은 배송 현황, 재고 상태 등 추측 금지 |
| **가격 임의 안내** | 프로모션 가격, 할인율 등은 변동 가능하므로 공식몰 확인 안내 |
| **경쟁사 비교** | 타사 제품을 직접 비교하거나 폄하하지 않기 |
| **과장 표현** | "최고", "완벽", "100%" 등 과장된 효능 주장 금지 |
| **감정적 대응** | 고객이 화나더라도 항상 침착하고 공감적으로 대응 |
| **임의 처리** | 권한 밖의 취소/환불/배송지 변경을 임의로 약속하지 않기 |

배송 관련 문의는 전체 문의의 약 **8%** 를 차지합니다.

**확인사항:**
- 주문 상태 (결제 완료 / 배송준비 / 출고 완료)

**응답 템플릿:**
```
안녕하세요 일비아입니다 :)

주문 후 영업일 기준 1~2일 이내 출고되며, 출고 후 1~2일 내 수령 가능합니다.
택배사 사정에 따라 다소 차이가 있을 수 있는 점 양해 부탁드립니다.

감사합니다 :)
```

#### 상황 B: 배송 조회 방법

**응답 템플릿:**
```

출고 완료 시 문자(알림톡)로 송장번호가 안내됩니다.
해당 택배사 홈페이지 또는 네이버 운송장 조회를 통해 배송 현황을 확인하실 수 있습니다.

감사합니다 :)
```

**확인사항:**
- 주문번호
- 현재 주문 상태 (배송준비 전인지 확인)
- 변경할 주소 정보

**응답 템플릿 (변경 가능 시):**
```
안녕하세요 일비아입니다 :)

배송준비 전이라면 배송지 변경이 가능합니다.
고객님 주문번호와 변경하실 주소 정보를 말씀해주시면 확인 후 변경 도와드리겠습니다.

감사합니다 :)
```

**응답 템플릿 (변경 불가 시):**
```

배송 완료 후 반품 접수를 통해 도와드릴 수 있습니다.

불편드려 죄송합니다.
```

```
번거로우시겠지만 받으신 상품 전체 사진 한 장 채널톡으로 전송 부탁드립니다.
```

### 3-2. 교환 / 환불 / 반품

교환/환불 관련 문의는 전체 문의의 약 **8%** 를 차지합니다.

---

#### 상황 A: 반품/환불 신청

**확인사항:**
- 주문번호
- 반품 사유 (단순 변심 / 불량 / 오배송)
- 제품 개봉 여부
- 수령일로부터 7일 이내인지

**1단계 - 주문번호 확인:**
```
안녕하세요 일비아입니다 :)

고객님 주문번호 말씀해주시면 확인하여 안내 도와드리겠습니다.
```

**2단계 - 환불 안내 템플릿 (주문 확인 후):**
```
환불 예정 금액 및 진행 절차 안내드립니다.

1️⃣ 실결제금액 : ○○,○○○원
(*고객님께서 실제 결제하신 금액)

2️⃣ 차감 금액 :
(*환불 금액에서 차감되는 금액)
- [개봉 상품명] : ○,○○○원
- 최초 배송비 : 3,000원 (*반품 배송비 무료)

➡️ 환불 예정 금액 : ○○,○○○원
(*실제 환불되는 금액, 카드 결제의 경우 부분 취소 처리)

▪️반품 제품은 번거로우시겠지만 재포장하시어 댁 앞에 놓아주시면 기사님 방문하여 회수 예정입니다. (*영업일 기준 1-2일 소요)
▪️반품 시 차감되는 최초 배송비는 상품 금액이 아닌 택배사 배송 서비스 비용입니다. 이미 상품이 고객님께 발송되면서 배송 서비스가 완료되어 택배사에 지불된 비용이기 때문에 반품 시 환불이 어려워 최초 배송비가 차감되는 점 양해 부탁드립니다.
▪️회수 제품 물류센터 입고 후 검수 진행되며 이상 없을 경우 즉시 환불 처리됩니다. (*카드 결제의 경우 카드사에 따라 실제 환불일은 상이할 수 있습니다.)
▪️반품 제품이 회수되지 않을 경우 고객센터로 연락주시면 빠르게 확인하여 처리 도와드리겠습니다.
```

> **중요:** 환불 금액 계산은 반드시 실제 주문 내역을 확인한 후 정확하게 안내해야 합니다. 금액을 임의로 추측하여 안내하지 마세요.

### 3-3. 제품별 문의

제품 문의는 전체 문의의 약 **10.1%** 를 차지합니다.
가장 문의가 많은 제품 순서: 식기세척기 세제(1,159건) > 건조기 시트(186건) > 캡슐 세제(93건) > 캡슐 표백제(9건)

---

#### 제품 1: 식기세척기 세제 (하트 타블렛) — 가장 많은 문의

**기본 제품 정보 (숙지 필수):**
- 린스 포함 올인원 타블렛
- LG, 삼성, SK, 밀레 등 모든 주요 식기세척기 호환
- 60정 / 120정 / 180정 구성
- 일반 가정용(6~8인용): 1알 / 14인용 이상 대형: 2알 권장

##### "세제를 어디에 넣어야 하나요?"

**응답 템플릿:**
```
안녕하세요 일비아입니다 :)

세제 투입구에 넣으셔도 괜찮고 식세기 바닥에 넣어서 사용하셔도 괜찮습니다.

다만, 발포 타블릿 특성상 수분이 닿으면 즉시 용해가 시작되기에
세제투입구가 완전히 건조된 상태에서만 세제 투입구를 사용하시는 것을 추천드립니다.
```

## 6. 자주 하는 실수 & 주의사항

### 6-1. 실제 데이터에서 발견된 주의점

#### 환불 금액 안내 실수
- **문제:** 환불 금액을 임의로 계산하여 안내한 후, 실제 금액과 차이가 발생
- **올바른 방법:** 반드시 주문 내역을 시스템에서 확인한 후 정확한 금액을 안내. 불확실하면 "확인 후 안내드리겠습니다"로 응대

#### 배송 상태 추측 안내
- **문제:** "내일 도착합니다", "오늘 출고됩니다" 등 확인 없이 답변
- **올바른 방법:** 실제 출고/배송 상태를 확인한 후 안내. "확인해보겠습니다"가 추측보다 낫다

#### 개봉 상품 반품 처리
- **문제:** 개봉된 상품의 반품을 접수해버리는 실수
- **올바른 방법:** 개봉 여부를 반드시 확인. 개봉 상품은 반품 불가, 자체 폐기 안내

#### 배송비 정책 혼동
- **문제:** 최초 배송비 차감(3,000원)을 안내하지 않고 "전액 환불"로 안내
- **올바른 방법:** 단순 변심 반품 시 최초 배송비 3,000원 차감됨을 반드시 안내. 불량/오배송만 전액 무료 반품

#### 채널별 정책 차이 미인지
- **문제:** 네이버/쿠팡 등 각 채널의 CS 정책이 자사몰과 다를 수 있음을 모르고 동일하게 안내
- **올바른 방법:** 각 채널별 반품/환불 정책은 해당 채널의 규정을 따름. 불확실하면 상급자에게 확인

#### 990원 이벤트 조건 오안내
- **문제:** "아무 상품이나 사시면 990원이에요"라고 안내
- **올바른 방법:** 공식몰 회원 대상, 할인 제외 실결제 3만원 이상 시 자동 적용. 비회원 미적용. 가장 할인율 높은 1개만 적용
