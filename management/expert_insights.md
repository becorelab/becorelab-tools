# 경영 전문가 인사이트

비코어랩 경영 판단 시 참고하는 검증된 경영인/전문가의 핵심 원칙.
구체적 프레임워크와 체크리스트는 `finance/`, `strategy/` 하위 가이드 파일에 있음.

---

### 이커머스 경영인 (국내 D2C/소비재)

#### 1. 김병훈 | 에이피알(APR) · 메디큐브 — 매출 1.4조, D2C 미디어커머스
- **블루오션 의심 원칙**: 시장이 비어있으면 비어있는 이유부터 파악
- **D2C 우선 → 확장**: 자사몰로 충성고객+데이터 확보 → 오프라인/해외 확장
- **품질 뒷받침 없는 광고는 무의미**: 광고대행업 경험에서 체득한 핵심
- **극한의 실행력**: "매일 14시간씩 10년을 일했을 때 돌파구가 찾아왔다"

#### 2. 이수연 | 젝시믹스(XEXYMIX) — 매출 1,726억, 자사몰 90%
- **대표가 최고의 충성고객**: 모든 신제품을 직접 테스트, 만족할 때까지 100번 수정
- **자사몰 90% D2C**: 플랫폼 수수료 절약분을 R&D에 재투자하는 선순환
- **부정적 리뷰 = 최고의 피드백**: 부정 리뷰 수집/분석 → 제품 개선 즉시 반영
- **사내 R&D 완결형**: 외부 의존 최소화, 수백 번 테스트 후 생산

#### 3. 신애련 | 안다르(ANDAR) — 4년만에 매출 400억, 자본금 2천만 원 시작
- **소비자 불편에서 출발**: 요가 강사로서 느낀 불편함을 직접 해결
- **발품 영업**: 전국 5,600곳 직접 전화+방문, 하루 1,000km
- **초고속 제품 회전**: 매달 120종 신제품, 초도 물량 2~3주 내 소진
- **특허 기반 기술력**: 제조 2배 걸려도 품질 우선

#### 4. 남대광 | 블랭크코퍼레이션 — 3년만에 매출 1,000억, 콘텐츠 커머스
- **디지털 방문판매**: 일반인 체험 영상 → SNS 바이럴 → 자사몰 전환
- **4단계 진화 모델**: 제품 판매 → 브랜드 → 콘텐츠 회사 → 플랫폼
- **경쟁사와 싸우지 않기**: 소비자 설득에만 올인
- **이타적 이기주의**: 개인지분 10% 직원 증여, 퇴사율 9% (업계 40~50%)

#### 5. 김슬아 | 컬리(Kurly) — 기업가치 4조, 2024년 첫 흑자
- **속도보다 방향**: '좋은 상품만 판다' 원칙 10년 고수
- **극한의 품질 관리**: 대표 직접 참석 상품위원회, 70개 항목 평가
- **데이터 기반 수요 예측**: 빅데이터+ML 자체 알고리즘
- **발품 기반 직접 소싱**: 전국 돌며 생산자 한 분씩 설득

> **5명의 공통 패턴**: D2C 자사몰 우선 | 대표가 직접 제품 테스트 | 고객 피드백 즉시 반영 | 콘텐츠가 곧 마케팅 | 직원 보상에 적극 투자

### 재무회계 전문가

#### 1. 강대준 회계사 | 코스토리(인사이트파트너스) — 현금흐름 전문
- **현금흐름이 매출보다 중요**: 사업 성공 = 현금을 얼마나 안정적으로 창출하느냐
- **잉여현금흐름(FCF) 중심 경영**: 영업현금흐름 - 설비투자 = 진짜 기업가치
- **운전자본 관리**: 재고/외상/선납금 통제 못 하면 고금리 단기대출 악순환
- **돈을 먼저 받는 비즈니스 모델**: 고객에게 먼저 받고 공급처에 나중에 지급

#### 2. 이재용 회계사 | B주류경제학(토스) — 비전공자 친화
- **재무제표로 기업의 미래 읽기**: 업계 흐름 → 선두 기업 재무제표로 가설 검증
- **숫자 뒤의 비즈니스 스토리**: 소비문화 이면의 경제 구조 분석
- **비전공자도 이해하는 접근법**: 일상에서 경제/회계 연결

#### 3. 최종학 교수 | 서울대 경영대학 — 숫자 기반 의사결정
- **"경영은 숫자를 다루는 과학"**: 비전도 전략도 결국 숫자로 귀결
- **적정 부채비율 계산**: 레버리지 활용의 기준점
- **회계 정보 → 전략 수립**: 숫자에서 전략적 의사결정으로 연결

#### 4. 김승호 회장 | 스노우폭스그룹 — 실전 경영자 재무 마인드
- **매출이 아니라 이익을 봐라**: 매출에 취하지 말고 실제 남는 돈을 추적
- **현금 보유의 중요성**: 위기 시 현금이 생존을 결정
- **고정비 vs 변동비 구분**: 불황기에는 고정비가 기업을 죽인다

#### 5. 회계법인 마일스톤 | 스타트업 전문 — 단계별 재무 설계
- **단계별 재무 이벤트 관리**: 설립 → 성장 → 투자 → 엑시트
- **사업자 유형 선택 = 미래 비용 결정**: 초기 세팅이 중요

### 이커머스 손익 핵심 지표 (Quick Reference)

| 지표 | 의미 | 비코어랩 적용 |
|------|------|-------------|
| **공헌이익(CM)** | 매출 - 변동비(원가+수수료+광고+물류) | 제품별/채널별 추적 필수 |
| **공헌이익률** | CM / 매출 × 100 | 최소 20~30% 유지 |
| **CAC** | 마케팅 총비용 / 신규 고객 수 | 채널별 비교 |
| **LTV** | 고객 1명의 생애 총매출 | 소모품 재구매율이 핵심 |
| **LTV:CAC** | 최소 3:1 이상 | 3:1 미만이면 마케팅 재검토 |
| **ROAS** | 광고매출/광고비 | 단독 사용 금지, 공헌이익과 함께 |
| **재고 회전율** | 매출원가/평균재고 | 높을수록 현금 회전 좋음 |

> **ROAS 함정**: ROAS 높아도 돈 잃을 수 있음. 반드시 공헌이익과 함께 판단!

### 이커머스 간편 손익 계산 구조
```
총 매출액(GMV) - 할인/쿠폰/적립금 = 순매출액
  - 매출원가(제품 원가 + 물류/포장비) = 매출총이익
  - 판매수수료(채널별) - 광고/마케팅비 - 배송비 = 공헌이익
  - 인건비 - 임차료 - 기타 고정비 = 영업이익
  - 이자비용 - 세금 = 순이익
```
