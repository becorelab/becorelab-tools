# CLAUDE.md — 비코어랩 소싱 파이프라인

## 프로젝트 개요
쿠팡 시장 분석 → 소싱 기회 발굴 → 1688 공급업체 탐색 → 견적 비교 → 발주까지의 **이커머스 소싱 풀 파이프라인**.

대표님의 소싱 업무 흐름:
1. **키워드 스캔** — 쿠팡에서 키워드별 상위 40개 상품 데이터 수집
2. **기회점수 분석** — 매출 분포, 진입 기대매출, 신상품 가중치 등으로 기회 평가
3. **리뷰 분석** — 상위 상품 리뷰에서 소비자 불만/니즈 추출
4. **상세 분석** — Claude AI로 시장 진입 전략 리포트 생성
5. **RFQ 발송** — 1688 공급업체에 견적 요청
6. **견적 비교** — 공급업체 응답 비교 → 발주 결정

## API 주요 엔드포인트

### 스캔
- `POST /api/scan/manual` — 수동 키워드 스캔 (API 모드)
- `POST /api/scan/wing` — 윙 연동 스캔 (판매량 데이터 포함)
- `GET /api/scan/<id>/poll` — 스캔 진행 상태 폴링
- `GET /api/scans` — 전체 스캔 목록

### 분석
- `GET /api/scan/<id>` — 스캔 상세 (상품 + 점수)
- `POST /api/scan/<id>/reviews` — 리뷰 수집 시작
- `POST /api/scan/<id>/detail-analysis` — AI 상세 분석
- `POST /api/scan/<id>/detail-chat` — 분석 결과 기반 대화

### 골드박스
- `POST /api/goldbox/start` — 골드박스 크롤링 시작
- `POST /api/goldbox/auto-scan` — 골드박스 상품 자동 스캔

### RFQ/견적
- `POST /api/rfq` — 견적 요청서 생성
- `POST /api/scan/<id>/rfq/generate` — 스캔 결과에서 RFQ 자동 생성
- `POST /api/quotation` — 견적 응답 등록
- `GET /api/rfq/<id>/compare` — 견적 비교

### 자동 탐색
- `POST /api/autoscan/start` — 시드 키워드에서 연관 키워드 자동 탐색
- `POST /api/autoscan/explore` — 카테고리 기반 자동 탐색
- `GET /api/opportunities` — GO 판정된 기회 상품 목록

## 소싱 전문가 인사이트 (AI 판단 가이드)

5명의 국내외 전문가 인사이트를 종합한 소싱 판단 기준. 소싱 하치는 아래 원칙을 내재화하여 대표님께 조언할 것.

> 참조 전문가: 단아쌤(김경은, 쿠팡/네이버 공식강사), Kevin King(Amazon FBA, Freedom Ticket), 이중엽·황재성(1688 실전 소싱), Bob Moesta(JTBD 프레임워크), Renaud Anjoran(Sofeast, 중국 QC/SCM)

### 1. 상품 기회 판단 — "팔리는 시장을 먼저 찾아라"

**경쟁강도 = 상품 수 ÷ 월 검색량** (단아쌤)
- 수치가 낮을수록 "찾는 사람은 많고 파는 사람은 적은" 시장
- 초보 기준: 월 검색 1,000~5,000 / 상품 수 1,000 미만이 이상적
- 대형 키워드 대신 **구매 의도 담긴 롱테일 키워드**가 실전 승률 높음

**시장 구조 분석** (Kevin King + 기존 scoring.py 기준)
- 매출 분포가 고른 시장 (상위3 독식 X)
- 상위 3개 셀러의 합산 전환율 **40% 미만** → 나머지 60%가 분산 = 진입 여지
- 300만원+ 판매자가 40~50% 이상
- 4~10등 평균매출 = 진입 시 기대매출
- 신상품 가중치 (매출/리뷰/1000) — 리뷰 적은데 잘 팔리면 수요 신호

**수익성 기준** (Kevin King)
- 순이익률 **최소 30% 이상** (플랫폼 수수료 전)
- 원가 대비 판매가 **3~5배** 마진 구조
- ROI **최소 100%**

### 2. 상품 기획 — "고객의 진보(Progress)를 팔아라" (Bob Moesta, JTBD)

**4가지 힘 분석** — 모든 상품 기획의 출발점
| 힘 | 방향 | 예시 (건조기 시트) |
|---|---|---|
| Push (현재 불편) | 구매 촉진 | "기존 섬유유연제 냄새가 너무 독해" |
| Pull (새 상품 매력) | 구매 촉진 | "무향 + 정전기 방지 동시에" |
| Habit (기존 루틴) | 구매 저항 | "원래 쓰던 거 있는데 굳이?" |
| Anxiety (새 상품 불안) | 구매 저항 | "효과 없으면 어떡하지?" |

**팔리는 상품 = Push + Pull > Habit + Anxiety**

**차별화는 "속성"이 아닌 "맥락"에서**
- 약한 차별화: "향이 좋아요" → 강한 차별화: "아기 옷에도 안심하고 쓸 수 있는 무자극 향"
- 경쟁 상품 별 1~3점 리뷰에서 **공통 불만 3가지** 추출 → 여기서 차별화 포인트 도출
- 비경쟁 구간(Non-Consumption) 공략: 사고 싶지만 현재 선택지가 없는 고객

**리뷰 분석 시 구매 패턴 역추적**
- "처음엔 별로였는데 → 냄새가 없어서 → 계속 사게 됐어요" 형태의 리뷰가 핵심 전환 패턴
- 인터뷰 10건이면 시장의 구매 패턴 3~5개를 커버 (전수조사 불필요)

### 3. 가격 전략 — "적정하게 비싼 것이 신뢰 신호"

- 시장 내 저가-중가-프리미엄 **가격 사다리** 파악 → 의도적 포지션 결정
- 너무 싼 가격은 품질 의심 유발 (소비재에서 특히)
- **소용량 입문가 전략**: 소용량 저가 SKU로 첫 경험 → 리뷰 확보 → 대용량 주력으로 전환
- FMCG 마진은 1회 판매가 아닌 **LTV(생애가치)**로 계산 — 첫 구매 손익분기 허용, 재구매에서 수익
- 번들 전략: 2+1, 묶음 판매로 객단가 올리면서 할인감 제공

### 4. 1688 공급업체 선정 — "싼 곳이 아니라 안정적인 파트너를 찾아라"

**공급업체 검증 체크리스트** (이중엽/황재성)
- 재주문율 **30% 이상** → 기존 고객 만족도 양호
- 평균 응답 시간 **6시간 이하**
- 평점 **4.7/5.0 이상**
- 정시 배송율 **99% 이상**
- 배송 보증(发货保障) 배지 → 3,000위안 예치, 미배송 시 환불
- 슈퍼팩토리: 연 매출 19억원↑, 공장 면적 1,000㎡↑

**공장(Factory) vs 무역상(Trader) 구별**
- 공장: 기계류 사진, 작업장 면적, 직원 수 50명+, 설비 목록 명시
- 무역상: "여러 공장과 장기 협력" 모호한 표현, 공장 사진 없음

**탐색 단계** (Kevin King)
```
타오바오 소량 검증 → 1688 도매 전환 → 공장 직발주 + QC 에이전트
```

### 5. RFQ/견적 비교 — "단가만 보지 마라" (Renaud Anjoran)

**견적서 필수 비교 항목**
| 항목 | 체크 포인트 |
|---|---|
| 단가 | EXW/FOB 기준, 수량 구간별 (100/300/500/1000개) |
| MOQ | 최소 발주 + 하한 협상 가능 여부 |
| 샘플비 | 비용, 수량, 납기 / 양산 시 환급 여부 |
| 납기 | 오더 확정 → 선적까지 일수 (보통 15~45일) |
| 결제조건 | T/T 비율 (30% 선금 / 70% 선적 전 권장) |
| 패키징 | 기본 vs 커스텀 추가 비용 |
| 인증 | KC, 식약처 등 해당 인증 보유 여부 |
| 불량 처리 | AQL 기준, 불량 시 재발송/환불 정책 |

**총 랜딩 코스트 공식** (표시 단가는 실제 원가의 60~70%)
```
총 원가 = 1688 단가 + 중국 내 배송비 + 포워딩비 + 보험료(1~2%)
        + 한국 관세(품목별 0~40%, 평균 8%) + 부가세 10%
        + 3PL/배대지 수수료 + 검수비 + 인증비
```

### 6. 협상 전략 — "관계 먼저, 거래 후순위"

**협상 5대 원칙** (Renaud Anjoran)
1. **관계(관시) 우선** — 첫 접촉에서 가격 압박 금지. 위챗 소통, 명절 인사
2. **성장 파트너로 포지셔닝** — "파일럿 후 분기마다 물량 증가 예정"
3. **단가 외 협상** — MOQ 완화, 납기, 결제조건, 불량 정책, 패키징 커스텀
4. **연간 총 발주량 기준 대화** — "이번 500개"가 아니라 "올해 2,000개"
5. **3곳 이상 경쟁 견적** — "다른 공급업체 견적도 받았지만 당신과 장기 거래 선호"

**소량 발주(MOQ 1,000개 이하) 전략**
- 파일럿 오더 프레임: 단가 10~20% 프리미엄 인정, 소량 생산 승인
- 색상/SKU 혼합으로 총량 맞추기: 3가지 색상 합산 500개(각 170개)
- 기존 ODM 베이스에 로고/패키지만 커스터마이징 → MOQ 100~300개 가능

### 7. 품질관리(QC) — "골든 샘플이 모든 분쟁의 기준"

**3단계 검수 프로세스** (Renaud Anjoran)
1. **샘플 단계**: 최소 2개 수령 (1개 봉인 보관 = 골든 샘플, 1개 테스트)
2. **생산 중(DUPRO)**: 생산 20~60% 시점 현장 검사 → 조기 불량 감지
3. **출하 전(PSI)**: AQL 2.5 기준 샘플링 검사, 사진/영상 증거 필수

**양산품 품질 저하 주요 원인**
- 원자재 저가 스왑 (특히 원단, 플라스틱 등급)
- 색상 드리프트 (팬톤 코드 명시해도 염료 배치 차이)
- 포장 다운그레이드 (샘플은 두꺼운 골판지, 양산은 얇은 박스)
