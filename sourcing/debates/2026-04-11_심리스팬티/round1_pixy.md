## 인증 의무 분석

### 한국 법령 체크리스트
- **전기용품 및 생활용품 안전관리법(전안법)**: 의류/속옷은 2017년 개정 이후 **안전기준준수대상 생활용품**으로 분류. KC 인증 의무는 아니나, **공급자적합성확인** 대상 가능성 — 확인 필요.
- **유아용 섬유제품**: 만 13세 이하 대상은 **안전확인 대상**. Top20에 **르브니 주니어 (초등학생 여아용)** 있음 → **KC 안전확인 필수 품목**. 주니어 라인 진입 시 리스크 높음.
- **생리팬티 (Top20 큰귀멍멍 "샘방지 위생 생리팬티")**: 흡수성 생리용품으로 분류될 경우 **의약외품 허가** 또는 **위생용품법** 적용 가능성. 흡수체 내장 시 식약처 규제 대상 → 일반 팬티로 등록 시 표시광고법 위반 리스크.
- **기능성 표시 규제**:
  - "항균", "소취" 표시 → **항균도 시험성적서 (KS K 0693 등)** 없이 기재 시 **표시광고법 위반**.
  - "쿨/아이스" 표시 → **Q-max 값** 시험 성적서 필요 (공정위 표시광고 심사지침).
  - "노라인", "무봉제" → 표현 자체는 문제없음.
- **섬유의 혼용률 표시**: 의류가공용 섬유제품은 **섬유제품 품질표시 기준 (KATS 고시)** 에 따라 소재 혼용률·취급 주의사항·제조국·제조자 표시 **의무**. 누락 시 리콜·과태료.

### 결론
일반 여성 심리스 팬티 = 인증 부담 낮음. **단, 주니어/생리팬티/항균·쿨 기능 강조 라인은 인증 장벽 높음.**

- **언더스미스** (2개): 자체 상표, 디자인 특허 보유 가능성 중간.

## 품질·반품 리스크

### 카테고리 특성상 **반품 재판매 불가**
- **전자상거래법 제17조 2항**: 속옷·수영복은 **사용 흔적이 있거나 위생상 재판매가 곤란한 경우 청약철회 제한 가능**. 하지만 **개봉만으로 반품 거절 불가** — 실제로는 "개별 위생 포장 훼손 시 반품 불가" 고지가 쿠팡에서 통용.
- **실질**: 반품 와도 우리가 **전량 폐기**. 반품률 5% = 원가 5% 통째 손실.
- **쿠팡 로켓 입점 시 반품률 상한 초과**하면 노출 강등 → 치명적.

### 클레임 패턴 (일반 의류와 다른 점)
1. **사이즈 불만**: Top20 평균 세트 구성 5~10매. 사이즈 안 맞으면 전량 반품 → **손실 배수 증가**.
2. **피부 트러블**: 속옷 직접 접촉 → **두드러기·가려움 클레임 = PL(제조물책임) 소송 가능성**. 면책 어려움.
3. **냄새/탈색**: 포장 시 섬유유연제·잔류 화학물질 냄새 민감. 중국 공장에서 흔한 이슈.
4. **봉제 풀림**: "무봉제"로 판매했는데 본딩 접합부 박리 → **허위 표시 공격**.

### 결론
평균 반품률이 일반 의류(15%)보다 낮아 보이지만, **반품 = 폐기**라는 점에서 **실질 손실률은 2~3배**.

## 플랫폼 제재 리스크

- **쿠팡**: 속옷은 성인 카테고리 인접 → **이미지 심사 엄격**. 모델컷에 과도한 노출 시 상품 숨김.
- **네이버**: "속옷" 키워드 검색 시 19금 필터 미적용이지만, 상세페이지에 피팅 사진 과도 시 **검색 노출 배제**.
- **표시광고 집중 단속 품목**: 공정위 매년 속옷·언더웨어 모니터링 대상. "항균 99%", "탈취", "다이어트 효과" 표기 시 **제재 1순위**.
- **의료기기/의약외품 오인 표시**: "생리 누수 방지", "요실금 케어" 표기는 **식약처 단속 대상**.

## 🔍 카테고리 특수 변수 (내가 발견한 것)

### 1. **"반품=폐기" 구조 → 원가 리스크 배수**
속옷은 반품돼도 재판매 불가. 일반 의류는 반품 후 재입고 가능. 이 차이가 ROI 계산에 결정적. **반품률 5%여도 실질 손실률은 5%**. 일반 의류 대비 체감 리스크 2배.

### 2. **사이즈 폭 강제 → 재고 복잡도 폭증**
AI 분석이 권장한 M~3XL 6단계 × 5색 = **30 SKU**. 일반 의류보다 사이즈 폭이 넓어야 함 (빅사이즈 수요 큼). 초도 발주 금액 급증 → **사장 재고 리스크**.

### 3. **생리팬티/주니어 라인 법적 구분**
Top20에 **주니어(르브니)** 와 **생리팬티(큰귀멍멍)** 혼재. 이 둘은 **완전히 다른 인증 체계**:
- 주니어 = **KC 안전확인 의무** (만 13세 이하)
- 생리팬티 = **흡수체 내장 시 의약외품/위생용품 규제**
무지하게 진입하면 등록 거부·판매 정지. **일반 성인 여성용 한 가지만 다뤄야 안전.**

### 4. **사진 촬영/모델 비용 숨은 원가**
속옷은 **모델컷 없이 못 팖**. 전문 란제리 모델 + 스튜디오 = 최소 200~500만 원. 일반 잡화 대비 콘텐츠 원가 2~3배. 브랜드 무명이면 **모델 섭외 자체가 어려움**.

### 5. **PL 보험 가입 사실상 필수**
피부 직접 접촉 제품 → **제조물책임보험** 미가입 시 클레임 1건으로 분기 이익 증발 가능. 보험료 연 수백만 원 → 고정비 증가.

### 6. **"심리스" 자체가 공정 기술 의존**
일반 봉제 공장에서는 못 만듦. **3D 홀가먼트 니팅 또는 라미네이트 본딩 설비** 보유 공장 필수. 공장 풀이 좁음 → 가격 협상력 낮음, MOQ 높음.

### 7. **계절성 없음은 장점, 단 경쟁 연중 치열**
경량패딩·잠옷 같은 시즌 변수 없음 → 플러스. 그러나 **신제품 비율 17.5%** = 시장 진입이 활발하지 않다는 뜻 = 기존 강자 진입장벽 높음을 간접 시사.

## 핵심 결론

### 리스크 관점에서 진행할 만한가?
**조건부 부정적.** 카테고리 자체가 iLBiA(생활용품/세탁제) 본업과 거리가 멀고, 리스크 변수가 5개 이상 중첩. 본업 확장이 아니라 **순수 커머스 도전**이 됨.

### 진행한다면 집중 대응 포인트
1. **반드시 성인 여성용 기본형 1개 라인만** (주니어·생리팬티·남성 제외)
2. **모델/니트 공장 선정 전 MOQ·샘플 3회 이상 검증** → 봉제 풀림, 냄새 이슈 사전 차단
3. **"항균·쿨·다이어트" 기능성 문구 일절 금지**, 시험성적서 없이는 "노라인·무봉제"만 주장
4. **개별 위생 포장 필수** + 상세페이지에 "개봉 시 반품 불가" 명시 (전자상거래법 허용 범위)
5. **PL 보험 사전 가입** + 혼용률/제조국/세탁 주의 표시 품질표시기준 준수
6. **초도 물량 최소화** — 사이즈·컬러 SKU 폭증 주의, M/L/XL 3단계 × 3색부터 테스트

## 점수 근거 한 줄
**반품=폐기 구조 + 사이즈 SKU 폭증 + PL 리스크 + 본업과 무관 → 리스크 대비 기회가 크지 않은 카테고리. 4점.**
