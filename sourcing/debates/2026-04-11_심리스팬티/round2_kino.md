# 키노 Round 2

## 다른 인사이트 수용
- 픽시 "반품=100% 폐기" + PL보험 연 수백만 원 → 내 원가표에 **폐기비 300~500원/매 + 보험 고정비** 재확인. 손익 악화 방향.
- 보리 "리뷰 1개 가치 2~3배 + 사이즈 락인 LTV 낮음" → CAC 회수 기간 1.5~2배 길어짐. 광고 ROAS 더 나빠짐.
- 모리 "롱테일 월 1,500~2,500만" → 이 매출대에선 고정비(모델/인증/PL) 분할 회수 자체가 불가.

## 모리 조건부 시나리오 마진 검증
임산부/생리팬티 + 3사이즈·2컬러 + 프리미엄가(5매 29,900원, 1매 5,980원) 가정:
- 알리 원가 1,200원 + 위생포장 200원 + 관부가세/물류 400원 = **착지 1,800원/매**
- 매당 판가 5,980원 → GP 4,180원(70%). 여기서 쿠팡 수수료 11%(658) + 광고 30%(1,794, ROAS 3.3) + 폐기 예비 400 + PL/모델 상각 500 = **실마진 828원/매 = 13.8%**.
- **40% 불가**. 좁은 SKU로 재고비는 줄지만 광고비(광고의존도 175) + 카테고리 구조비(PL/인증/모델)가 먹어버림. 프리미엄가 올려도 기능성 팬티는 5매 35,000원이 심리 저항선 — 더 못 올림.
- 유일한 40% 경로 = 자사몰 직판 + 구독(LTV 3회↑). 쿠팡 진입 시 절대 불가.

## 점수 조정: 3 → 3 (유지)
조건부 시나리오로도 마진 40% 달성 불가 확인. 오히려 좁은 SKU는 **재고 회전↓ → 캐시 회수 지연** 부작용.

## 한 줄 결론
쿠팡 진입 = 마진 0~14% 확정, 자사몰 구독 외엔 40% 경로 없음. **비추 유지**.
