# 픽시 🦋 — 리스크 평가 (Round 1)

## 1. 부분 가발
- 점수: **4/10** (위험도 중상)
- **인증 필요 여부**:
  - 일반 헤어피스/집게가발(비전기, 비화학처리, 성인용): KC 의무 인증 대상 아님 — 공산품.
  - **단, 다음 경우 리스크 발생**:
    - 인모(human hair) 수입 시 → CITES/검역 이슈는 없으나, 원산지·위생처리 증빙 필요(식약처 대상은 아님).
    - 어린이 겨냥(공주 가발, 코스프레) → **어린이제품 공통안전기준(유해원소 8종, 프탈레이트)** 대상 → KC 안전확인 필수.
    - 염색·코팅 처리한 인조모 → 중금속(납, 카드뮴) 이슈 가능.
  - **"항암가발"** 키워드 검색량 3,860건 확인 → 의료 목적 표방 시 **의료기기 오인 광고** 리스크.
- **IP 리스크**: **높음**
  - Top15 중 브랜드 상품이 11개(SEVOREN, 미제르바, 나나민, 샤일리, 지니판, 에어릴리, 더창, 끌레오라미 등) → **레드오션, 디자인 카피 천지**.
  - 특히 "시스루뱅", "C컬 레이어드", "정수리커버" 같은 형태는 이미 브랜드들이 디자인 출원 가능성 있음 — **확인 필요**.
  - 알리바바 무브랜드 소싱 → 국내 브랜드와 동일 형태일 가능성 매우 높음.
- **품질/클레임 리스크**: **매우 높음**
  - 가발은 반품률·클레임률이 패션잡화 중 최상위 카테고리. 색상 차이, 냄새(화학처리), 탈색, 엉킴, 두피 알레르기.
  - 정수리 가발은 "자연스럽지 않다" 클레임 多 → 네이버 리뷰 별점 관리 지옥.
  - 50~60대 타깃 → 환불 요구 강하고 플랫폼 중재 많음.
- **근거**: 검색 키워드에 "항암가발", "남자가발" 섞여 있어 의료 오인 영역 진입 가능성 + 브랜드 15개 중 11개가 이미 자리잡음 = 후발주자가 들어가면 IP/리뷰 모두 불리.

## 2. 옷 압축팩
- 점수: **9/10** (가장 안전)
- **인증 필요 여부**:
  - **KC 인증 의무 없음**. 전기제품 아님, 식품접촉 아님, 어린이제품 아님.
  - PE/PA 필름 재질 → 환경부 자원재활용법상 포장재 분리배출 표시 정도만 필요.
  - 펌프 포함 제품도 수동 펌프는 비전기 → 별도 인증 불요.
  - **불확실 — 확인 필요**: 진공 흡입구에 밸브 금속/고무 부품 있을 경우 RoHS 수준 확인 권장(의무 아님).
- **IP 리스크**: **낮음~중간**
  - Top15 브랜드 분산도 높고, 기능성 특허라기보다 규격(사이즈, 옷걸이 후크) 차이. 디자인권 리스크 낮음.
  - "옷걸이형 압축파우치" 구조는 **LG생활건강, 3M, 이케아** 등 대기업 진입 이력 있어 — 대형 브랜드 특허 존재 가능성 있으나, 중국 OEM 수준 범용 구조는 공용 영역(public domain).
  - 주의: **"진공 펌프 + 일방향 밸브"** 특정 구조는 국내 실용신안 등록 사례 있음 → 밸브 구조 카피는 **확인 필요**.
- **품질/클레임 리스크**: **낮음**
  - 구조 단순, 기능 명확(부피 압축). 클레임 포인트는 "기밀 불량(공기 새어나옴)" 1가지 → QC만 잘하면 해결.
  - 과대광고 리스크도 "40% 압축"처럼 수치 명시 가능 → 실측하면 방어 가능.
  - 반품률 낮은 카테고리(재사용 가능 소모품, 배송 중 파손 적음).
- **근거**: 카테고리 자체가 규제 청정지대. 광고의존도 152 중간, 신제품비율 60%로 신규 진입자 많으나 특허 장벽 낮아 누구나 들어오는 구조 = **안전하지만 차별화는 다른 팀원 영역**.

## 3. 허리베개
- 점수: **5/10** (중간 — 표현 규제 주의)
- **인증 필요 여부**:
  - 섬유 커버 + 메모리폼/우레탄 충전재 → **KC 안전확인 대상 아닌 가정용 섬유제품**이나, 커버 소재에 따라 **전기용품및생활용품안전관리법(가정용 섬유제품 안전기준)** 적용 가능 → pH, 폼알데하이드, 아릴아민계 염료 기준 **자율안전확인(KC)** 대상일 수 있음. **확인 필요**.
  - 어린이용 베개로 표시하면 → **어린이제품공통안전기준** KC 필수.
  - 메모리폼 충전재는 VOC(휘발성유기화합물) 냄새 이슈 → 법적 규제는 없으나 클레임 유발.
- **IP 리스크**: **중간~높음**
  - Top15 중 브랜드 다수: 닥터센스, 올곧, 히어로인라이프, 라모노, 허쉬베개, 스탠바이미, 굿코어, 마메, 씨가드 등 13/15 브랜드.
  - **올곧 벨트형 허리베개 20만원, 월매출 5,500만원** → 국내 브랜드 확실히 자리잡음. 카피 시 분쟁 가능성.
  - "스탠바이미" = 국내 LG 브랜드명과 동일 → **상표권 검토 필수**. LG전자 "StanbyME" 상표권 등록되어 있어 혼동 유발 표기 금지.
  - C커브, 요추 형태 → 의료기기 특허는 없으나 디자인권은 개별 확인 필요.
- **품질/클레임 리스크**: **매우 높음 — 법적 리스크**
  - ★★★ **가장 큰 위험**: "**디스크**", "**요추**", "**바른자세 교정**", "**회복**" = 모두 **의료기기 오인 광고**.
  - Top15 상품명에 "디스크", "요추", "회복", "기능성", "바른자세" 다 들어가 있음 → **의료기기법 제26조 위반** 리스크. 식약처·공정위 단속 대상.
  - 플랫폼 제재 사례 多 — 네이버/쿠팡 상품 강제 삭제, 광고 반려, 이미 판매된 제품 전수 리콜 사례 존재.
  - 허쉬베개 리뷰 2,088개 / 스탠바이미 4,854개 → 후기 쌓인 브랜드가 레퍼런스 독점, 후발 주자 신뢰 획득 어려움.
- **근거**: 의료기기 오인 광고 단속 사례가 2024~2025년에 급증. 허리·경추 카테고리가 가장 많이 걸리는 영역. 상세페이지·광고 문구 전수 검수 필요하고, **"디스크 완화"** 같은 문구는 아예 못 씀.

## 내가 반대하는 후보
- **후보: 허리베개**
- **이유**:
  1. **의료기기 오인 광고 리스크가 구조적**이다. 경쟁사 전부 "디스크/요추/회복" 문구 쓰는데, 우리는 쓰면 단속 대상, 안 쓰면 검색·전환 모두 죽음. 딜레마.
  2. **상표 지뢰**: "스탠바이미"는 LG전자 TV 브랜드 상표 혼동 이슈. 이런 지뢰가 카테고리 전반에 깔려 있음.
  3. **섬유 KC 자율안전확인 대상 여부 불명확** — 확인 안 하고 들어가면 나중에 행정처분.
  4. **리뷰 장벽**: 상위 상품 리뷰 2,000~5,000개. 후발 주자는 2년은 뒷북.

## 가장 안전한 후보
- **후보: 옷 압축팩**
- **이유**:
  1. **규제 리스크 제로에 가까움**: KC 의무 없음, 식품/전기/어린이제품/의료기기 전부 해당 없음.
  2. **과대광고 리스크 낮음**: "공간 절약", "부피 압축 40%" — 실측치로 방어 가능, 의료·건강 표현 전무.
  3. **IP 리스크 낮음**: 구조가 단순하고 범용, 국내 실용신안은 밸브 구조만 주의하면 회피 가능.
  4. **반품/클레임 관리 쉬움**: 기밀 QC 1가지만 잡으면 끝. 알레르기·체형 차이·색상 불만 같은 변수 없음.
  5. **어린이/의료 타깃 아님** → 대부분의 까다로운 규제에서 자유.

---

**픽시 결론**: 리스크만 놓고 보면 **옷 압축팩 > 부분 가발 > 허리베개**. 허리베개는 기회점수 83.3이 매력적이지만 **법적 리스크가 수익보다 크다**. 부분 가발은 레드오션+반품지옥이고, 옷 압축팩만이 "편하게 장사할 수 있는" 카테고리입니다 대표님 🦋
