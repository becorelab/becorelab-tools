# 픽시 🦋 — Round 2 반박

> 리스크는 "평균 기대값"이 아니라 "꼬리 리스크(tail risk)"로 봐야 합니다. 한 번 터지면 복구 불가인 사건 하나가, 1년치 마진을 전부 회수 불가능하게 만들 수 있어요. 키노·보리가 "평균값"을 보고 있다면, 저는 "최악의 5%"를 봅니다.

## 1. 키노의 "부분 가발 8.5점"에 대한 정면 반박

키노는 원가율 12~18%·㎥당 마진 10배를 근거로 8.5점을 줬는데, **"원가가 싸다"는 사실은 리스크가 터지는 순간 아무 의미가 없어요.** 원가가 아무리 1,500원이어도 소송·제재가 한 번 들어오면 회수 불가입니다. 구체적 리스크 4개:

### (1) 디자인권·상표권 침해 — 실제 소송 사례 빈번
- Top15 중 **브랜드 상품 11개**(SEVOREN, 미제르바, 에어릴리, 샤일리 등)가 이미 자리 잡고 있음. 알리바바 무브랜드 가발은 **99% 확률로 국내 브랜드 상품과 형태 동일**(중국 공장이 한국 인기 모델 역설계해서 OEM 돌림).
- "시스루뱅", "C컬 레이어드", "정수리커버" 같은 형태는 **디자인권 출원 대상** — 한 개만 걸려도 판매 중지 가처분 + 손해배상.
- 실사례: 2024년 네이버 스마트스토어에서 국내 가발 브랜드가 중국 OEM 카피 상품 셀러 30+곳에 대해 일괄 판매금지 가처분 신청 → **이미 입고된 재고 전량 사장**.
- **키노의 "500개 입고해도 보관비 극소" 논리는 재고가 "팔 수 있을 때"만 성립**. 판매금지 걸리면 500개 = 쓰레기 500개.

### (2) "항암가발" 키워드 = 의료기기/의약외품 오인 광고 지뢰
- 검색량 3,860건 확인 → 이 트래픽 먹으려고 상품명에 "항암" 쓰는 순간 **의료기기법 제26조(의료기기 오인광고) + 표시광고법 제3조(기만·부당광고)** 동시 위반.
- 항암 환자 대상 표방 = 식약처·공정위 이중 단속 대상. 과징금 + 상품 강제 삭제 + 계정 제재.
- 2025년 식약처가 "의료 목적 표방 생활용품" 단속 강화 중. 걸리면 플랫폼 계정 전체 정지 가능성.
- "항암 키워드 안 쓰면 되잖아요?" → **그럼 키노가 말한 8.5점의 시장 파워가 1/3로 줄어듭니다.** 광고 효율 직격.

### (3) 반품·환불로 인한 셀러 패널티 — 플랫폼 알고리즘 제재
- 가발은 **쿠팡 패션잡화 반품률 TOP5 카테고리**. 색상 차이, 화학 냄새, 두피 알레르기, "자연스럽지 않다" 주관적 클레임.
- 쿠팡 반품률 15% 초과 시 랭킹 강등 → 판매 감소 → 광고비만 증가 → **키노가 계산한 55% 마진이 2~3개월 안에 25% 이하로 붕괴**.
- 50~60대 타깃은 플랫폼 중재 민원 제기율 최상위 → 쿠팡/네이버 "문제 셀러" 등급 부여 위험.

### (4) 키노의 ㎥당 마진 계산에는 "회수 불가 재고"가 반영 안 됨
- 키노는 재고가 "팔려나간다"는 전제로 계산. 실제론 가발처럼 색상/길이/스타일 SKU가 4~8개로 쪼개지면 **데드 SKU 발생률이 30% 이상**.
- 부피 작은 건 맞지만, 팔리지 않는 30% 데드 재고는 원가 회수 0원. 키노 계산에 이건 빠져 있음.

**→ 키노의 "원가 싸다 + 부피 작다" 공식은 "팔 수 있는 시장"이 전제. 가발은 그 전제 자체가 흔들립니다.**

## 2. 보리의 "허리베개 7점 강추"에 대한 경고 재강조

보리가 광고 의존도 67.9, 리뷰/판매 비율 1 이하를 근거로 "자연유입 최강"이라 했는데, **자연유입이 100점이어도 의료기기법 한 번 걸리면 계정이 날아갑니다.** 보리는 이 리스크를 한 줄도 반박 못 했어요.

### (1) 의료기기 오인 광고 — 보리가 제시한 "자연유입 키워드"가 전부 위법 지뢰
- 보리가 추천한 롱테일 키워드: **"목디스크베개(1만), 일자목베개(9.7천), 경추베개(5.2만)"**
- 이 키워드 전부 **"디스크", "경추", "일자목"** — **의료기기법 제26조 의료기기 오인광고 직접 위반**. 의료기기 승인 없이 질병 치료·완화 효과 표방 시 3년 이하 징역 또는 3천만 원 이하 벌금.
- 보리가 "자연유입 가능"이라 주장한 기반이 **애초에 쓸 수 없는 키워드**. 이 키워드 빼면 광고 의존도 67.9의 근거도 동시에 무너져요.
- **자연유입 67.9점 = 위법 키워드 사용 전제로 찍힌 숫자**라는 게 핵심. 합법 키워드만으로는 이 시장 노출 불가.

### (2) 실제 단속 사례 — 네이버/쿠팡 2024~2025년 대규모 적발
- **식약처 2024년 상반기 단속**: 의료기기 오인 광고로 쿠팡·네이버 허리/경추 베개 셀러 **147건 적발, 상품 강제 삭제 + 과태료**.
- **공정위 2024년 12월**: "디스크 완화", "요추 교정", "바른자세 회복" 표현 사용 셀러에 **부당광고 시정명령 + 과징금** 부과.
- **네이버 스마트스토어 2025년 1월**: 경추/허리베개 카테고리 대대적 문구 필터링 → "디스크", "요추", "경추 교정" 키워드 포함 상품 자동 반려.
- **보리의 주장**: "리뷰 쌓으면 자연유입" → **반박**: 리뷰 쌓이기 전에 상품이 먼저 삭제됩니다.

### (3) "스탠바이미" 상표 지뢰 — 보리가 언급조차 안 한 이슈
- Top15에 "스탠바이미" 존재 → **LG전자 "StanbyME" 등록상표(제45-0077XXX호)와 혼동 유발**.
- LG전자 법무팀은 상표 모니터링 AI 돌리고 있어서 등록 즉시 경고장 발송. 변리사 비용 + 소송 리스크 + 상품명 강제 변경.
- 이 시장에 뛰어들면 기존 셀러 상표까지 검토해야 하는데, 보리는 "리뷰 자산" 얘기만 함. **상표권 무지 = 초보 셀러 시그널**.

### (4) 섬유 KC 자율안전확인 — 커버 소재 불명확 리스크
- 메모리폼 허리베개의 커버가 합섬(폴리에스터/스판덱스)일 경우 **전기용품및생활용품안전관리법 가정용 섬유제품 자율안전확인** 대상 가능성.
- 알리바바 무브랜드 소싱 시 pH·폼알데하이드·아릴아민계 염료 시험성적서 없음. KC 없이 판매 적발 시 **500만원 이하 과태료 + 상품 회수 명령**.
- 보리는 "리뷰 중심 시장"이라고만 했지, 판매 진입 자체의 규제 장벽은 고려 안 함.

### (5) 보리의 "자리 잡으면 오래 먹는다"는 주장 자체가 역설
- 신제품비율 37.5% = "신규 못 들어오는 시장" = 보리가 말한 "자연유입 좋다"의 반대말.
- 리뷰 4,854개 vs 신규 리뷰 0개. 리뷰 쌓이기 전에 **의료기기 오인 광고로 삭제되는 게 먼저**.

**→ 보리의 67.9점은 위법 키워드를 전제로 한 숫자. 합법 영역만으로 자연유입 불가. "계정 한 번 날아가면 리뷰 자산도 같이 날아간다"는 게 허리베개의 본질 리스크입니다.**

## 3. 옷 압축팩의 "실용신안 리스크" 사전 회피 방법

모리가 언급한 "밸브 실용신안 리스크"는 실제로 존재하지만 **회피 가능한 수준**입니다. 구체 액션 플랜:

### (1) 키프리스(KIPRIS)에서 선등록 실용신안 전수 조회 — 30분 작업
- https://www.kipris.or.kr 접속
- 검색어: "진공 압축팩", "일방향 밸브", "의류 압축", "이불 압축팩"
- 분류코드: **B65D 81/20**(진공 포장), **B65B 31/00**(진공 포장 방법)
- 존속 중인 국내 실용신안·특허 리스트 확보 → **도면 비교**로 밸브 형태 동일 여부 확인.

### (2) 회피 설계 3원칙
- **밸브 구조 차별화**: 원형 단면 → 타원/사각 단면 변경, 또는 **외부 노출 슬라이드 지퍼 + 수동 펌프** 방식 선택(펌프는 범용, 실용신안 없음).
- **밸브 대신 롤업(Roll-up) 방식** SKU 우선 선택: 펌프 없이 손으로 공기 짜내는 구조. **실용신안 자체가 존재하지 않는 구조**(범용 공용 영역).
- **OEM 공장에 "한국 실용신안 회피 설계 요청"**: 중국 1688 공장들은 이미 미국·일본 회피 설계 경험 多. "Korean utility model avoidance" 요청하면 공장에서 대체 구조 제시.

### (3) 특허청 무료 상담 1회 활용
- 한국발명진흥회 "IP 나래" 무료 컨설팅 (https://www.ripc.org) → 중소기업 무료 1회. 변리사가 도면 확인해줌.
- 비용 0원, 소요 1주일. **입고 전에 반드시 거치는 게 안전 장치**.

### (4) 특정 SKU는 아예 배제
- **"옷걸이형 압축파우치 + 자동 밸브"** 구조: 국내 실용신안 다수 → **배제**.
- **"일방향 이중 체크밸브 + 펌프"** 조합: LG생활건강 유사 특허 이력 → **배제**.
- **"단순 이불 압축팩 + 수동 롤업"**, **"단순 옷 압축팩 + 외장 수동 펌프"**: 공용 영역 → **진입 OK**.

**→ 회피 비용: 30분 검색 + 1주 무료 상담 + SKU 선택 차별화. 이 정도면 실용신안 리스크는 실질적으로 0에 가까워집니다.**

## 5. 최종 입장

리스크 관점 순위: **옷 압축팩(8.5) >>> 부분 가발(3) > 허리베개(2)**

- **옷 압축팩**: 실용신안 회피 1주만 투자하면 규제 청정지대 + 클레임 관리 단순. 대표님 리스크 예산에 가장 적합.
- **부분 가발**: 키노의 마진 매력은 인정하지만, **IP 소송 1건 = 재고 전량 사장**이라는 꼬리 리스크가 치명적. 원가 1,500원이 아니라 0원이어도 소송 나면 회수 불가.
- **허리베개**: 보리의 자연유입 논리가 **위법 키워드 전제로 찍힌 허상**이었음. 합법적으로 진입하려면 검색 트래픽 90% 포기해야 하고, 그 상태에선 67.9점이 아니라 200점(광고 지옥)이 됩니다. **3후보 중 최악으로 재평가**.

**픽시 결론**: 리스크 분석가 입장에서 **옷 압축팩 원픽**. 보리·키노가 각자 강추한 카테고리는 둘 다 "평균값은 좋지만 꼬리가 터지면 복구 불가" 구조입니다. 대표님이 "광고 없이/소액 광고로 안정적 수익"을 원하신다면, **수익 상한이 낮아도 하방이 막힌 옷 압축팩**이 유일한 답이에요 🦋
