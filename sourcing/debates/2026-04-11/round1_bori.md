# 보리 🐻 — 자연 유입 평가 (Round 1)

> 관점: 광고 없이 또는 소액 광고로 자연 유입이 가능한가. ad_dependency 수치를 "왜 그런지" 키워드·리뷰 구조로 해석.

---

## 1. 부분 가발
- **점수: 3/10** (광고 없이 팔기 매우 어려움)
- **광고 의존도: 177.0** — 3개 후보 중 **최악**. 150 이상이면 광고 없이는 사실상 진입 불가 영역.
- **롱테일 키워드 기회**:
  - Top15 키워드 광고가중치는 전부 0인데 의존도가 177인 건 "겉은 자연검색 가능해 보이지만 실제 매출은 광고·리뷰파워로 굴러간다"는 뜻. 키워드 구조 자체가 함정.
  - 빅 키워드가 "50대헤어스타일(1.9만), 가발(1.8만), 스핀스왈로펌(1.4만), 앞머리(1.1만)" 같은 **정보성/스타일성 키워드** 중심. 구매 전환이 붙는 "부분가발(7천)·정수리가발(6천)" 같은 구매의도 키워드는 검색량이 절반 이하로 내려감.
  - 롱테일로 "50대헤어스타일·베이비펌·스핀스왈로펌"을 먹으려면 블로그/유튜브 콘텐츠 자산이 필수인데, iLBiA는 뷰티 콘텐츠 자산 제로.
- **리뷰/판매 비율 해석**:
  - 매출 1위 미제르바(월판 630개, 리뷰 37) → **리뷰 대비 판매 17배**. 리뷰가 쌓인 게 아니라 광고로 밀어서 팔리는 전형적 패턴.
  - 9위 나나민(월판 575개, 리뷰 24) → **24배**, 10위 샤일리(월판 403, 리뷰 23) → **17배**. 이 비율은 "최근 광고 태워서 막 뜨는 신상" 신호.
  - 반대로 리뷰 많은 15위 끌레오라미(리뷰 239) 월판 279개 → **1.2배**. 정상적 자연유입 숫자. 즉 **상위권은 광고가, 하위권은 리뷰가 버티는 구조**. 신규 진입자는 광고 안 태우면 Top10 근처도 못 가.
  - 신제품비율 57.5% + HHI 44.1 → 상위가 계속 교체되는 시장. "신상 광고 빵-" 해야 자리 잡힘.
- **iLBiA 브랜드 시너지**:
  - 일비아는 세제·건조기시트·청결용품 브랜드. **가발과 고객 겹침 0%**. 기존 자사몰·스마트스토어 단골에게 크로스셀 시나리오 자체가 안 나옴. 자연유입 보조지표 최악.
- **근거 종합**: 의존도 177 = 광고 화력 없이는 Top 노출 불가 + 브랜드 시너지 0 = 소액 광고로는 매출 안 찍힘.

## 2. 옷 압축팩
- **점수: 6/10** (소액 광고 + 자산 쌓으면 자연유입 기대 가능)
- **광고 의존도: 152.4** — 중간. 150 언저리라 "광고 꺼도 일부는 자연 유입으로 살아남음" 경계선.
- **롱테일 키워드 기회**:
  - 빅 키워드 "이불압축팩(1.1만)·옷정리함(1만)·이불압축파우치(9.7천)"는 **명확한 구매의도 키워드**. "스타일 검색어"인 가발과 달리 여긴 전부 "문제 해결" 키워드라 전환율이 구조적으로 높음.
  - 롱테일 구조가 **진짜 풍부함**: 압축팩/옷압축팩/의류압축팩/겨울옷압축팩/패딩압축팩/이불압축팩/이불정리함/이불가방/이불보관함/옷수납함/옷보관함/겨울옷보관 — **세분화된 변형 키워드 12개 이상**. 상품명 조합만 잘해도 롱테일 다수 먹을 수 있음.
  - "옷정리노하우(2천)"는 정보형 키워드 — 블로그/쇼츠로 콘텐츠 자산 붙이기 딱 좋음.
- **리뷰/판매 비율 해석**:
  - 2위 메리달(월판 1188, 리뷰 38) → **31배**, 6위 니드잇(월판 1831, 리뷰 70) → **26배**. 상위 2개는 **광고 화력 100% 의존**. 그래서 의존도 152.
  - 하지만 3위 홈소이(월판 557, 리뷰 118) → **4.7배**, 13위 ERTY(월판 199, 리뷰 95) → **2.1배**, 15위 모리보(월판 634, 리뷰 37) → 17배지만 리뷰 쌓이는 중. **중위권은 리뷰 기반 자연유입 비중이 명확히 존재**.
  - 즉 "상위 1~2등은 광고 괴물, 3~10등은 리뷰로 먹고 삶" — **3~7위 구간은 신규 진입자가 리뷰 전략으로 치고 들어갈 여지 있음**.
  - 신제품비율 60% → 시장이 여전히 열려 있음. 자리 못 잡은 구간 많음.
- **iLBiA 브랜드 시너지**:
  - 생활용품·수납정리 = iLBiA 생활용품 브랜드 정체성과 **겹침 있음**. 세제·건조기시트 구매 고객에게 "계절옷 정리 압축팩" 크로스셀 자연스러움. 자사몰 단골 풀에 바로 얹을 수 있음.
  - 브랜드 적합도 중상.
- **근거 종합**: 의존도 152는 "광고 의존"이지만 롱테일 구조가 풍부하고 리뷰 기반 중위권이 살아있음 + iLBiA 크로스셀 가능 = **자연유입을 "만들 수 있는" 시장**.

## 3. 허리베개
- **점수: 7/10** (3개 중 자연유입 가능성 최고)
- **광고 의존도: 67.9** — 3개 중 **압도적 1위 (100 이하 우수 구간)**. 시장이 "광고 없이도 굴러간다"는 신호.
- **롱테일 키워드 기회**:
  - 빅 키워드가 "베개(13.8만), 배게(5.6만), 경추베개(5.2만), 배게베개(3.4만), 베게(2.6만)" — **어마어마한 검색량 풀**. 광고가중치 전부 0.
  - 단, 빅 키워드는 경쟁 포화 확실 → 노려야 할 건 롱테일: "허리베개(1.2만), 옆잠베개(2.3만), 목디스크베개(1만), 일자목베개(9.7천), 베개추천(1만), 메모리폼베개(1.1만)". **롱테일 하나당 검색량이 1만 내외로 크고, 구매의도 강함**.
  - "먼작귀/농담곰"처럼 캐릭터 키워드까지 섞여 있어 키워드 다변화도 풍부.
- **리뷰/판매 비율 해석** (여기가 핵심):
  - 1위 닥터센스(월판 1085, 리뷰 484) → **2.2배**
  - 2위 생활콕(월판 787, 리뷰 644) → **1.2배**
  - 6위 허쉬베개(월판 738, 리뷰 **2088**) → **0.35배** ← 리뷰가 판매의 3배
  - 7위 스탠바이미(월판 1081, 리뷰 **4854**) → **0.22배** ← 리뷰 누적 절대 강자
  - 11위 마메(월판 5015, 리뷰 **5568**) → **0.9배** ← 월판 5천개가 리뷰로 굴러감
  - → **상위권이 리뷰 누적으로 버티는 전형적 "자연유입 시장"**. 신규가 치고 들어가려면 초기 리뷰 100~300개만 쌓이면 롱테일부터 자리 잡기 가능. 가발(광고 화력 전쟁)과 정반대 구조.
  - 신제품비율 37.5% → 시장 회전율 낮음 = 자리 잡으면 오래 먹는 구조 = 광고비 회수 쉬움.
  - 한 가지 주의: HHI 37.8로 매출 분산은 좋지만 **상위 리뷰 장벽이 매우 높음**(4854, 5568). 후발주자가 Top3 진입은 힘들고 "틈새(벨트형·C커브·높이조절) 롱테일 키워드"로 Top10~15를 노리는 그림.
- **iLBiA 브랜드 시너지**:
  - 침구/수면/건강 카테고리 → iLBiA 생활용품과 겹침은 **약함**. 세제 사는 고객이 허리베개 바로 사진 않음.
  - 다만 "가족을 사랑하는 브랜드" 철학과 "수면·건강" 메시지는 연결 가능. 브랜드 확장 여지는 있음. 가발보다는 훨씬 자연스러움.
- **근거 종합**: 의존도 67.9는 거짓말 안 함. 리뷰/판매 비율이 1 이하로 내려가는 상품이 5개 이상인 시장은 **광고 없이도 검색·재구매로 굴러가는 건강한 시장**. 브랜드 시너지만 보조로 챙기면 됨.

## 내가 반대하는 후보
- **후보: 부분 가발**
- **이유**:
  1. 광고 의존도 177 — 측정 3개 중 꼴찌. 수치상으로도 자연유입 부적합.
  2. 상위권 리뷰/판매 비율이 17~24배. 이건 "리뷰가 쌓여서 파는 게 아니라 광고비로 미는 시장"이라는 직접 증거.
  3. 키워드 상위가 "스타일·연령대 검색어"라 구매의도 전환율 낮고, 롱테일 가져가려면 뷰티 콘텐츠 자산 필수 — iLBiA엔 제로.
  4. iLBiA 생활용품 단골에게 크로스셀 불가. 자연유입의 보조 엔진(기존 고객 풀)도 작동 안 함.
  5. **대표님이 강조하신 "광고 없이/소액 광고로 적정 매출" 기준에서 가장 멀리 있는 후보**.

## 강력 추천
- **후보: 허리베개**
- **이유**:
  1. 광고 의존도 **67.9** — 3개 중 유일하게 100 이하 우수 구간. 수치 자체가 "자연유입 시장" 증명서.
  2. 리뷰/판매 비율이 1 이하인 상품이 5개(허쉬·스탠바이미·마메 등). 이는 **"리뷰 자산 → 재구매·검색 유입"** 사이클이 실제로 돌아간다는 가장 강한 신호.
  3. 롱테일 키워드 구조가 풍부(경추베개·옆잠베개·목디스크베개·일자목베개·허리베개 각 1~2만 검색량)하고 전부 광고가중치 0. 상품명·상세페이지 최적화만으로 노출 가능.
  4. 신제품비율 37.5% (가발 57.5%, 압축팩 60% 대비 안정적). 자리 잡으면 광고 꺼도 오래 먹는 구조.
  5. 브랜드 시너지는 약하지만, **대표님이 가장 중요시한 "광고 의존도" 기준**이 압도적이라 이 한 지표만으로도 1순위 충분.

- **차선: 옷 압축팩**
  - 의존도 152는 부담이지만 롱테일 12개 + 리뷰 기반 중위권 + iLBiA 크로스셀 가능성으로 커버됨. 허리베개가 브랜드 시너지 이슈로 탈락하면 즉시 1순위 승격 가능.
