# 하치 🐣 — Round 3 악마의 변호인

4명의 잠정 Top인 **옷 압축팩(2733)**에 대해 "이거 하면 망하는 시나리오"를 집요하게 파고들어 정말 넘어갈 만한지 검증한다. 그리고 탈락 후보 2개(부분가발·허리베개)에 대해서도 최종 이유를 못박는다.

---

## 🎯 옷 압축팩 — 망하는 시나리오 6가지

### 시나리오 1. 🚨 계절성 폭탄
- AI 분석 `risk_factors`에 명시됨 — "계절성 수요, 겨울옷 보관 시점 집중"
- 보리도 "연 광고비 2,100만원, 2년차에 끄면 매출 0원" 경고
- **망하는 조건**: 10~12월 집중 진입 → 11월 재고 부족 or 12월 이후 재고 폭탄
- **방어**: "여행용"으로 포지셔닝하면 계절성 완화 가능 (Top 상품 중 "니드잇 여행용"이 월 1,996만원 + "아케이 여행용"이 2,152만원)

### 시나리오 2. 🚨 부피/보관비 폭탄 (키노의 핵심 경고)
- 추천 사양: 대형(70x100) + 특대형(80x120) + 옷걸이형 10매 세트
- 쿠팡 제트배송 대형 박스 구간 진입 → 보관비 월 15~25만원/500개
- 파렛트당 수익 압축팩 97~195만원 vs 가발 3,300~5,500만원 (키노 계산)
- **방어**: **규격 전략 변경** — 옷걸이형/특대형 배제, 여행용 "소형 롤업(20x30, 30x40) 단일 SKU"만 진입. 크기 절반 → 보관비 절반

### 시나리오 3. 🚨 불량률 → 반품 지옥
- AI 분석: "압축 유지력 저하 → 환불 증가, 불량률 관리 필수"
- 압축팩 특성상 밸브 1군데만 새면 전량 반품
- 쿠팡 반품률 5% 초과 시 노출 강등
- **방어**: 알리바바 1688 샘플 3사 이상 테스트 후 QC 상위 1개만 선정. 이중 지퍼+역류방지 밸브 필수 사양 고정

### 시나리오 4. 🚨 골리앗 리뷰벽
- Top 상품 리뷰 분포: **리브맘 5,016 / 아케이 2,659 / 헤인느 1,393 / 파인 933**
- 신규 진입하려면 리뷰 1,000개 이상 쌓아야 Top10 진입 가능
- 리뷰 100개 쌓는데 체험단 + 광고 400~600만원
- **방어**: 리뷰 수천개 상품 직접 경쟁 금지. 중위권(30~60위, 리뷰 50~200)에 포지션. 롱테일 키워드 공략

### 시나리오 5. 🚨 가성비 전쟁
- 1매당 1,080원 ~ 18,450원 = 단가 17배 차이 = 가격 파괴 진행 중
- 이미 레드 오션 가격대
- **방어**: 최저가 경쟁 금지. "프리미엄 롤업 + 여행가방 호환" 포지션 → 중상위 가격대 유지

### 시나리오 6. 🚨 밸브 실용신안 (픽시 지적)
- KIPRIS 미조회 상태에서 진입하면 경고장 위험
- **방어**: 픽시 제안대로 "밸브 없는 롤업 수동 압축" 방식으로 회피. KIPRIS 30분 조회로 사전 확인

## ✅ 옷 압축팩 최종 판정: **통과 (조건부)**

**넘어갈 만한가?** → **YES, 단 4가지 조건 충족 시**
1. 여행용 "소형 롤업" 단일 SKU로 시작 (옷걸이형/특대형 제외)
2. 밸브 없는 수동 압축 구조 선택 (실용신안 회피)
3. 1688 QC 3사 비교 후 최상위 1사만 선정 (불량률 방어)
4. 체험단 200 + 소액 광고로 중위권 진입 (리뷰 골리앗 회피)

**제외 시 망함**: 위 4개 중 하나라도 빠지면 키노/보리 경고가 현실화됨.

## 🗑 탈락 후보 최종 이유

### 부분 가발 (349)
- 키노 혼자 강추 (원가율 12~18%) vs 3명 전원 반대
- **픽시 결정타**: 무브랜드 알리바바 가발 99%가 국내 브랜드 카피 → IP 침해 1건 = 재고 전량 사장
- **보리 결정타**: 광고 의존도 177 + 광고비 주문당 8,000원 = 마진 반토막 → 키노 원가 우위 증발
- **모리 결정타**: 3후보 중 시장 규모 최소 (월 1,402만원)
- **탈락 사유**: 마진은 높지만 **하방 리스크가 복구 불가** (IP 소송 한 방이면 끝)
- **처리**: 보류 태그. 인증/카피 이슈 없는 차별화 사양 발견 시 재검토

### 허리베개 (302)
- 보리 혼자 강추 (자연유입 우수) vs 3명 반대
- **픽시 결정타**: 보리가 제시한 롱테일 키워드("목디스크/일자목/경추베개") **전부 의료기기법 제26조 직접 위반** → 보리의 자연유입 수치가 위법 키워드 전제였음이 드러남
- **키노 결정타**: 원가율 25~45% 최악, 부피 대형, 파렛트당 수익 가발 대비 1/50
- **모리 결정타**: Top 상품 리뷰 2,000~5,500건 골리앗 5개, 신제품비율 37.5% 최저 (진입 장벽)
- **탈락 사유**: 자연유입의 "전제"가 법적으로 막혀있음. 합법 키워드로만 가면 광고 의존도 다시 올라감
- **처리**: 보류 태그. 의료기기 오인 표현 완전 배제한 "디자인/인테리어 베개" 포지션 재구상 시 재검토
