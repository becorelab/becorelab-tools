# 모리 🦊 — 배수구 트랩 시장성 평가 (Round 1)

## 두 스캔 비교 (329 wing vs 2411 manual)

| 지표 | Scan 329 (wing) | Scan 2411 (manual) | 해석 |
|---|---|---|---|
| 기회점수 | 71.6 | 75.3 | manual이 조금 높음 |
| Top10 평균가 | 14,284원 | 20,027원 | **manual이 +40%** (고가 상품 편입) |
| Top10 월매출 | 3,782만 | 4,230만 | 비슷 |
| 월판매 | 2,957개 | 2,946개 | 동일 |
| **광고 의존도** | **49.1** | **72.6** | 🚨 manual 스캔 때 광고 과열 |
| 신제품률 | 27.5% | 20.0% | wing 때 신규 유입 더 활발 |
| HHI | 39.6 | 37.3 | 둘 다 높은 집중도 |

**결론**: 두 스캔이 같은 시장을 봤는데 **광고 의존도가 49→73으로 급등**. 이건 신규 진입자들이 오가닉으로 못 뚫어서 광고비 퍼붓고 있다는 신호. Top1 냄새제로는 자연 유입, 나머지는 광고로 밀어내는 구조로 고착화 중. **오가닉 진입 창이 닫히는 중**이라고 본다.

## 시장 규모 분석

- Top10 월매출 합산 ≈ 3.8~4.2억, 연 환산 ≈ **45~50억**
- 상위 30위까지 가면 연 100억 내외 (HHI 37~39는 상위 집중형)
- 검색량: `하수구트랩` 2.4~3만, `배수구트랩` 1만, `화장실하수구냄새` 7.5천 — **키워드 총합 6~7만/월**, 견고하나 폭발형 아님
- 단품 중위권(월 1,000~1,500만) 진입 시 연 1.5억 내외 — **가능**하지만 골리앗 그림자 짙음

## 1등 골리앗 뚫기 전략 — 냄새제로 SM-50D (리뷰 32,997)

**이 숫자가 진짜 벽이다.** 리뷰 3.3만은 축적 기간 2~3년 + 로켓 자연 유입 + 탑셀러 플라이휠. 신규로 이걸 정면 돌파 = 자살.

### 우회 전략 3개
1. **세탁기/건조기 전용 2구 호환 트랩** (briefing Top9 "올인원 세탁기배수구트랩" 월 2,505만, 리뷰 1,891 이미 검증) — 냄새제로는 바닥배수 전용, 세탁실 트랩은 비어있음
2. **싱크대 방향 분기** — 검색량 `싱크대배수구` 7,130, 별도 카테고리로 착지 가능
3. **디자인 프리미엄** — 현 Top10 전원이 흰색 플라스틱 공산품 외관. 욕실 인테리어 트렌드(매트블랙/브러시골드) 활용 가능하나 단가 압박

**현실 판단**: 1~3번 어떤 것도 냄새제로를 "뚫는" 게 아니라 "회피"하는 전략. 즉 이 카테고리에서 1등은 불가, 중위권 안착이 최대치.

## 중위권 구조

- 월 1,000~3,000만원 구간이 10~15위 내 8개 이상 포진 — **중위권 두텁다 (긍정)**
- BUT 이들 대부분 리뷰 400~1,000+ 보유, 2~3년 운영된 상품들
- 신규 진입자가 광고 없이 침투할 여지 = Top9(올인원 세탁기배수구트랩) 처럼 **세그먼트 파생형**으로만 가능
- 광고 의존도 72.6 = 중위권은 광고로 유지되는 구조, 광고 끄면 바로 죽는다는 뜻

## 🔍 카테고리 특수 변수 (자력 발견)

1. **"세대 넘버 인플레" 현상** — '7세대' 키워드가 Top15에 4개. 버전 네이밍으로 차별화를 포장하는 카테고리 = 실제 기술차이 미미, **마케팅 전쟁터**. 신규 진입 시 "8세대" 네이밍 가능.
2. **계절성 약함, 그러나 여름 스파이크** — 악취/벌레 키워드 → 6~8월 검색량 1.5~2배 예상. 봄(4월) 진입은 여름 피크 준비에 맞춤.
3. **반품률 리스크** — briefing이 지적한 "사이즈 미스매치" = 이 카테고리 고질병. CS 공수가 일반 공산품 대비 2~3배.
4. **장바구니 단가 낮음** — 7,000~20,000원대. 로켓 마진 짜내기 어려운 구간 (수수료 + 배송비 감안).
5. **오가닉 창이 닫히는 중** — 두 스캔 간 광고 의존도 급등은 구조적 포화 신호.

## 📦 브랜드 적합성 (iLBiA vs 오모모)

**→ 오모모 (아이디어 유통 상품) 적합**

- iLBiA는 "일상의 편리함 + 세제/탈취제" 중심 — 하수구 트랩은 **기능성 하드웨어**라 iLBiA 감성 브랜드와 결이 다름
- 하수구 트랩 = 인테리어 잡화/공구 느낌 → 오모모의 아이디어 상품 포지션이 맞음
- **단, 핵심 주의**: 추천 키워드가 **"고체탈취제"**(검색량 3.2만)로 뜬 건 매우 흥미로움 → 이 카테고리를 **iLBiA 고체탈취제 진출 교두보로 쓰는 시나리오**도 존재 (아래 확장 플랫폼 참고)

## 🚀 확장 플랫폼 관점

배수구 트랩이 "첫 진입"이면 의미 있는가? **조건부 Yes.**

### 시나리오 A: 단품으로 끝 (오모모)
- ROI 계산 단순, 손절 쉬움, 확장성 낮음 → 4점

### 시나리오 B: "욕실 악취 솔루션" 라인업 발판 (iLBiA)
- 하수구 트랩 → 고체탈취제(준비중!) → 배수구 세정제 → 섬유탈취제 → **iLBiA 욕실/생활 카테고리 장악**
- 쿠팡 연관 추천 "이 상품 본 사람이 본 상품" 파이프라인 구축 가능
- **추천 키워드가 "고체탈취제"로 뜬 건 쿠팡 AI가 이미 두 카테고리를 연결해 본다는 신호** — 이거 대박 포인트 🎯
- 시나리오 B라면 권장도 **6.5점**으로 상향 가능

### 시나리오 C: 세탁실 SKU 발판 (iLBiA)
- 세탁기 전용 2구 트랩 → 세탁조 세정제 → 건조기 시트 → **세탁실 원스톱**
- 건조기시트 기존 라인과 교차 셀링 가능

## 실행 역량 가점 (iLBiA 경험 활용 가능성)

- iLBiA 생활용품 운영 경험 → CS 대응 숙련도 ○ (반품률 리스크 완화)
- AI 이미지 생성 → 공산품 렌더링 쉬움, 누끼/사용씬 생성 유리 ○
- 디자인 차별화 → 매트블랙/우드톤 렌더로 심미 차별화 가능 ○
- BUT **공산품 금형/몰드 투자**가 차별화 핵심인 카테고리 — AI 이미지만으로 뚫기 부족
- 가점: **+0.5** (시나리오 B면 +1.5)

## 핵심 결론

1. 시장은 **견고하지만 오가닉 창이 닫히는 중** (광고 의존도 49→73 급등)
2. 1등 골리앗(리뷰 3.3만)은 정면 돌파 불가 → **세그먼트 파생형**(세탁기 전용/싱크대용)으로만 진입 가능
3. 중위권 두텁지만 광고로 연명 → 신규가 월 1,500만 찍으려면 광고비 감수
4. **단품 ROI만 보면 4점** (매력 낮음)
5. **BUT "iLBiA 고체탈취제 교두보"로 보면 6.5점** — 추천 키워드 "고체탈취제" 교차 신호가 결정적
6. 대표님 결정 포인트: 이거 **단품 장사** vs **카테고리 장악 플랜 1호** 중 뭐로 쓸지
