# 보리 🐻 — 배수구 트랩 자연 유입 평가 (Round 1)

## 진행 권장도: 7/10

## 광고 의존도 해석
- wing 49.1 vs manual 72.6 — **23포인트 격차**. 같은 카테고리인데 왜 이렇게 차이 나느냐?
- 답: Top 리스트 구성이 거의 동일하고 리뷰 수/매출도 비슷한데 **광고 노출 기준이 다른 시점에 측정**된 결과. wing 스캔에서 광고 태그 붙은 상품이 덜 잡혔다는 뜻 = **광고 없이도 Top 진입한 상품이 실재**.
- 보수적으로 잡아도 **60% 내외**로 보는 게 맞음. iLBiA 기준(40% 이하 선호)보다 높지만, 심리스 팬티/압축팩 대비 **훨씬 양호**.
- 결정적: **키워드 15개 모두 광고가중치 0**. 이건 "키워드 검색 시 광고보다 유기 노출이 더 중요" 신호. 쿠팡에서 이 정도면 SEO 잘 짜면 진입 가능.

## 키워드 구조 & 롱테일 공략
- 상위 키워드 분산: `변기막혔을때`(44K) → `고체탈취제`(32K) → `하수구트랩`(29K) → `세면대부속품`(17K) → `변기부속품`(15K)
- **머리 키워드 없음**. Top3가 각각 다른 맥락(수리/탈취/제품명). 이건 **롱테일 천국**.
- 공략 키워드 후보:
  - `세탁기배수구트랩` (6,740) — 세탁기/건조기 확장 맥락, 일비아 라인업과 완벽 매칭
  - `화장실하수구냄새` (7,450) — 문제 해결 키워드, 구매 전환 높음
  - `하수구냄새제거` (10,430) — 탈취제 카테고리와 크로스
- 2~3개 사이즈 SKU만 있어도 키워드별 최적화 상세페이지 뿌리기 가능 (AI 이미지 시대).

## 리뷰/판매 비율 분석
- **냄새제로 SM-50D**: 리뷰 33,000 / 월판매 8,500 → 리뷰 소진율 **3.88배**
  - 해석: 3~4년 누적 리뷰 골리앗. 신규 진입자가 리뷰로 따라잡기 어려움
  - 하지만 **월매출 1.28억인데 리뷰는 이미 포화** → 신규 리뷰 증가율 둔화 가능성
- **안티스멜**: 리뷰 287 / 월판매 3,745 → 리뷰 소진율 **0.077** = 신상인데 판매 폭발
  - 이건 **광고로 올라온 신상**. 보리 관점에서 안 좋음
- **루센트리**: 리뷰 290 / 월판매 2,926 → 유사 패턴 (신상+광고)
- **의미심장**: **리빙트리**(리뷰 3,663 / 판매 6,561) — 7,090원 저가, 리뷰 대비 판매 많음 = **가성비 축에서 자연유입 살아있음**
- 골리앗(SM-50D) 뚫기보다 **리빙트리 포지션(가성비 자연유입)** 공략이 현실적.

## 📦 iLBiA 브랜드 시너지 ⭐ 이번엔 결정적!
- iLBiA 라인업: 세제 / 건조기시트 / 캡슐세제 / 얼룩제거제 / **섬유탈취제 준비중** / **고체탈취제 준비중**
- 하수구 트랩 = **악취 차단 = 탈취 카테고리**. iLBiA 탈취 축과 **직계 혈통**.
- 특히 **추천 키워드가 "고체탈취제"**. 이건 우연이 아님 — 쿠팡 알고리즘이 "하수구트랩 사는 사람 = 고체탈취제 관심" 판정. **iLBiA가 고체탈취제 준비중이라는 사실과 정확히 맞물림**.
- 크로스셀 시나리오:
  - 하수구 트랩 → 고체탈취제 (욕실 탈취 동선)
  - 하수구 트랩 → 섬유탈취제 (생활악취 동선)
  - 건조기시트 사용자 → `세탁기배수구트랩` (세탁실 동선)
- **"광고 끄고 6개월 뒤 매출 살아있냐"**: 광고 의존도 49~60%지만 **iLBiA 브랜드 페이지 안에서 번들링**하면 광고 없이도 자연 노출 확보. 단독 SKU라면 의존도 위험하지만, **브랜드 라인업 일원**으로 붙으면 크게 다름.

## 🚀 확장 플랫폼 관점
- 이건 단순 하수구 트랩 진입이 아니라 **"iLBiA = 욕실/세탁실 악취 해결 브랜드"** 포지션 구축 첫 SKU가 될 수 있음.
- 순서: 하수구 트랩 진입 → 리뷰 500 확보 → 고체탈취제 런칭 (같은 상세페이지 링크드) → 섬유탈취제 번들 → **욕실 악취 솔루션 풀세트**
- 심리스 팬티/압축팩이 "일회성 소싱"이었다면, **이건 브랜드 장기 자산**. 보리 관점에서 가장 강한 시너지 후보.

## 🔍 카테고리 특수 변수 (자력 발견)
- **사이즈 호환성 리스크**: 50/70/100mm 제각각. 소비자 반품 사유 1위가 "안 맞음". 초기 리뷰 평점 관리 위험.
- **반품/CS 리스크**: "효과 미흡"도 큰 사유. 실험 영상/시험 성적서 없으면 평점 4.3 벽 못 뚫음.
- **골드박스 진입 가능성**: 중저가(5~8K 스윗스팟) + 욕실용품 = 골드박스 정기 노출 카테고리. iLBiA 브랜드면 골드박스 협상력 있음.
- **겨울 비수기 없음**: 악취 차단 니즈는 계절 독립적 (여름 > 겨울이지만 12월에도 판매 유지). 연중 매출 안정성.

## 신제품률 해석
- wing 27.5% / manual 20% → **건강한 회전**. 40% 넘으면 광고빨 판치는 시장이고, 10% 밑이면 굳어진 시장.
- 20~27%는 **"신규 진입 가능 + 기존 브랜드도 살아남는"** 이상적 구간.
- Top20 안에 `브랜드없음` 상품 2개 있음 — 브랜드 없이도 뚫리는 카테고리 = iLBiA 브랜드 달면 **브랜드 프리미엄 확보 가능**.

## 핵심 결론
심리스 팬티에서 내가 놓친 "브랜드 시너지"가 이번엔 **교과서급으로 존재**함. 추천 키워드가 "고체탈취제"로 나온다는 건 알고리즘이 이미 두 카테고리를 연결해 놓은 것이고, iLBiA 고체탈취제 준비중이라는 타이밍과 정확히 맞물려. 광고 의존도 60% 내외는 단독 SKU 기준으로는 부담이지만 **iLBiA 브랜드 라인업 일원**으로 들어가면 자연 유입 경로가 브랜드 자체에서 공급됨. 골리앗(SM-50D)은 정면 싸움 피하고 **"iLBiA 욕실 악취 솔루션"** 포지션으로 리빙트리 자리(가성비 자연유입) 노리는 게 정답.
