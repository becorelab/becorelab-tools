# 키노 🐱 — 배수구 트랩 수익·물류 평가 (Round 1)

## 진행 권장도: 6.5/10

## MOQ 시나리오별 1688 원가 추정

배수구 트랩은 ABS+실리콘 소재의 성숙 카테고리. 1688에서 "地漏防臭器/地漏芯" 검색 시 공장 단가 밴드가 잘 형성돼 있음. **단품 기준 추정:**

| MOQ | 1매 FOB (RMB) | CIF (원) | 비고 |
|---|---|---|---|
| 500 | 3.8~4.5 RMB | 1,050~1,250원 | 소량, 테스트 발주. 단가 10~15% 할증 |
| 1000 | 3.2~3.8 RMB | 900~1,080원 | 표준 첫 발주. OEM 불가, 중립 패키지 |
| 3000 | 2.5~3.0 RMB | 720~870원 | **스윗스팟**. 금형비 없는 기존 몰드 재활용 가능 |
| 3000 (세트3개) | 7.5~9.0 RMB | 2,150~2,550원 | 세트 판매 시 실질 원가 |
| 5000 | 2.0~2.5 RMB | 600~730원 | 컬러/로고 커스텀 가능, 공장 직거래 가능 |

※ CIF = FOB + 해운 LCL(RMB당 +15%) + 관세(0%, HS 3924) + 환율 190원/RMB + 통관비. 심리스 팬티에서 800~1200원으로 과대 추정한 걸 반성 — 플라스틱 사출품은 실제로 **500~900원대**가 현실적.

## 가격 포지션 분석

**두 스캔 가격대 차이의 원인:**
- scan 329 평균 14,284원 → 중저가 군(리빙트리 7,090, 에코바스 7,900, 내집클로버 9,680, 알뜰생활 10,900) 비중이 Top10에 더 많음
- scan 2411 평균 20,027원 → 7세대 프리미엄 군(안티스멜 18,900, 국민트랩 23,200, 에어스케이프 24,900) 비중이 확대

**결론: 14,900~18,900원 포지션이 가장 유리.** 이유:
1. Top1 SM-50D가 14,900원(리뷰 33,000)으로 바닥 방어선 확립 → 이 아래로 가면 가격전쟁, 위로 가면 리뷰 없이 경쟁 불가
2. 15,900~17,900원 = "SM-50D보다 +1,000원 내외 프리미엄" 포지션이 무리 없음
3. 20,000원 이상은 안티스멜/국민트랩/에어스케이프가 '7세대' 브랜딩으로 선점 → 후발주자 진입 난이도 높음

## 마진 40% 역산

**판매가 16,900원 (단품, SM-50D +2,000원 프리미엄) 시나리오 @ MOQ 3000:**

| 항목 | 금액 | 비고 |
|---|---|---|
| 판매가 | 16,900원 | |
| 쿠팡 수수료 | -1,690원 | 10.8% (생활용품) |
| 로켓그로스 배송·보관 | -3,100원 | 소형 기준 |
| 포장 (OPP+박스) | -350원 | |
| 원가 (CIF) | -820원 | MOQ 3000 중간값 |
| **순이익** | **10,940원** | |
| **마진율** | **64.7%** | ✅ 40% 초과 달성 |

**판매가 14,900원 (SM-50D 동가) 시 마진율:**
- 순이익 8,940원 / 14,900원 = **60.0%** — 여전히 여유 있음

**세트 3개입 19,900원 시나리오:**
- 원가 2,400원, 수수료 2,150원, 배송 3,200원, 포장 500원 → 순이익 11,650원 / **58.5%**

**→ 단품이든 세트든 마진 40% 달성은 쉽다.** 진짜 관건은 "가격 낮춰도 Top10 진입할 수 있는 리뷰 확보" — 수익성이 아니라 마케팅.

## 부피·물류

- **크기**: 지름 10cm × 높이 5~7cm, 무게 80~120g → 로켓그로스 **소형 사이즈 적용 가능** ✅
- **단품 vs 세트**: 단품은 SM-50D와 직접 경쟁 → 불리. **세트 2~3개입 권장**. 이유:
  1. 원가 분산(포장/배송비 1회) → 객단가 +50% 상승
  2. "화장실+세탁실+베란다" 멀티팩 스토리 가능
  3. SM-50D는 단품만 팔아서 사이즈별 믹스팩이 niche로 비어있음
- **포장**: 소형 박스(10×10×8cm) + 설치 가이드 인쇄 — 부피 매우 작음, 500개 파렛트 기준 CBM 0.8m³ 수준

## 🔍 카테고리 특수 변수 (자력 발견)

1. **"사이즈 불일치 반품" 리스크가 카테고리 최대 덫**
   - Top10 리뷰에서 공통 불만: "내 배수구에 안 맞아요"
   - 대응: **멀티사이즈 어댑터 3종(50/75/100mm) 기본 동봉** → 반품률 절반으로 꺾일 수 있음. 원가 +150원에 반품 -3% 방어면 순이익 +500원 효과

2. **"냄새차단 완벽"이 기능 표준어가 됨 → 차별화 여지 축소**
   - 7세대, 끝판왕 등 세대 경쟁 과열 → 후발주자가 "8세대"를 외쳐도 안 먹힘
   - 대응 각도: **"세탁기 2구 연결"(올인원 25,800원 선례)** 또는 **"디자인 블랙 컬러"** niche

3. **재구매 주기 6개월~1년 → LTV 낮음**
   - 한번 사면 오래 쓰는 내구재. 광고 효율 나쁠 수 있음
   - 다행히 광고의존도 49.1(wing)로 자연유입 양호 → 광고비 부담 낮음

4. **"악취" 키워드 검색량은 있으나, 단가가 낮아 매출 규모는 중간급**
   - 월 검색 "하수구트랩" 24,210 vs "배수구트랩" 10,300 → 합쳐도 보통
   - Top10 평균 월매출 3,700~4,200만원 → 2~5등 진입해도 월 3,000만원대가 현실적 상한

5. **인증 불필요** — HS 3924(플라스틱 가정용품), KC 불필요, 식약처 무관 → 진입장벽 zero ✅

## 📦 브랜드 적합성 (수익 관점)

- **iLBiA**: 고체탈취제 준비 중이라는 점이 결정적. "욕실 악취 토탈케어" 브랜드 스토리 구축 가능. **고체탈취제 + 배수구 트랩 번들** 구성 시 iLBiA가 유력하지만, 현재 iLBiA는 '섬유/세탁' 중심이라 브랜드 일관성 약함.
- **오모모 (아이디어 유통)**: **이 후보에 더 적합**. 이유:
  - '멀티사이즈 어댑터 올인원' 같은 아이디어 기믹이 오모모 컨셉과 일치
  - iLBiA 브랜드 자산을 쓰기엔 배수구 트랩이 브랜드 코어와 거리가 있음
  - 리스크 격리 — 실패해도 iLBiA 본진에 영향 없음

## 🚀 확장 플랫폼 수익성

**카테고리 확장 시 원가 분산 효과 ★★★★☆**
- 싱크대 트랩, 세탁기 배수구 트랩, 욕실 그레이팅, 바닥 트렌치까지 같은 공장에서 OEM 가능
- 공장 통합 발주로 **컨테이너 LCL → FCL 전환 가능** → 물류비 30% 절감 가능
- 고체탈취제(iLBiA 준비중) + 배수구 트랩 크로스셀 가능 → 같은 "욕실 악취" 구매 여정
- **단, SKU 하나로 끝내면 ROI 낮음.** 3~4 SKU로 카테고리 점유 각오가 있어야 발주 규모 경제 나옴

## 핵심 결론

- **수익성**: 마진 40%는 쉽게 넘김(60%+). 가격전쟁 여지 없음 ✅
- **물류**: 소형 사출품으로 로켓그로스 최적 ✅
- **함정**: 수익성이 아니라 **Top1 SM-50D 리뷰 33,000개라는 골리앗**. 가격보다 리뷰가 관건
- **권고 포지션**: 오모모 브랜드, MOQ 3000 단품+세트 2종 동시 론칭, 16,900원/19,900원
- **확장 전제**: 배수구 트랩 단독으로 끝내지 말고 "욕실 악취 3종 세트(트랩+고체탈취제+변기세정제)" 로드맵으로 접근해야 원가·물류 레버리지 확보
