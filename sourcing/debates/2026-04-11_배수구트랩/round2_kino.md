# 키노 🐱 Round 2 — 배수구 트랩 수익성 재평가

## 광고 의존도 급등 반영 (모리 지적 수용)
- 49.1 → 72.6 = **CPC 상승 구조 고착 중**. 단품 ROAS 계산 재수정.
- 기존 가정: 판매가 16,900 × 마진 64% = 단위익 약 10,800원
- 광고 반영: 쿠팡 CPC 평균 400~600원 + 전환율 2~3% 가정 → **CAC 15,000~25,000원** = 단위익 거의 소멸
- 즉 **오가닉/크로스셀 없는 단품은 손익분기 턱걸이**. 내가 Round 1에서 마진 64%만 보고 낙관했던 부분 정정.

## 브랜드 재고 — 보리 iLBiA 주장 수용 (부분)
- 보리 포인트의 결정타: **"추천 키워드 = 고체탈취제" = 쿠팡 알고리즘이 이미 연결**
- 이건 수익성 관점에서도 큰 의미 — **크로스셀 CAC가 0에 수렴**. 브랜드몰 안에서 재노출되면 광고비 한 번으로 여러 SKU 회수.
- 오모모 단품: 단위 상품 ROI만 보면 광고 과열로 회수 불확실
- iLBiA 번들: 고체탈취제·섬유탈취제·건조기시트와 교차 회수 → **LTV 관점 수익성 역전**
- **수정 입장**: 기본은 iLBiA, 단 초기 테스트 SKU는 오모모로 런칭해 리스크 흡수 후 iLBiA 이관 (모리 시나리오 B 동의)

## 단품 vs 교두보 수익성 비교
| 구분 | 단품 (오모모) | 교두보 (iLBiA) |
|---|---|---|
| 마진율 | 64% | 64% |
| CAC | 15~25K (광고 의존) | 5~10K (브랜드 크로스셀 반영) |
| 단위 순익 | -5K ~ +5K | +5K ~ +8K |
| 회수 기간 | 단품당 독립 | 라인업 공유, 3~6개월 |
| LTV | 1회성 (재구매 6~12개월) | 번들 재구매 상승 |
| 연매출 천장 | 1.5억 | 3~5억 (라인업 합산) |

**수익성 관점 결론**: 교두보 압승. 단품은 광고 과열 때문에 숫자가 안 나옴.

## 점수 조정: **6.5 → 7.5**
- 단품 ROI만 보면 5점으로 내려야 함 (광고 의존도 급등 반영)
- 교두보 가점 +2.5 (크로스셀 CAC 절감 + LTV 상승 + 라인업 연결)
- 최종 7.5/10

## 한 줄 결론
**광고 과열로 단품은 수익성 붕괴 위험 → iLBiA 교두보로 가야 숫자 살아남. 브랜드는 iLBiA, 첫 SKU만 오모모로 테스트 런칭 후 이관.**
