# 픽시 🦋 Round 2 — 배수구 트랩 리스크·법무 재평가

## 1. 브랜드 논쟁: 오모모 vs iLBiA (리스크 분산 관점)

**여전히 오모모 우세. 보리의 "교과서급 시너지"는 매력적이지만 리스크 관점에선 오히려 위험 신호.**

### 보리 주장 요약
- 추천 키워드 "고체탈취제" = 쿠팡 알고리즘이 카테고리 연결
- iLBiA 탈취 라인업(고체탈취제 준비중)과 직계 혈통
- 브랜드 번들로 자연 유입 확보

### 픽시 반박 (리스크 관점)
1. **브랜드 오염(brand contamination) 리스크** — 하수구 트랩은 **사이즈 호환 반품률이 카테고리 고질병**(모리·키노·보리 전원 지적). 리뷰 평점 4.3 벽 못 뚫는 카테고리에 iLBiA 브랜드를 달면, 저평점 리뷰가 iLBiA 탈취제 본진으로 전이됨. 쿠팡 브랜드 스토어는 모든 상품의 별점이 상호 노출됨.
2. **리콜/CS 전염 리스크** — 플라스틱 사출 공산품은 **크랙/변색/악취 역류** 같은 물리적 결함이 LTV 6개월~1년 사이 터짐. 키노 지적 "재구매 주기 1년 = 내구재" = **결함 발생 시점이 리뷰에 느리게 반영**되는 구조. iLBiA 섬유탈취제 런칭 직전에 트랩 결함 리뷰가 터지면 자충수.
3. **"준비중" 상품에 의존한 시너지** — 보리 시나리오는 iLBiA 고체탈취제 출시 타이밍에 전적으로 의존. 고체탈취제가 6개월 지연되면 트랩 단독으로 광고 의존도 60%를 버텨야 함.
4. **리스크 격리 원칙** — 키노도 정확히 지적("실패해도 iLBiA 본진에 영향 없음"). 신규 카테고리 첫 진입은 오모모로 테스트 → 리뷰 500 + 평점 4.5 이상 검증 후 iLBiA 번들 편입이 정석.

**결론: 오모모로 단품 론칭 → 6개월 운영 데이터 보고 → iLBiA 편입 여부 결정 (2단계 진입).** 보리의 시너지 주장은 Phase 2에서 유효.

## 2. 모리의 "광고 의존도 급등 경고" (49→73) — 리스크 신호로 수용

**받아들인다. 이건 오가닉 창이 닫히는 강한 신호.**

- 보리는 "측정 시점 차이"로 해석했지만, 픽시는 **구조적 포화 신호로 더 무겁게 봄**
- 근거: 광고 의존도 급등 + 신제품률 하락(27.5 → 20.0%) **동반 발생** = 신규 진입자가 광고로 밀어붙이다 떨어져 나가는 사이클
- 리스크 조정: **진입 시점을 최대한 앞당겨야 함** (봄 진입 → 여름 피크 캐치). 한 분기 미루면 광고비 ROAS 구조적 악화 가능
- 실무 반영: **초기 광고 예산을 "의존도 60% 기준"으로 책정**. 키노 마진 64%에서 **광고비 15% 흡수 여력** 확보해야 (아래 4번 연결)

## 3. Top 상품 마케팅 카피 리스크 — KIPRIS 사전 조회 분류코드

Top10에 **"완벽차단", "7세대", "끝판왕", "냄새제로"** 등 **실증 불가 최상급 표현 + 세대 넘버링**이 만연. 후발주자 진입 시 2개 법적 리스크 동시 발생.

### 리스크 A. 표시광고법 (공정위)
- **"완벽차단/100% 차단"** → 실증자료 없으면 **부당표시 행위** (표시광고법 제3조 1항)
- **"7세대/끝판왕/냄새제로"** → 기만적 표시 가능성. "세대" 표현은 **기술 진보 실증** 요구됨
- **안전 카피 가이드**:
  - ❌ "완벽차단", "100%", "냄새ZERO"
  - ⭕ "냄새 차단 설계", "악취 역류 저감", "이중 실리콘 구조"
  - ⭕ 실리콘 역류방지 테스트 영상(1회 촬영)으로 실증자료 확보

### 리스크 B. 특허·실용신안 침해 (KIPRIS 사전 조회 필수)
**분류코드 (IPC/CPC):**
- **IPC E03F 5/04** — 하수구/배수구(drainage inlets, gully traps) **핵심 분류**
- **IPC E03C 1/12** — 세면대/싱크대 트랩(traps for wash basins)
- **IPC E03C 1/28** — 배수구 냄새 방지(stench-traps)
- **IPC E03D 9/00** — 변기 부속품 (싱크대/세면대 확장 SKU)
- **IPC F16K 15/14** — 체크밸브/역류방지 밸브 (실리콘 디스크 방식)
- **IPC F16K 15/20** — 플랩/덮개식 역류방지

**검색 키워드 (한국어):**
- `하수구트랩`, `배수구트랩`, `악취차단`, `역류방지`, `실리콘디스크`, `오수역류`

**검색 사이트:**
- KIPRIS (http://www.kipris.or.kr) — **실용신안 + 디자인권 동시 조회**
- 특히 **디자인권(design patents)** 주의 — 실리콘 꽃잎형/다단 플랩 형상은 디자인권 등록 빈번
- **"7세대/8세대" 작명권** → 상표권 KIPRIS 상표검색 별도 조회 (류 11류·21류)

**실무 권고:**
1. 1688에서 공장 샘플 3종 받기 → **샘플 사진 + 단면도**로 KIPRIS 조회
2. 특허 1건이라도 걸리면 **설계 변경 or 다른 공장**으로 회피
3. 오모모 브랜드로 론칭 시 브랜드명 "OMOMO" 또는 부제명(예: "OMOMO 드레인가드") 상표 출원 동시 진행 (11류·21류 10~15만원)

## 4. 키노 마진 64% — 리스크 흡수 여력 있는가?

**여유 있다. 단, 광고비 흡수 후 실질 마진 40% 선을 지켜야 진입 정당성 확보.**

### 리스크 계상 후 재산정 (판매가 16,900원 기준)
| 항목 | 키노 원안 | 픽시 리스크 조정 | 차이 |
|---|---|---|---|
| 판매가 | 16,900 | 16,900 | - |
| 원가 CIF | -820 | **-970** | 멀티사이즈 어댑터 3종 동봉(+150) |
| 쿠팡 수수료 | -1,690 | -1,690 | - |
| 로켓그로스 | -3,100 | -3,100 | - |
| 포장 | -350 | **-500** | 설치가이드 인쇄+실증자료 QR |
| **광고비 (신규)** | 0 | **-2,535** | **광고 의존도 60% 대응 (판매가 15%)** |
| **반품 충당** | 0 | **-507** | 반품률 6% × 원가왕복 가정 (판매가 3%) |
| **특허 회피 설계 변경** | 0 | **-85** | 공장 변경 리스크 금형 소액 할증 (판매가 0.5%) |
| 순이익 | 10,940 | **7,513** | |
| **마진율** | 64.7% | **44.5%** | **여전히 40% 상회 ✅** |

**해석:**
- 리스크 풀 계상 후에도 **44.5% 마진** 확보 = **키노의 64% 쿠션은 튼튼함**
- 단 **광고 의존도 72 시나리오**(모리 비관)로 더 강하게 잡으면 광고비 -3,380원 → 마진 **39.6% (40% 턱걸이)** — 여유 거의 사라짐
- **안전 판매가는 17,900원** (+1,000원 프리미엄) → 16,900원 대비 순이익 +900원 = 마진 46.5% (비관 시나리오에서도 42% 유지)

**결론: 마진 64% 쿠션은 리스크 3종(광고·반품·특허 회피) 모두 흡수 가능. 단 비관 시나리오 대비 판매가 17,900원 상향 권장.**
