# 픽시 🦋 — 배수구 트랩 진짜 데이터 재토론

## 진행 권장도: 6/10 (1차: 7) ↓1

1차 7점은 "단품 시장, 카피 만연 회피만 하면 OK"라는 전제였음. 진짜 데이터 보니 **2개 세트가 표준 + 저가 골리앗 3개 + 5개 세그먼트 혼재** → 카피·광고법 리스크 면적이 1차 가정보다 2배 넓음. 그래도 세면대 팝업 단독 세그먼트(8,900원)는 청정 지역이라 조건부 진행 가능.

---

## 5개 세그먼트별 리스크 분석

### 1. 세면대 팝업 트랩 (루엔티스 8,900원) — 🟢 **리스크 낮음**
- 팝업 = 설치형 기구(세면대 배수구에 삽입). KIPRIS 분류 **E03C 1/22** (세면대 배수 마개/팝업) → **일반 하수구 트랩(E03F 5/04)과 다른 IP 카테고리**
- 카피 오염도 **낮음** — "100%", "7세대", "냄새제로"가 이 세그먼트엔 거의 안 붙어 있음 (루엔티스 리뷰 27개 = 신생)
- 리스크: 설치 호환성 클레임 (세면대 배수구 지름 32/40mm 차이) → 상세페이지에 치수 명기 필수
- 전안법/KC 의무 **없음** (기계식 부품)

### 2. 욕실 하수구 메인 — 🔴 **리스크 최고**
- "7세대", "100%악취차단", "냄새제로" 카피가 1·3·4등에 다 박혀 있음
- 후발주자가 유사 표현 쓰면 **표시광고법 제3조(부당한 표시·광고)** + **공정위 비교광고 심사지침** 동시 위반 가능
- 저가 골리앗 3개(이츠원츠 2,845원·리빙트리 3,495원·에코바스 2,995원)와 1매 단가 경쟁 불가 (대표님 8,900원 → 3배)

### 3. 세탁기 배수구 — 🟡 **중간 리스크**
- "1+1 세탁기 호스 연결형 11,900원" = 사실상 신규 틈새
- 리스크: **호스 규격(24/28/32mm) 오인 구매** → 반품률 상승 예상
- 카피 오염도는 낮은 편이지만 "누수 100% 차단" 유혹 회피 필수

### 4. 유가/봉수 — 🔴 **진입 불가**
- 해리 봉수 트랩 3,200원 단품 = 바닥 가격대
- 8,900원으로 진입하면 "왜 2.8배?" 설명 불가 → 전환율 붕괴

### 5. 싱크대 겸용 (YOPENCE) — 🟡 **중간**
- "겸용" 표현이 식품접촉면 오인 유발 가능 → **식품위생법 제9조(기구·용기 기준)** 저촉 리스크
- 진입하려면 "식기 직접 접촉 아님" 명시 필수

## 카피 회피 필수 키워드 (1차 + 확장)

**1차 경고 유지**: "7세대", "완벽차단", "100%", "냄새제로", "ZERO"

**확장 (실제 데이터에서 추출)**:
- "악취차단" 단독 사용도 위험 — "악취 저감에 도움" 식으로 완화 필수
- "끝판왕" (에어스케이프) — 최상급 표현, 표시광고법 제3조 위반 소지
- "꽉막아" (홈뉴) — 절대적 효과 암시
- "프리미엄" (살림언니) — 객관 근거 없이 쓰면 위반
- "벌레 제로" — **해충 차단 효과 표방 시 생활화학제품 안전기준법 저촉 가능** (살충 효능 광고는 의약외품/방역용품 인허가 영역)
- "안티스멜", "스멜컷" — 영문 조어도 실질적으로 "냄새제로"와 동일 해석됨

**안전 대안 표현**:
- "배수구 냄새 고민을 줄여주는" / "설치 간편" / "내구성 향상 설계" / "물 흐름 방해 최소화"

## KIPRIS 추가 분류코드

| 세그먼트 | IPC 분류 |
|---|---|
| 욕실/세탁기 하수구 트랩 (1차) | **E03F 5/04** (Drain traps) |
| 세면대 팝업 트랩 (신규 발견) | **E03C 1/22** (Plug stoppers for wash basins), **E03C 1/23** (Pop-up waste valves) |
| 실리콘 팩킹/밸브 구조 | **F16K 15/14** (Check valves with flexible members) |
| 악취 차단 플랩 | **E03C 1/298** (Trap seals with flexible flap) |

→ 대표님 8,900원 루엔티스가 팝업형이면 **E03C 1/22 + E03C 1/23** 선행기술 조사 추가 필수. 1차 브리핑(E03F 5/04)만 봤으면 놓쳤을 영역.

## 구성별 안전도

| 구성 | 광고법 안전도 | 비고 |
|---|---|---|
| **단품 8,900원** | 🟢 안전 | 세면대 팝업 단독이면 가격 단순·오인 없음 |
| 1+1 (동일품 2개) | 🟡 주의 | "1+1" 표기는 OK지만 할인율 표기 금지 (원가 허위 리스크) |
| 2개 세트 (표시가/매수) | 🟡 주의 | 1매 단가 병기 의무는 없지만 쿠팡 알고리즘이 세트단가로 노출 → 2,845원 골리앗과 직접 비교당함 |
| 4개 세트 (살림언니식) | 🔴 위험 | "묶음 할인" 표현 시 원가·할인율 증빙 필요, MOQ 1,000개면 재고 회전 부담 |

**결론**: 세면대 팝업 8,900원은 **단품으로 가는 게 광고법+재고 양쪽 다 안전**. 세트 구성은 욕실 메인 진입 시에만 고려, 그마저도 후순위.

## iLBiA 브랜드 진입 시 리스크 (대표님 고민 중)

### 🔴 iLBiA 진입 시 리스크
1. **카테고리 정체성 훼손**: iLBiA = 세제/탈취제 계열. 배수구 트랩(기계부품) 붙으면 "브랜드가 뭐 하는 회사?" 혼선 → 소비자 신뢰도 지표(브랜드 서치량) 희석
2. **표시광고법 연좌제**: iLBiA 제품 하나에 "100%" 카피 걸리면 **전 라인업 리뷰 신뢰도 동반 하락** (쿠팡 셀러 평판 지표)
3. **반품/설치 클레임 리스크**: 기계식 부품은 세제 대비 반품률이 구조적으로 높음 → iLBiA 셀러 점수에 직접 타격
4. **KC 리스크 전이**: 향후 iLBiA가 생활화학제품(탈취제) 심사 받을 때, 동일 셀러 계정의 기구류 카피 이슈가 조사 대상에 포함될 수 있음

### 🟢 오모모 진입 시 이점
1. 오모모 = 아이디어 유통 브랜드 → 카테고리 자유도 높음
2. iLBiA 본진(세제/탈취) 보호
3. 실패해도 iLBiA 브랜드 자산 무손상

**픽시 결정**: **iLBiA 진입 금지. 오모모 전용.**

## 진짜 추천

### 안전한 세그먼트
**세면대 팝업 트랩 (루엔티스 포지션) + 오모모 브랜드 + 단품 8,900원**
- 이유: 카피 청정 지역 + 다른 IP 분류 + 저가 골리앗 없음
- MOQ 1,000개 → 월 40~60개 판매 가정 시 16~25개월 재고. **MOQ 500개로 협상하거나, 500개 1차 발주 후 반응 보고 2차**가 현실적

### 회피할 카피 (상세페이지/광고 소재 공통)
- 금지: "7세대", "100%", "완벽", "ZERO/제로", "끝판왕", "꽉막아", "벌레 제로", "냄새 박멸/퇴치"
- 주의: "프리미엄", "악취차단" (단독 사용 금지, "저감에 도움" 수준으로)
- 안전: "설치 간편", "세면대 팝업형", "실리콘 팩킹 적용", "배수 흐름 고려 설계"

### 필수 상세페이지 요소
1. 세면대 배수구 지름 호환표 (32/38/40mm)
2. 재질 명시 (ABS/스테인리스/황동)
3. "식품 직접 접촉 불가" 고지 (싱크대 겸용 회피)
4. 살충·살균 효능 **미표기** (생활화학제품법 저촉 방지)
