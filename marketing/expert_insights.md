### 1. 브랜드 전략 — "판매만을 위한 마케팅은 이제 그만"

**브랜드 빌딩 6:4 원칙** (서기석, 前 이케아CMO)
- 브랜드 빌딩(장기) : 퍼포먼스(단기) = **6:4** 예산 배분
- 단기 매출만 쫓으면 광고비 의존도가 높아져 자멸
- 브랜드 캠페인은 최소 6~12개월 후 효과 → 꾸준함이 곧 전략

**Mental Availability** — 소비자가 "건조기 시트"를 떠올렸을 때 자동으로 일비아가 연상되도록

**브랜드 스토리텔링** (이광배, 콜린스)
- "Lead with the story, not the product" — 제품 설명보다 감정적 내러티브가 먼저
- 상세페이지 상단: 브랜드 철학과 감성 서사 → 하단: 성분/인증/비교표
- 차별화는 기능(복제 가능)이 아닌 **독특성(복제 불가)**에서

**일비아 포지셔닝 선언문** (박재현)
> "일비아는 **안전한 성분을 고집하는 부모**를 위해 **가족이 안심하고 쓸 수 있는 생활세제**를 만드는 브랜드입니다"

### 3. 쿠팡 전략 — "로켓그로스가 알고리즘에 올라타는 전략" (권동동, OSC, 장사왕, 이강로, 서형윤, 김서현, 이지현, 박혜연)

**쿠팡 SEO 상위노출 7가지** (OSC)
1. 카테고리 정확도 (최우선)
2. 상품명 핵심 키워드 배치
3. 상품 속성 100% 입력
4. 이미지 최적화 (화이트 배경 + 실사용)
5. 판매 속도(Velocity) — 초반 쿠폰 투입
6. 리뷰 10개 이상 (노출 가중치 급상승)
7. CPC 광고로 초기 트래픽 확보

**노출점수 3요소** (이강로, 연매출 100억)
- 출고일(배송속도) + 상품점수(필수항목 정확 입력) + 가격경쟁력 → 80점 이상 유지
- 신상품 등록 후 초기 2주 = 알고리즘 부스트 구간 → 이 기간 광고+쿠폰 집중
- 메인 상품 3~5개 집중 육성 > 상품 수 늘리기 (리뷰 축적이 복리로 작용)

**상품명 최적화** (서형윤)
- 키워드 3단계: 메인(월 검색 5만+) + 세부(1~5만) + 롱테일(1만 미만)
- "건조기시트"보다 "건조기 섬유유연제 시트 향기 오래가는" 형태가 실구매 전환율 높음
- 소비자 언어로 작성 — 리뷰에서 자주 쓰이는 단어를 상품명에 반영

**로켓그로스 핵심**
- 건조기 시트·캡슐 세제 = 소형 분류 → 출고비 800원 수준, 마진 유리
- 30일 완판 기준 입고 (보관비 폭증 방지)
- 로켓배지 = CTR 30%+ 향상 효과
- 구독 기능 연동 → LTV 극대화 (소모품 특성)
- 물류센터 2~3곳 분산 입고 → 수도권 외 지역 노출 확대 (김서현)

**쿠팡 광고 운영**
- 초기 ROAS 300~350%에서 시작 → 2~4주 학습 후 400~500%로 상향
- 재구매율 높은 소모품은 초기 ROAS 200%도 허용 (LTV 고려) (박혜연)
- 골든타임 예산 집중: 오후 6~9시 → ROAS 평균 30% 향상
- AI 스마트광고로 시작 → 2주 데이터 쌓이면 수동 전환해 비효율 키워드 제거
- 캠페인 고도화: ①일예산 설정 → ②주요 키워드 CPC 조정 → ③비효율 제외 → ④성과 키워드 10~20% 증액
- 키워드 분류: 고효율(400%+) 집중 / 중간(200~400%) 관찰 / 저효율(200% 미만) 제외
- 번들 상품(2팩, 3팩) → 객단가 상승 + 물류비 절감
- 쿠팡 쇼츠(짧은 영상) 등록 시 체류시간↑, 전환율 15~20% 향상 (김서현)

**핵심 상품 집중 전략** (이지현, 월 1.5억 셀러)
- 20/80 법칙: 등록 상품 중 상위 20개가 전체 매출 80% → 핵심 상품에 광고·재고 집중
- 신규 상품은 위탁으로 시장 반응 테스트 후 검증되면 사입 전환
- 최저가 경쟁 대신 "공식 스토어" 인증 + 정품 보증 → 가격 대신 신뢰로 경쟁

### 4. 네이버 전략 — "적합도 + 인기도 + 신뢰도" 3축 관리 (단아쌤, 김용빈, 서미진, 김도균, 정진수, 최재혁, 박노성)

**스마트스토어 상위노출**
- 상품명 = [브랜드] + [핵심기능] + [제품명] + [용량/규격] 순서 (서미진)
- 상품명 앞 15~20자가 핵심 — 중복 키워드·특수문자 삽입은 페널티 (서미진)
- 경쟁 상위 상품과 **동일 카테고리** 설정 필수 + 최하위 카테고리까지 정확 지정
- 스토어 '알림받기(소식받기)' 기반 CRM 전환 → 신상품·기획전 고지로 재구매 유도

**계단식 순위 상승 전략** (김도균, 빅파워셀러)
- 경쟁 낮은 세부 키워드(예: "드럼건조기용 시트 무향")로 먼저 실적 축적
- 상위 키워드(예: "건조기 시트")는 실적 쌓인 후 알고리즘이 자동 확장
- 신상품 최신성 버프(2~4주) 기간에 쇼핑검색광고 집중 → 리뷰 10개 확보가 결정적

**3단계 매출 성장 공식** (서미진)
- 노출 → 클릭 → 전환 각 단계 독립 분석
- CTR 낮으면 대표이미지·가격 문제 / 전환율 낮으면 상세페이지·리뷰 부족

**SNS-스마트스토어 연결** (정진수, SNS국가대표)
- 네이버 블로그 후기 → 통합검색 노출 → 외부 트래픽이 인기도 점수에 반영
- AiTEMS(AI 개인화 추천) 공략: 클릭→구매 전환율이 높아야 추천됨
- 데이터랩 쇼핑인사이트로 계절 수요 사전 포착 (건조기 시트: 가을~겨울 검색 2~3배)
- 틈새 키워드 황금비율: 월 검색 1,000~5,000 + 경쟁 상품 300개 미만

**네이버 광고** (최재혁)
- 쇼핑검색광고: 확장소재(추가제목·홍보문구·가격링크) 적극 활용
- 품질지수 = CTR + 상품 관련성 → 같은 입찰가에도 CTR 높으면 상위 노출 + CPC 절감
- 소규모 브랜드: 쇼핑검색광고 80% + 파워링크(브랜드명 방어) 20% 배분
- 키워드 3분류: 브랜드("일비아") / 카테고리("건조기 시트") / 니즈형("섬유 냄새 제거")
- ROAS 300% 기준선: 달성 키워드만 예산 집중, 미달 키워드 즉시 일시정지
- 쇼핑커넥트: 외부 소재 연동 추가 노출

**브랜드스토어 전략** (박노성)
- 입점 조건: 상표등록 + 정품 보증 + 일정 판매 실적
- 검색 결과 최상단에 브랜드 전용 영역 → CTR 일반 스토어 대비 1.5~2배
- 기획전 참여: '친환경 생활용품', '주방/청소/세탁' 테마 → 인기도 단기 부스팅
- 상세페이지 FABE 공식: Feature→Advantage→Benefit→Evidence
- 브랜드 키워드 검색량 월별 트래킹 → SNS/블로그로 검색 수요 인위적 생성

**재구매율 관리** — 자동 재구매 쿠폰으로 22% → 32% 달성 사례 (김용빈)

### 5. 메타 광고 전략 — "소재가 타겟팅이다" (송재욱, BAT, AMPM, 장건혁, 심재완, Foxwell, Heath, Shackelford)

**캠페인 구조**
- Advantage+ 쇼핑 캠페인(ASC) 적극 활용 (소규모 브랜드도 AI 최적화 혜택)
- ASC + BAU(기존 캠페인) 밸런스: 초기 BAU 60% : ASC 40% → 성과 확인 후 ASC 비중 상향 (심재완)
- 단일 CBO에 신규+리타겟팅 오디언스 함께 → Meta AI가 예산 배분 최적화 (Foxwell)
- 일 예산 최소 5~10만원 확보 → 알고리즘 학습 가능
- 월 400만원 이하라면 2주 집중 집행이 1달 분산보다 효과적 — 빠른 신호 수집이 학습 속도↑ (Foxwell)

**소재 제작 원칙** — "크리에이티브 = 새로운 타겟팅" (Foxwell)
- 메타 AI는 소재 자체를 보고 "누구에게 보여줄지" 판단 → 새 고객 세그먼트 확장 = 새 소재 제작
- **크리에이티브 믹스 5종**: ①대표가 아이폰으로 직접 찍은 영상 ②고품질 브랜드 광고 ③고객 반론 해소 ④다양한 페르소나 ⑤AI 생성 소재 10~15% (Foxwell)
- "못생긴 광고(ugly ads)"가 완성도 높은 광고보다 전환율 높은 사례 다수 → 진정성 > 완성도 (Shackelford)
- 처음 3초 후킹: "문제 제시 → 해결책 → 증거" 구조
- UGC 스타일 짧은 영상 > 정적 이미지 (CTR 2~3배 차이) (심재완)
- 동일 제품 소재 **최소 5개** 동시 운영 → 7일 내 CTR 기준 미충족 시 폐기, 충족 시 확장
- 향별 크리에이티브 각각 제작 필수 (코튼블루/베이비크림/바이올렛머스크 × 다른 소구점)
- 매월 신규 소재 2~3개 "통합 테스트" → 기존 승리 소재와 비교 (심재완)

**ABCD 수익성 진단** (Shackelford)
- AOV(객단가) → Business Economics(원가/마진) → Creative Conversion(소재 전환율) → Distribution(미디어 바잉)
- ROAS가 낮으면 소재 문제인지 구조 문제인지 먼저 구분
- 객단가 낮으면 번들/묶음으로 AOV 선 개선 후 광고 집행

**기술 기반 세팅** (장건혁)
- CAPI(서버사이드) + 픽셀 이중 연동 필수 — iOS 이후 30~40% 전환 누락 보완 (Heath)
- 중복 제거(deduplication) 설정 필수 — CAPI는 픽셀 보완이지 대체 아님
- 카페24 자사몰: 카페24 메타 픽셀 앱에서 CAPI 연동 지원 (무료)
- Business Manager 인증 먼저 완비 (도메인 인증 + 픽셀 + CAPI) → 계정 제한 리스크 방지
- Attribution: 7일 클릭 / 1일 뷰가 소규모 계정에서 가장 안정적 (Foxwell)

**타겟팅**
- 초기: 광범위(브로드) 오디언스로 데이터 수집
- 구매 이벤트 50건 이상 쌓인 후 최적화 시작 (Heath)
- 협력광고(쿠팡·오늘의집 연동) → 플랫폼 모수 활용 리타겟팅
- 리타겟팅 예산: 전체의 15~20%, 장바구니/방문 이탈 오디언스 최우선

**콘텐츠 × CRM × 메타 3각 설계** (심재완)
- SNS 콘텐츠로 신규 노출 → 메타 광고로 유입 → CRM(카카오/이메일)으로 재구매
- 생활세제 소모품 특성상 이 3각 구조가 LTV 극대화에 최적

**경쟁사 벤치마킹** — Meta 광고 라이브러리에서 3~6개월 이상 장기 집행 중인 광고 = 성과 좋다는 신호 (Heath)

**핵심 지표**: CPM, CTR, ROAS(구매전환), 빈도수(3 이하 유지), CAPI 매칭률(80%+ 목표)

**예산 배분** (월 100~500만원 기준)
- 메타 50~60% / 네이버 쇼핑 30~35% / 쿠팡 10~20%
- 채널 분산보다 **1~2개 채널 집중 마스터 후 확장**
- 첫 달은 ROAS 기대 접고 "학습 데이터 수집" 목적으로 집행 (Heath)

### 6. 바이럴/SNS 전략 — "커뮤니티 신뢰가 곧 구매력"

**인플루언서 시딩** (BAT크루, 김건우)
- 나노(1천~1만): 제품 무료 제공만으로 리뷰 확보
- 마이크로(1만~10만): 제품 + 10~30만원 → 평균 60% 높은 참여율
- 선정 기준: 팔로워 수보다 **30~40대 여성 비율 60%+, 참여율 3%+, 살림/육아 카테고리**

**체험단 전략** (태그바이, 레뷰)
- 월 30~50만원으로 블로그+인스타 체험단 30~50명 확보
- 리뷰 1건 = 평균 검색 유입 3~5명 → 50건 = 월 150~250명 추가 유입
- 대형 프로모션 **2주 전** 시딩 → 검색 수요 급증 시 리뷰가 상위 노출

**맘카페 바이럴** (아이보스)
- 맘카페 추천은 광고보다 **10배 강한 전환력**
- 침투 4단계: 소규모 지역 맘카페 시딩 → 자발적 후기 유도 → 핫딜 등록 → 확산
- 노골적 광고 글 = 삭제+차단 → 반드시 **자연스러운 경험담 형식**

**SNS 콘텐츠**
- 인스타 릴스: Before/After 세탁 비교 (생활용품 최고 저장율)
- 유튜브 숏폼: "주부가 진짜 쓰는 세제" 리얼리티 포맷
- 블로그: "건조기 시트 추천", "식기세척기 세제 비교" 등 SEO 키워드 공략

**소규모 예산 플랜**
| 월 예산 | 전략 |
|---|---|
| 30만원 | 나노 10명 시딩(무료) + 맘카페 직접 후기 |
| 50만원 | + 레뷰/태그바이 체험단 1회 + 마이크로 1명 유료 |
| 100만원 | + 릴스 부스팅 광고 30만원 + 라이브커머스 1회 |

### 7. 상세페이지 — "브랜드 미니사이트" (박재현, 노준영, 김용훈)

**구조 (위→아래)**
1. 감성 이미지 + 핵심 브랜드 메시지 ("가족을 위한 안전한 선택")
2. 문제 제기 ("기존 세제의 강한 향, 걱정되지 않으셨나요?")
3. 해결책 (제품 소개 + 차별점)
4. 사회적 증거 (리뷰 수, 재구매율, "○만 가정이 선택")
5. 성분/인증 (KC인증, 피부 테스트 결과 시각화)
6. 사용법 + 비교표 (경쟁 제품 대비)
7. 구매 유도 (묶음 할인, 정기배송 혜택)

**핵심**: 리뷰에서 자주 나오는 키워드를 상세페이지 헤드라인으로 활용 (김용훈)

### 8. 리텐션/재구매 — "첫 구매보다 재구매가 3~5배 효율적"

- 건조기 시트·세제 = **정기구독 전환율 높은 소모품** → 쿠팡 정기배송, 네이버 구독 필수
- 구매 후 CRM: 구매완료 → 사용법 안내 → 재구매 타이밍 알림 → 리뷰 요청
- 재구매 쿠폰 자동화
- 광고 ROAS보다 **LTV 기준 CPA** 설정 (김용훈)

### 9. 시즌/이벤트 캘린더 (김용빈, 단아쌤)

| 시기 | 수요 | 액션 |
|---|---|---|
| 2월 | 봄맞이 대청소 | 세제류 프로모션 |
| 3~4월 | 황사 → 세탁 급증 | 재고 선제 확보 |
| 6월 | 여름 에어컨 청소 | 세제 신제품 런칭 적기 |
| 9~10월 | 가을 이불 세탁 | 건조기 시트 시즌 |
| 11월 | 블프/쇼핑 메가세일 | 전 채널 프로모션 총동원 |
| 명절 전 | 선물세트 수요 | 묶음 상품 객단가 상승 |

**시즌 소싱은 최소 3개월 전** / 시즌 피크 4주 전 로켓그로스 입고 완료

### 10. 오픈마켓 전략 — "보조 채널이지만 현금회전과 MD 레버리지의 보물" (이길만, 전진수, 강태균, 김아름, 김호진)

**플랫폼별 특성 차이**
| 플랫폼 | 딜 구조 | 알고리즘 핵심 | 수수료 |
|---|---|---|---|
| 11번가 | 쇼킹딜(타임딜), 슈팅배송딜 | 최근 판매량 + 구매만족도(셀러 평점) | 8~12% |
| G마켓 | 슈퍼딜, G9 투데이특가 | 등록 완성도 + 스마일배송 여부 | 9~12% |
| 옥션 | 빅딜 | 가격 경쟁력 + 비교쇼핑 세그먼트 | 9~12% |

**딜 행사 승인 전략** (이길만, 前 11번가 광고팀)
- 11번가 쇼킹딜: 동일 카테고리 최저가 대비 -5~10% 제안 → 용량 번들로 단가 조정이 승인률 높음
- G마켓 슈퍼딜: 카드사 할인 연계로 체감 가격↓
- 스마일배송 등록 시 노출 가산점 → 배송 단순한 소비재는 반드시 전환

**ESM+ 통합 운영** (전진수, 강태균)
- 향 3종 각각 별도 상품으로 등록 + 키워드 세분화 (동일 상품도 키워드별 복수 등록 가능)
- 상품명 앞 15자에 핵심 키워드 집중 + 구어체 롱테일 적용
- 정산 주기 주 1회(쿠팡 월 2회보다 빠름) → 현금회전 유리

**MD 컨택 & 기획전** (김아름)
- MD 제안서 필수 항목: ①카테고리 시장 규모 ②경쟁 대비 차별점 ③예상 월 판매량 ④기획전 가격
- 신규 셀러는 파트너센터 '셀러 직접 기획전 신청' 먼저 → 데이터 확보 후 MD에 제시
- 오픈마켓 판매 데이터 = 향후 오프라인(백화점/마트) MD 협상 레버리지

**광고 운영** (이길만)
- 신규 셀러: 모바일 파워클릭(G마켓)부터 최저 입찰가로 ROI 테스트 후 확장
- 11번가: 포커스클릭 → 초기 리뷰 30개 확보 후 집행해야 ROAS 2배 차이

**수익 구조 설계** (김호진)
- 오픈마켓 수수료(10~12%) + 광고비(5~10%) + 반품/CS(3~5%) → 공급가 대비 판매가 최소 2.5배
- 오픈마켓 마진율이 쿠팡 로켓보다 5~8%p 높은 경우 있어 가격 설정 자유도 높음

**오픈마켓 가격 = 쿠팡 대비 +3~5%** → 가격비교 유입 방어 + 쿠팡 메인채널 보호

### 11. 멀티채널 운영 원칙 (장사왕)

- 가격 체계: **쿠팡 기준가 > 오픈마켓 ≥ 스마트스토어** (배송 신뢰 프리미엄)
- 한 채널 할인이 타 채널 가격 신뢰를 깨는 리스크 관리
- 주력 재고 = 쿠팡 / 보조 = 오픈마켓 / 재고 소진 = 오픈마켓 타임딜
- 통합 관리 솔루션 도입 → 17채널 일괄 등록·가격·재고 연동
