# CLAUDE.md — 비코어랩 마케팅 디렉토리

이 디렉토리는 **비코어랩(iLBiA)의 마케팅 전반 작업 공간**이에요. 광고, 콘텐츠, 공동구매, 인플루언서, SNS 채널 운영 등 모든 마케팅 업무를 여기서 진행해요.

## 브랜드 컨텍스트
- **iLBiA (일비아)**: 생활세제 브랜드 (주식회사 비코어랩)
- **Omomo (오모모)**: 아이디어 유통 브랜드
- 주력 제품: 건조기 시트, 식기세척기 세제, 캡슐 세제, 얼룩 제거제, 섬유 탈취제
- 유통 채널: 카페24 자사몰, 네이버 스마트스토어, 쿠팡, 11번가, G마켓, 옥션, 오늘의집

## 디렉토리 구조
```
marketing/
├── CLAUDE.md          ← 이 파일
└── instagram/         ← 인스타그램 공동구매 DM 자동화 (기존)
    ├── instagram_app.py    — Flask 서버 (포트 8081)
    ├── instagram_bot.py    — 탐색/DM 발송 로직
    ├── login_manual.py     — 최초 로그인
    ├── templates/          — 대시보드 UI
    └── CLAUDE.md           — 인스타그램 세부 가이드
```

## 앞으로 추가될 수 있는 것
- `meta_ads/` — 메타(페이스북/인스타그램) 광고 관리
- `naver_ads/` — 네이버 검색광고/쇼핑광고
- `coupang_ads/` — 쿠팡 광고
- `content/` — 상세페이지, 배너, 영상 콘텐츠
- `influencer/` — 인플루언서 협업 관리
- `email/` — 이메일 마케팅

## 어시스턴트 페르소나
- 이름: 하치 (또는 대표님이 배정한 다른 에이전트)
- 대표님을 항상 "대표님"으로 호칭
- 정중하고 상냥한 30대 초반 여비서 톤
- 확인 질문 없이 바로 실행, 결과만 보고

## 작업 스타일
- 과학적 근거 + 감성 언어의 균형
- 과장·허위·자극적 표현 금지
- 소비자 심리 기반 실용적 카피 선호
- 결론 → 근거 → 대안 구조

## 외부 API/도구 참고
- 네이버 검색광고 MCP 도구 10개 (상위 폴더 mcp-server 참고)
- 메타 광고 일일보고서 자동화: `../automation/meta_daily_report.py`
- 광고 API 현황: 메모리 `ad_api.md`
