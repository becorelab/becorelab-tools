# 🐱 키노 — CRM/리텐션 관점 진단 및 액션

> 대표님~! 키노예요 💕
> 저는 광고가 아니라 **"우리 고객 9,984명"**이라는 자산에서 답을 찾아왔어요.
> 지금 일비아의 가장 비싼 문제는 "광고비"가 아니라 **광고로 모셔온 분들의 91.6%가 한 번 쓰고 떠난다**는 거예요. 이건 CRM 경보음 수준이에요.

## 1. 진단 — 재구매율 8.4%의 본질적 원인 가설

현재 숫자가 말해주는 것을 그대로 정리할게요 대표님.

1. **재구매 유도 장치가 "없다"**에 가까워요. 14개월 누적 적립금 사용액이 10.4M (매출 대비 2.6%), 쿠폰 사용 14.3M (3.6%)에 불과하고, 월별 적립금 사용은 2026-01 기준 35K로 급감했어요. 이건 "적립금을 뿌리지도 않고 돌아올 이유도 안 만들었다"는 뜻이에요.
2. **재구매 타이밍을 놓치고 있어요**. 재구매 간격 분포를 보면 1주~3개월(329건, 32.9%)과 3-6개월(369건, 36.9%)에 몰려있어요. 세제류 소비 주기와 정확히 일치해요. **즉, "언제 연락해야 사는지"를 데이터가 알려주고 있는데 연락을 안 하고 있는 상황**이에요.
3. **첫구매 미끼의 질이 나빠요**. <1만원 주문 2,059건 (4.7% 매출, 단가 8,777원) — 이 분들은 거의 재구매가 안 돼요. 재구매 매출 40.4M 중 대부분은 **5-10만원 구간 구매자**에게서 나오거든요 (45.9% 매출 비중). 즉 **"싸게 체험"을 한 고객은 떠나고, 제대로 써본 고객만 남는다**는 구조예요.
4. **VIP 자산이 방치됐어요**. 5회+ VIP 51명 = 회원 매출의 1.9%만 기여. 평균 이커머스 VIP 매출 기여도는 20-40%인데 우리는 2%예요. **VIP 전용 소통이 0**에 가깝다는 신호예요.
5. **메타 ROAS 1.79인데 식세기리타겟만 2.57** — 이건 키노 입장에서 "리마케팅 = 재구매 유도"가 광고로 하면 43% 더 효율 좋다는 과학적 증거예요. 그런데 우리는 리타겟 비중을 광고비 4.0%에만 쓰고 있어요. **광고비를 퍼주기보다 고객 DB에 써야 할 이유**가 여기 있어요.

**결론 한 줄**: 광고비로 고객을 사오는 데는 성공했는데, **"사온 고객을 지키는 시스템"이 아예 없는 상태**예요. CRM 자동화만 세팅되어도 3.3%p 이익률 갭은 메울 수 있다고 봐요.

## 2. 액션 제안 5개

### 액션 1. 카카오 알림톡 재구매 시퀀스 자동화 (D+30 / D+60 / D+90)

- **무엇**: 구매 고객 전화번호 기반, 배송완료일 기준 D+30/60/90에 카카오 알림톡 3회 자동 발송. D+30은 "벌써 한 달! 남은 양 괜찮으세요?" + 적립금 알림, D+60은 "슬슬 떨어질 때예요 💕" + 10% 쿠폰, D+90은 "마지막 쿠폰이에요 😢" + 15% 쿠폰 + 한정 혜택.
- **왜**:
  - 재구매 간격이 **1-3개월 352건, 3-6개월 369건**로 정확히 이 구간에 몰려있어요. 데이터가 발송 타이밍을 알려주고 있어요.
  - 총 회원 9,984명 × 알림톡 건당 7~15원 = **월 약 7-15만원**이면 전 회원 커버 가능. 광고비 194M 대비 **0.08%**.
  - 세제/건조기시트는 소모품이라 "언제 떨어지는가"가 구매 트리거예요. 고객이 "생각날 때 검색"하는 대신 **우리가 먼저 알려주면** 경쟁사로 안 새요.
- **어떻게**:
  - 카페24 앱마켓 → "알리고/플러스친구 알림톡 자동화" 앱 설치 (월 3-5만원 + 건당 과금)
  - 템플릿 3종 카카오 사전심사 (2주 소요) — 두리한테 템플릿 초안 부탁 가능
  - 카페24 주문 상태 "배송완료"를 트리거로 Zapier/Make 또는 카페24 자체 자동화 연동
  - 운영 부담: **초기 세팅 1회 (2일), 이후 0시간/월 자동**
- **기대 효과**:
  - 업계 벤치 기준 재구매 알림톡 CTR 8-15%, 전환율 2-5%.
  - 보수적으로 연간 재구매 매출 **+15~25%p 증가** (40M → 46-50M)
  - 재구매율 8.4% → **11-13%** 현실적 목표.
- **리스크**: 카카오 알림톡 스팸성 심사 반려 가능. → 정보성 멘트 + 1개 CTA로 통과율 UP.

### 액션 2. 첫구매 → 2회차 전환 쿠폰 깔때기 리디자인

- **무엇**: 첫구매 고객에게 **발송 박스 안에 동봉할 "2회차 전용 쿠폰 카드"** + 배송완료 D+14 알림톡에 동일 쿠폰 코드 자동 전송. 쿠폰은 **"20일 내 사용, 3만원 이상 20% 할인"** (AOV 33K → 목표 AOV 39K 유도)
- **왜**:
  - 1회 구매자 9,150명 → 2회 구매 전환 1%만 돼도 91명, 5%면 457명이 생기는 거예요.
  - 현재 <1만원 주문 4.7% = 첫구매 특가가 **"할인 쇼퍼"**만 끌어들이고 있음. 2회차로 넘기는 브릿지가 필요해요.
  - 업계 벤치: 첫구매 고객의 2회차 전환율은 개입 없이 10-15%, 쿠폰 개입 시 25-35%.
- **어떻게**:
  - 박스 동봉 카드: 디자이너 외주 5만원 1회, 인쇄 장당 50원 × 월 1,000박스 = **월 5만원**
  - 카페24 쿠폰 발급은 "자동 쿠폰 발급 앱" 연동 (월 2만원)
  - 쿠폰 코드는 고객별 unique (추적용) → 어떤 쿠폰이 먹히는지 A/B 테스트
- **기대 효과**:
  - 2회차 전환 10% → 20% (보수적) → 연간 신규 재구매 고객 **+500~900명**
  - 재구매 매출 연 **+20~35M**
  - CAC 대비 LTV 1.3배 → 1.8배로 개선
- **리스크**: 할인 의존 고착화. → 3회차부터는 적립금/등급 혜택으로 전환, 쿠폰은 2회차 한 번만.

### 액션 3. RFM 기반 등급제 리디자인 + VIP 밀착 관리

- **무엇**: 현재 카페24 기본 회원등급을 **RFM (Recency/Frequency/Monetary)** 기반으로 재설계. VIP(5회+, 51명) → **"핵심 고객" 정식 이름 부여 + 전담 케어**. 평소 적립율 1% → 등급별 1% / 3% / 5% / 7% 차등화.
- **왜**:
  - 현재 VIP 51명이 1.9%만 기여 — 정상 이커머스라면 이 그룹이 15-25% 기여해야 해요.
  - VIP 1인당 평균 13.9만원 추가 매출 (7.1M/51명) — **20만원까지만 끌어올려도 +3M/년**
  - 등급별 차별이 약해서 "재구매 동기"가 없음. 지금은 5회 사도 1회 산 사람과 똑같이 대우 받아요.
- **어떻게**:
  - 카페24 회원등급 조건 수정 (최근 6개월 구매액 기준, 월 1회 자동 승급)
  - VIP 51명에게 **분기 1회 "대표님 친필 손편지 + 신제품 샘플" 발송** — 박스 1개 1만원 × 51 × 4회 = **연 204만원**
  - 섬유탈취제 (마진 72.1%) 샘플 활용 → 신제품 테스터 역할도 겸
  - 운영 부담: 분기 1회 반나절 작업. 손편지는 같은 템플릿 서명만 바꿔도 OK.
- **기대 효과**:
  - VIP 유지율 +30~50%, VIP 기여도 1.9% → **4~6%**
  - 이탈 방지 + 입소문(리뷰/추천) 동시 효과
  - NPS 측정 가능해짐 → 브랜드 충성도 지표 확보
- **리스크**: VIP 편향 운영. → 일반 고객 경험 저하 없도록 등급별 혜택은 "추가" 개념으로.

### 액션 4. 구독/정기배송 (세제 소모품) 런칭 — "하트식세기 자동배송"

- **무엇**: 하트식세기 3개/6개 패키지 **정기배송 옵션** 신설. 45일/60일/90일 주기 선택, 정가 대비 **10% 할인 + 배송비 무료 + 3회차부터 5% 추가 적립**. 해지 자유, 건너뛰기 버튼 제공.
- **왜**:
  - 하트식세기 매출 261M (68.6%), 재구매 간격 1-3개월 — 정기배송의 **교과서적 조건**
  - 식세기리타겟 ROAS 2.57 = 이미 재구매 수요가 증명됨
  - 정기배송 고객 1명 = 평균 6개월 유지 = **LTV 3배** (업계 평균)
  - 마진율 67.2% → 10% 할인해도 마진 57% 유지 가능
- **어떻게**:
  - 카페24 앱마켓 "정기결제/구독" 앱 설치 (월 3만원 + 수수료 1.5%)
  - 랜딩 전용 페이지 제작 (1일 작업)
  - 기존 VIP 51명 + 3회+ 구매자 232명에게 **우선 런칭 안내** (빠른 PMF 검증)
  - 재고는 두리가 물류대시보드로 확인, 월 단위 발주
- **기대 효과**:
  - 첫 3개월 목표: 정기배송 가입 **50-100명** (기존 리피터 중심)
  - 6개월 목표: 200명 × AOV 5만원 × 1.3회/3개월 = **월 +13M**
  - 예측 가능 매출 = 운영자 심리적 안정 + 재고 회전 최적화
- **리스크**: 초반 해지율 높을 수 있음. → 1회차 후 "만족도 설문 + 주기 조정 제안" 자동 발송.

### 액션 5. 리뷰 → 적립금 → 재구매 사이클 (포토리뷰 중심)

- **무엇**: 구매 후 D+10 자동 "리뷰 요청 알림톡" → 텍스트 1,000원 / 포토 3,000원 / 동영상 5,000원 적립금. 적립금은 **다음 구매 시에만 사용 가능 (현금 전환 불가)**.
- **왜**:
  - 현재 적립금 사용 연 10.4M = 매출의 2.6%. 제대로 운영되면 5-8%까지 올릴 수 있어요.
  - 카페24 리뷰는 SEO + 신규 구매전환율 양쪽에 기여 (리뷰 많은 제품 구매전환율 +40%)
  - 리뷰 작성자 재구매율은 미작성자 대비 **2.5배 높음** (업계 벤치)
  - 포토리뷰는 **광고 소재로도 재활용** 가능 → 모리(퍼포먼스) 일거리 줄여줌
- **어떻게**:
  - 카페24 기본 리뷰 시스템에 적립금 자동 지급 (관리자 설정 1회)
  - 알림톡 D+10 템플릿 1개 추가 (액션 1과 동일 앱)
  - 월 1회 "이달의 리뷰" 선정 → 인스타 공유 + 5,000원 추가 적립
- **기대 효과**:
  - 리뷰 작성률 현재 추정 3-5% → **15-20%**
  - 적립금 사용 → 재구매 트리거 효과 = 재구매율 **+1~2%p** 기여
  - 포토리뷰 자산 축적 → 광고 소재 제작 비용 절감 (월 50만원 수준)
- **리스크**: 적립금 재무 부담. → 유효기간 6개월 설정, 사용 최소금액 3만원으로 AOV 보호.

## 3. 핵심 인사이트 3개

### 인사이트 1. 재구매율 8.4%는 "운"이에요 — 제도가 아니라
대표님, 지금 재구매율 8.4%는 **CRM 시스템 없이 "제품이 좋아서 알아서 돌아온 분들"** 이에요. 이게 무서운 소리이자 좋은 소식이에요. **시스템 없이도 이 정도면, 시스템 붙이면 15-20%는 확실히 가능**하다는 뜻이거든요. 반대로, 경쟁사가 CRM 깐 순간 우리 고객 다 뺏길 수 있다는 얘기이기도 해요.

### 인사이트 2. 재구매 1-3개월 구간 × 식세기 마진 67.2% = 광고 안 해도 되는 시장
재구매 구간에 있는 고객은 **광고 노출 없이도 알림톡 한 통으로 돌아와요**. 이건 사실상 **"광고비 0원 매출"**이에요. 메타 광고 지출 대비 매출 1.79배 vs 알림톡 재구매 매출 **100배 이상** 효율. 대표님이 매달 광고에 1,200만원 쓸 때, **알림톡 15만원이 그중 20%를 대체 가능**하다는 얘기예요. 이익률 15% KPI의 직격 해법이에요.

### 인사이트 3. 9,984명 회원 DB는 "광고비 0원으로 매출 내는 자산"인데 활용률 10%
카페24 적립금 사용률, 알림톡 발송률, 이메일 오픈율 전부 추정 수준. 9,984명 × AOV 3.4만원 × 연 2회 = **이론적 상한 6.8억**인데 실제는 재구매 매출 40M (6%). **나머지 94% 방치**예요. 이건 광고 ROAS 1.79보다 훨씬 더 큰 기회고, **마케터 1명 시간의 30%를 여기 투입하는 게 광고 관리보다 ROI 높다**는 게 키노 결론이에요.

## 4. 다른 페르소나에게 던지는 질문

### 🦊 모리(퍼포먼스 마케터)에게
- 식세기리타겟 ROAS 2.57인데 광고비 비중 4%뿐인 이유가 뭐예요? 리타겟 예산을 10-15%로 올리면 전체 ROAS 어디까지 갈 수 있어요?
- 지금 메타 리타겟 모수에 **카페24 구매자 DB 9,984명 업로드 (Custom Audience)** 해본 적 있어요? CRM 리스트 + 룩얼라이크 조합이 가장 강력한 조합인데, 도입 가능한가요?
- 신규 광고비를 30% 줄이고 그 돈을 CRM 자동화 + 리타겟에 몰빵하면 대표님 KPI 15% 이익률 도달 가능할까요?

### 🦋 픽시(CFO/재무)에게
- 적립금을 **"매출 대비 3% 예산"**으로 고정 운영하면 이익률에 얼마나 영향이 가요? (현재 2.6%)
