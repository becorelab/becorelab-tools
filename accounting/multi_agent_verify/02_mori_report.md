## 계산 공식 (행단위)
```
gross_row            = 수량 × 옵션+판매가
coupon/point/ship    = 주문번호 첫 등장행에만 적용 (주문당 1번)
refund/rank/app/extra= 모든 행에 적용
net_revenue          = gross - coupon - refund - point - rank - app - extra
net_revenue_with_ship= net_revenue + shipping
cost_row             = (cost_map[standard] 우선, 없으면 cost_map_monthly[YYYY-MM][standard]) × 수량
profit               = net_revenue - cost
margin_rate          = profit / net_revenue
```
