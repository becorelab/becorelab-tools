## 🎯 KPI 갭 정의 (픽시 손익공식)

```
이익률 = 마진율(광고 전) - 광고비/매출

현재:  60.0% - 47.9% = 12.1%
목표:  60.0% - 45.0% = 15.0%
```

→ **광고비를 매출의 45% 이하로 깎거나 마진율을 63%로 끌어올리거나** (또는 둘 합)

## 🚀 실행 액션 5개 (우선순위 순)

### 액션 1. 🚨 즉시 출혈 차단 (1주일 안에)
**=> 모리 + 픽시 + 보리 컨센서스**

**무엇을:**
- 좀비 트래픽 캠페인 `241115 Website Traffic Campaign` **즉시 OFF** (300일간 6.57M 소진, ROAS 0.03)
- 건조기시트 단독 광고 OFF (ROAS 0.80, 마진 43.9% → 적자 구조). 단, 식세기 묶음 소재는 유지
- 적자 SKU 5종을 사은품 라인업에서 **제거** → 섬유탈취제 100ml로 교체
  - 수세미 -46.3% / 틴케이스 -83.8% / 얼룩100ml -80.2% / 다목적세정제 -37.6% / 슈즈렉 -37.3%

**누가/공수**: 마케터 4시간 (1회성)

**기대 효과**:
- 광고비 절감 월 ~1M
- 사은품 손실 절감 연 2.2M
- **이익률 +1.0~1.5%p**

**리스크**: 사은품 체감 혜택 ↓ → 후기 적립금 인상으로 상쇄

### 액션 2. 🔄 CRM 자동화 깔기 (2-4주 안에)
**=> 4명 만장일치 ⭐**

**무엇을:**
- **카카오 알림톡 D+30/60/90 재구매 시퀀스** (배송완료 트리거 자동)
  - D+30: "남은 양 괜찮으세요?" + 적립금 알림
  - D+60: "슬슬 떨어질 때예요 💕" + 10% 쿠폰
  - D+90: "마지막 쿠폰이에요 😢" + 15% 쿠폰
- **첫구매 → 2회차 쿠폰 깔때기**: 박스 동봉 카드 + D+14 알림톡 (3만원 이상 20% 쿠폰)
- **포토 리뷰 → 적립금 사이클**: D+10 알림톡 + 포토 3,000원/일반 1,000원

**왜**: 재구매율 8.4% (이커머스 평균 1/3 수준), 재구매 간격이 1-3개월에 정확히 몰려있어서 데이터가 발송 타이밍을 알려주고 있음. 회원 9,984명 × 알림톡 7-15원 = 월 7-15만원이면 전체 커버.

**누가/공수**: 카페24 앱마켓 (알리고/플러스친구 + 리뷰 자동화) 월 5-10만원. **초기 세팅 16시간 → 이후 0시간/월 자동**

**기대 효과**:
- 재구매율 **8.4% → 12-13%**
- 재구매 매출 연 **+15-25%p** (40M → 46-50M)
- **이익률 +1.5~2.0%p**

**리스크**: 카카오 알림톡 스팸 심사 반려 → 정보성 멘트로 통과율 ↑

### 액션 3. 📈 광고 마진 믹스 개선 (1-3개월)
**=> 모리 + 픽시 + 보리 동의**

**무엇을:**
- **섬유탈취제 광고 비중 7.5% → 15%로 두 배 증액** (마진 72.1%, 손익분기 ROAS 1.39 → 현재 1.46이라 유일 흑자 카테고리)
- **식세기리타겟 예산 4% → 10-12%** (ROAS 2.57로 최고 효율, 14개월 누적 7.7M밖에 안 씀)
- **크리에이티브 리프레시 사이클**: 매주 신규 소재 2개, frequency 2.5 도달 시 자동 교체
- **카페24 회원 9,984명 → 메타 Custom Audience 업로드 + Lookalike 1%** 생성 (CRM × 광고 시너지)

**왜**: 광고비를 더 쓰는 게 아니라 **어디에 쓰느냐**를 바꾸는 게임. 식세기 한 카테고리 83% 집중 → 75%로 분산.

**누가/공수**: 마케터 풀타임 업무로 흡수 (이미 광고 운영 중)

**기대 효과**:
- 블렌디드 ROAS **1.79 → 2.00+**
- 카테고리 마진 믹스 개선으로 마진율 **60% → 61~62%**
- **이익률 +1.0~1.5%p**

**리스크**: 섬유탈취제 학습 기간 동안 초기 CAC 높음 → 2주 테스트 예산 1.5M 상한 고정

### 액션 4. 🔧 데이터 인프라 정비 (한 달 내, 거의 1회성)
**=> 픽시 + 보리 동의**

**무엇을:**
- **cost 누락 752건 / 16.9M 마스터 등록** (건조기시트 27개 묶음, 섬유탈취제/얼룩제거제 묶음 패키지 우선)
- **BOM 미등록 SKU 카페24 등록 차단 룰** (스크립트, 하치가 짤 수 있음)
- **광고비/매출 45% 절대 컷라인 월별 대시보드** (45% 초과 시 익월 광고예산 -10% 자동 트리거)
- 손익분기 ROAS = **1.95** 캠페인별 강제 룰

**왜**: 지금 보고된 이익률 12.1%는 cost 누락으로 **실제로는 1%p 더 낮을 가능성**. 진짜 숫자 모르면 의사결정이 헛돌게 됨.

**누가/공수**: 대표님 + 하치 4-8시간 (1회성)

**기대 효과**:
- 측정 정확도 +1%p 보정
- 광고비 절대 컷라인이 단독으로 **이익률 +2~3%p 가능** (픽시 시뮬레이션)
- 운영 룰 정착 → 재발 방지

**리스크**: 일부 묶음 SKU가 마이너스 마진으로 드러나면 가격 조정 필요

### 액션 5. 🎁 광고 외 매출 레인 추가 (2-3개월, 외주 중심)
**=> 보리 단독 강력 추천 + 광고 의존도 분산**

**무엇을:**
- **카카오선물하기 입점**: 섬유탈취제 + 캡슐세제 + 얼룩제거제 3종 (수수료 10-13%, 광고 안 쓰는 매출)
- **체험단 플랫폼 외주** (오마드/디너의여왕/레뷰): 월 30-50건 섬유탈취제 + 얼룩제거제. 광고비 대신 제품 원가만 나감
- **LP/상세페이지 외주 일괄** (크몽/숨고 150-300만원): 메인 LP + 하트식세기 + **"iLBiA 3종 하우스키핑 세트"** 신규 SKU 페이지
- **결제 단계 업셀**: "3천원 더 담으면 섬유탈취제 100ml 증정"

**왜**: 식세기 단일 의존 68.6% 분산 + 광고비 안 쓰는 매출 레인 확보. 자사몰 전환율 1.5% → 2.0% (외주 LP 평균 개선폭).

**누가/공수**: 초기 외주 의뢰 (대표님 20시간 / 1회), **상시 운영 거의 0** (ERP 연동)

**기대 효과**:
- 광고 안 쓰는 매출 월 **+3-8M** (카카오선물하기 + 체험단)
- 자사몰 AOV 35K → 42K (LP 외주)
- **이익률 +0.3~0.8%p**
- 식세기 단일 의존 리스크 분산

**리스크**: 카카오선물하기 심사 2-4주 / 외주 디자인 품질 편차 → 포트폴리오 검증된 업체 + 중간 체크 2회 계약
