## 1. 진단 — 12% → 15% 갭의 본질 (5줄)

대표님~ 숫자로 먼저 정확히 정의해드릴게요 💕

1. **14개월 종합**: 매출(net+배송) **404M** / 광고비(VAT 포함) **약 194M** / 원가 **139M** / **이익(광고후) 약 49M / 이익률 12.1%**.
2. **광고비 점유율**: 매출 대비 **47.9%** — 마진율(60%)에서 광고비 점유율(47.9%)을 빼면 **이익률 12.1%**가 정확히 나옵니다. 이게 손익공식이에요.
3. **갭의 본질**: 15% 달성 = "광고비/매출을 **45% 이하**로 깎거나 마진율을 **63%로 끌어올리거나**". 둘 다 약 **3%p 개선**이 필요합니다. 액수로는 **약 12M/년 추가 이익**.
4. **광고비 점유율은 *악화* 중**: 메타 보고서 기준 광고비가 2025-01 7.9M → 2026-01 22.0M로 **2.8배** 폭증. ROAS는 1.8~2.4 박스권. 즉 **매출 성장의 절반 이상이 광고비로 빨려나가는 구조**입니다.
5. **결론**: 마진율은 이미 60%로 양호 → **광고비 컷이 1순위**, 적자 SKU 출혈 차단이 2순위. 매출 톱라인 푸시는 광고 효율 손익선 확보 전까지 **금지**.

## 2. 액션 제안 5개 (재무 관점)

### 액션 ① 손익분기 ROAS 컷라인 = **1.95** 강제 적용
- **무엇**: 메타 캠페인 단위로 ROAS **1.95 미만은 즉시 OFF**. (월 검토 주기 → 주간으로 단축)
- **왜 (숫자)**:
  - 픽셀 ROAS는 카페24 실매출의 약 1.0~1.05배로 거의 1:1. 마진율 60% 기준으로 **광고비/매출 ≤ 45%**가 되려면 ROAS ≥ **1/0.45 ≈ 2.22 (픽셀 기준 약 2.0~2.1)** 필요.
  - 현재 카테고리별 평균 ROAS — 건조기시트 0.80 / 얼룩제거제 1.0 / 캡슐세제 1.24 / 섬유탈취제 1.46 → **컷라인 미달 4개 카테고리**.
  - 미달 카테고리 합산 지출 = **약 23.4M** (건조기 3.4 + 얼룩 1.9 + 캡슐 3.8 + 섬유 14.3). 이 중 회복 불능 SKU만 OFF/리할당하면 **광고비 8~12M 절감 가능**.
- **기대 효과**: 광고비 5% 컷 → **이익률 +2.4%p** (12.1% → 약 14.5%)
- **리스크/조건**: 매출 단기 -8~12% 가능. **단, 마진율 1위인 섬유탈취제는 컷 대신 소재 교체로 ROAS 끌어올리는 우선 시도** (매출 비중 5% 키울 기회).

### 액션 ③ 정산시트 cost 누락 SKU 원가 즉시 등록 (1,694만원 누락)
- **무엇**: 모리 보고서 기준 **752건, 16,941,200원**의 cost 누락 분(no_cost) 마스터에 원가 등록.
- **왜 (숫자)**:
  - 2025-04 84건 3.4M, 2025-10 127건 4.6M, 2025-11 143건 0.96M, 2025-12 82건 0.7M, 2026-01 101건 2.6M, 2026-02 152건 3.0M.

- **기대 효과**: **이익률 보고 정확도 +1%p 보정**. 실태 파악 후 **잘못 굴러가는 묶음 SKU 발견 가능성** 큼.

### 액션 ④ 광고비/매출 절대 컷라인 = **45%** 월별 강제 룰

- **왜 (숫자)**:
  - 14개월 평균 47.9%. 월별 편차 — 광고비 폭증 월(2026-01: 카페24 광고 22.0M / 매출 48.3M → **45.6%**), 양호 월(2025-04: 8.5M / 16.5M → **51.5%**, 어… 양호 아닌데요? 4월도 위험).
  - 사실상 **지난 14개월 중 광고비/매출 < 45% 달성한 달은 손에 꼽힙니다**. ROAS 보다 이 한 줄이 본질.
- **기대 효과**: 전체 광고비를 14개월 평균 47.9% → **44%**로 3.9%p 컷 → **이익률 +3.9%p (12.1% → 약 16%)**. **유일하게 단독으로 KPI 15% 달성 가능한 액션**.
- **리스크/조건**: 매출 추정 하락 -5~10%. **단, 직원 2명 / 자금 빠듯이라는 대표님 상황에서는 매출보다 현금이 우선**. 이건 무조건 해야 합니다.

### 액션 ⑤ 섬유탈취제 카테고리 매출 비중 5% → 10%로 두 배 증대 (마진믹스 개선)
- **무엇**: 마진율 1위 **섬유탈취제(72.1%)** 의 노출/광고 비중을 의도적으로 식세기 일부에서 이전.
- **왜 (숫자)**:
  - 카테고리별 마진율: 식세기 67.2% / 건조기시트 54.2% / 캡슐세제 60.3% / **섬유탈취제 72.1%** / 얼룩 50.0%.
  - 섬유탈취제는 매출 비중 4.93% (18.7M)밖에 안 됨. 만약 비중 10%로 끌어올리면 **+18M 매출**이 마진율 72%에서 발생 → **+13M 추가 이익**.
  - 광고 ROAS는 1.46으로 컷라인 미달이지만 **마진율이 높아서 손익분기 ROAS는 1.39** (1/0.72). 즉 **유일하게 ROAS 1.46이 흑자 구간인 카테고리**.
- **기대 효과**: 마진믹스 개선으로 전체 마진율 **60% → 약 61.5%**, 이익률 **+1.5%p**.
- **리스크/조건**: 식세기 카테고리에서 이전 시 식세기 매출 단기 감소. **소재 테스트 없이 광고비만 옮기면 ROAS 더 떨어질 수 있음** → 모리한테 소재 먼저 만들어달라고 요청 필수.

## 3. 핵심 인사이트 3개 (다른 페르소나가 흥분할 때 막아야 할 함정)

### 인사이트 ① "ROAS 1.79니까 광고는 흑자야!" 라는 함정
- **광고비/매출 비율 47.9%** 가 본질입니다. 마진율 60% 회사에서 광고비가 47.9%면 **이익률 12% = 거의 손익분기**. ROAS 1.79는 매출 기준이지 이익 기준이 아닙니다.
- **손익분기 ROAS = 1/(1-마진율 잔여) = 1/0.4 = 2.5** (광고비/매출 40% 기준). 우리 모든 카테고리가 손익분기 미달입니다 (식세기리타겟 2.57만 흑자).

### 인사이트 ② 재구매율 8.4%는 광고비 회수의 사형선고
- 전체 회원 9,984명, 재구매 834명. **15회 이상 재구매 51명(0.5%)**. 재구매 매출 기여 = 40.5M / 380.6M = **10.6%**.
- 즉, **신규 한 명을 광고로 데려와도 90.6%는 한 번 사고 떠난다**. CAC 회수가 안 되는 구조 → 광고비를 더 쓰면 절대 안 되는 회사입니다.
- **CRM/리텐션 살리기 전까지 광고 푸시는 자살행위**. 키노한테 이거 무조건 1순위로 던져야 해요.

### 인사이트 ③ 매출 폭증 월(2025-10: 68M)은 이익률 함정의 전형
- 2025-10 매출 68M (월평균 대비 2배), **extra_discount 17.8M** 이라는 거대한 할인 발생. 광고비 12.7M / 광고비/매출 25.6%로 좋아 보이지만…
- **할인 17.8M = 사실상 마진을 깎아서 매출을 만든 것**. 마진율은 63.5%로 평균보다 약간 낮음. **할인+광고를 합친 마케팅 비용** 으로 봐야 진짜 이익률이 보임.
- **할인을 마케팅 비용으로 보면 30.5M / 68M = 44.9%** → 진짜 효율은 평균. **"할인 폭탄 매출"에 흥분하지 말 것**.
