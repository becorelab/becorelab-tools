파일명: `{정산일}_판매수수료 정산.xlsx`
- 헤더: R2 (R1은 빈 행)

| 16 | 판매가(A) | 단가 |
| 17 | 판매수량(B) | 수량 |
| 18 | 판매액(A*B) | 총 판매액 |
| 24 | **정산대상액** | ⚠️ 수수료 이미 차감된 금액 |
| 27 | 판매수수료 | 참고용 (차감하면 안 됨) |
| 28 | 판매수수료 VAT | 참고용 |

#### ⚠️ 핵심: 정산대상액 = 수수료 차감 후
```
정산대상액 = (판매액 - 판매자쿠폰) × (1 - 8.58%)
           = 판매액 - 쿠폰 - 수수료 - 수수료VAT
```
**정산대상액에서 수수료를 다시 빼면 이중차감 오류!**

## 3. 정산 스케줄

### 주정산 구조 (1차 70% + 2차 30%)
```
정산대상액 × 70% = 1차 지급액
  → 광고비, 빠른정산, CFS 물류비 추가 차감
  → 정산주기 종료 후 약 20영업일

정산대상액 × 30% = 2차 지급액
  → 판매월의 익익월 지급
  → 추가 차감 가능
```

### 빠른정산 (셀러월렛)
- 구매확정 금액의 90% 익일 선지급
- 체크카드 수수료 0%
- 계좌인출 시 일 0.012%
- 1차 정산에서 이용액 차감

### Wing에서 확인할 것
- 정산 상세 > 각 정산주기별 지급액/차감 내역
- `정산대상액 × 70% = 지급액` 검증
- CFS 물류비: 할인적용가(A-B) × 1.1 과 일치하는지
- 광고비: Wing 차감액 = 실제 광고비 (키워드 보고서와 차이 있음)

## 4. 광고비 처리

| 출처 | 특징 | 용도 |
|:--|:--|:--|
| Wing 정산 차감 | 실제 정산에서 빠지는 금액 | **순마진 계산 기준** |
| 키워드 보고서 | 키워드 광고만 포함 | SKU/옵션별 배분 비율 |

```
Wing 광고비 > 키워드 보고서 합계
→ 차이 = 디스플레이/브랜드/기타 광고 (키워드 보고서 미포함)
→ SKU별 배분: 키워드 비율로 Wing 총액을 배분 (추정)
```

### 광고 파일 (키워드 보고서)
- 파일: `A00940134_pa_daily_keyword_YYYYMMDD_YYYYMMDD.xlsx`
- 44열 포맷 (캠페인ID 포함, 대표님 수동 다운로드)
- col 8: 광고집행 옵션ID ← **float 타입 주의!** `str(int(float(v)))` 변환 필수
- col 15: 광고비

## 5. 마진 계산 공식

```
마진 = 정산대상액 - 원가 - 물류비(VAT포함) - 광고비(Wing)

정산대상액: 판매수수료 파일 col24 (수수료 이미 차감됨)
원가: 옵션별 원가 × 수량
물류비: CFS 할인적용가(A-B) × 1.1 (배송완료일 기준)
광고비: Wing 정산 차감 총액 (SKU별은 키워드 비율 배분)
```
