# 쏘핑네흰집 X 일비아 공동구매 트래킹 (2026-06-22 ~ 06-25)

## 쓰는 법
스마트스토어에서 '전체주문발주발송관리' 엑셀을 받아서 (비번 `1111`) 하치한테 주시면 됩니다.
또는 직접:

```bash
python3 track.py "<엑셀경로>"        # 새 파일 반영 + 리포트
python3 track.py --report           # 파일 없이 누적 마스터로 리포트만
```

## 동작
- 주문을 **상품주문번호로 누적 병합**(`orders.json`) → 발송완료로 목록에서 빠진 주문도 보존
- **결제일 기준** 일별·시간대별 추이 집계
- 옵션 세트를 풀어 **제품 단위로 환산** (재고 차감용)

## 산출 파일
| 파일 | 내용 |
|---|---|
| `일별_매출.csv` | 일자별 건수/수량/주문금액/정산예정 |
| `일별_제품수량.csv` | 일자별 제품 판매 수량 (재고용) |
| `시간대별_매출.csv` | 날짜·시간대별 주문 |
| `orders.json` | 누적 주문 마스터 (자동 관리) |

## 환산 규칙 메모
- **풀코스 기획세트** = 건조기시트 3종 + **캡슐세제 2개** + 캡슐표백제 2개 + 얼룩제거제 1개 (세제=캡슐세제 확정)
- 캡슐세제 2+2=4개 / 1+1=2개, 캡슐표백제 동일
- 식세기세제 "3개+용기+집게" = 식세기세제 3개 (용기·집게는 사은품)
- 건조기시트 향별로 분리 집계
